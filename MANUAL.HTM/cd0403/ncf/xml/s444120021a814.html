Helpnet


・
By touching "Help Net" in the information mode screen, you can use an emergency call system and helpnet.See the helpnet system for details.

Main menu


・
By touching the "main menu" in the information mode screen, you can use the total mobility service "G -BOOK".For more information, see the G -Book ALPHA system.

amusement


・
By touching "amusement" in the information mode screen, you can use the amusement application (AutoLive, G-Sound, etc.).For more information, see the amusement.

FM multi -weight


・
By touching "FM multi -layer" on the screen during the information mode screen, it will be shifted to the FM multiple screen and displayed the character information of the FM multi -fold general character.There is also information that cannot be displayed while driving.
FM multiple screen





FM multiple mode function list

Item
Content
table of contents
Displays up to 6 items of the FM multi -heavy general character broadcast from up to 6 items (select information from "1" to "6" on the screen).In addition, you can call the table of contents screen by touching the "table of contents" on the screen on a screen other than the table of contents.
Record / call / erasure
When you touch the "memo record" on the screen, the character information currently displayed is recorded on the navigation computer (up to 20).By touching "memo call" on the screen, the recorded character information is called and displayed.In addition, the recorded information is deleted by touching "erasure" or "all erasure" on the screen.If you touch "Erase", the character information currently displayed is deleted, and if you touch "All Erasive", you will delete all the character information remembered in the HDD navigation.
Automatic sending / stop
When there are multiple pages of FM multi -heavy general character broadcasting, each page is automatically sent by touching "Automatic Send" on the screen.In addition, automatic sending can be stopped by touching "stop" on the screen.In addition, automatic sending while driving cannot be done.
Automatic Selection Bureau
By touching the "Automatic Selection Bureau" on the screen, we will select a broadcasting station that transmits FM multiple character information.
Emergency broadcast reception
When the FM multiple emergency information is received during the character information display, the display is automatically switched.
Interlocking program
By touching "program linkage" on the screen, it switches to linked information programs.


TEL (telephone book display)


・
Touch "TEL" on the screen during the information mode screen, then move to the phonebook screen by touching "transportation / navigation" on the phone screen, and is remembered on the hard disk (HDD).You can display the phone number list of (JAF and Road Traffic Information Center, etc.).
・
If the phone number is entered at the destination, the registered memory point, and the special memory point, you can also display the phone number at that point.
Telephone book screen





TEL (hands -free call)


・
The voice recognition switch / switch / Mike of the Options Optional Option Settings will be transferred to the telephone screen by touching the "TEL" on the screen while displaying the information mode screen, and hands -free calls can be made with Bluetooth compatible mobile phones.For more information, see Hands -free calls.

Peripheral monitor


・
By touching the "peripheral monitor" on the screen during the information mode screen, you can move to the back guide monitor setting screen and set the back guide monitor.

Operation guide


・
By touching the "Operation Guide" on the screen during the information mode screen, you can see the operation description on the navigation function, audio function, G -BOOK ALPHA, and voice recognition function.

maintenance


・
By touching the "maintenance" on the screen during the information mode screen display, it will be transferred to the maintenance menu screen, displaying various maintenance information such as the oil change period and input (timing, distance, dealer, etc.).can.In addition, by touching "personal items" on the screen during the maintenance menu screen display, it has been transferred to the personal item screen, and it is possible to display personal information such as the license renewal time and input the time.

・

Etc


・
It is displayed when the ETC is installed in the supplies, and by touching it, it can be migrated to the ETC screen, and you can display each information display of ETC (Electronic Toll Collection: Automatic Fee Collection System) (usage history, traffic fee, etc.).。

setting


・
By touching the "Settings" on the screen during the information mode screen display, you can set the continuation display of each operation screen such as the audio operation screen and information screen.In addition, you can select the switch color of the setting screen (screen other than the map) from two types (green orange).
・
In addition to the above settings, if the optional communication device is connected, each setting can be performed to use the communication function (G -BOOK ALPHA, Hands -free Helpnet).
・
The personal information erasing switch allows you to erase information such as the memory point of the navigation, the hands -free phonebook, and the sound library recorded in the sound library.
