Auto -leveling configuration

●
The auto -leveling system is composed of headlamp leveling computer, high control sensors, etc.

List of main components of auto -leveling system


Components
function
Head lamp leveling computer
The vehicle posture and driving status is determined by the high control sensor and the combination meter, and the headlamp leveling actuator is controlled according to the condition.
When the abnormality of the height control sensor is detected, the headlamp leveling warning indicator lamp in the combination meter lights up.
Head lamp leveling actuator
The control signal from the headlamp leveling computer drives the headlamp reflector up and down.
Hight control sensor
The height of the vehicle is detected and the height change signal is output to the headlamp leveling computer.
Combination meter
Built -in head lamp leveling warning warning indicator lamp.
Output the vehicle speed signal to the headlamp leveling computer.
