Back guide monitor system configuration ("Wide Multi AV Station" equipped car)

Main parts configuration map




Back guide monitor system configuration parts


Components
Major function
Back monitor camera (made by Matsushita Electric Industrial)
It is attached to the backdoor -only camera cover, and outputs the back guide monitor computer to the back guide monitor computer.
Back guide monitor computer (made by Aisin Seiki)
Attached to the lower part of the driver's seat, Ignition Switch, neutral start switch (A/T car), backup lamp switch (M/T car), combination meter, and "Wide Multi AV Station" signals to back monitor cameras.ON/OFF.
The image signal is output to the "Wide Multi -AV Station" by the signal from the back monitor camera and the steering angle sensor.
“Wide Multi AV Station” (made by Fujitsu Ten)
Video signals from the back guide monitor computer display the rear video and each guide line on the screen.
Navigation computer (made by Aisin Aye Dublyu)
Output the RGB video signal to the back guide monitor computer.
Output vehicle angle speed data to the back guide monitor computer.
Steering angle sensor
It is attached to the steering column and outputs the steering angle signal to the back guide monitor computer.
Neutral start switch
Output the shift position "R" signal to the back guide monitor computer.
Combination meter
Output the vehicle speed signal to the back guide monitor computer.
Right front speaker
The voice signal from the back guide monitor computer outputs the guide voice when selecting a column guide mode.
