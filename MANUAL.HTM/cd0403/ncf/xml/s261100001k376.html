30 seconds auto lock function


・
If you leave the unlock button of the transmitter key and leave the dooran lock for about 30 seconds, you will automatically lock all the doors and prevent left in the dooran lock.The auto lock function is canceled when one of the following conditions is established within 30 seconds.

Auto lock function cancellation condition

・
When any door is opened.
・
When all doors are locked.
・
When the key plate is inserted into the ignition key cylinder.


Repeated function


・
After the door lock, if the lock operation (driver's door lock position switch off) cannot be confirmed for some reason, the lock operation will be performed again about 1 second.
