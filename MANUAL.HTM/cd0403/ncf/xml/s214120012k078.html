Auto preset function (when using radio)


・
By pressing the auto preset (AUTO/P) button, the tuner selection memory button (1-6 1 to 6) is automatically tuner selection of broadcasting stations (up to 6 stations each of AM/FM) among the broadcasting stations that can be received at the current location.) Preset (memory).

2003 specification "Wide Multi AV Station" (CD / MD type) operation


・
The operation of the "Wide Multi AV Station" can be performed using a panel switch on the front of the main unit, a touch switch, and a voice recognition system that is displayed on a retailer.

Touch switch operation

As in the past, the operation after calling the audio mode screen by panel switch operation is performed by directly touching the switch frame (touch switch) displayed on the screen with the finger.


Audio operation

After pressing the voice recognition utterance switch, displaying the audio recognition mark on the screen, and speaking toward the audio recognition microphone, it is possible to operate each function such as the switching of audio sauce, which is excellent in use.AndFor more information about voice operation, see the visual system (“Wide Multi AV Station”).(References visual system (“Wide Multi AV Station”) - 2003 specification “Wide Multi AV Station” operation)
