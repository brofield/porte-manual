Automatic Transx Axle Overview [U340F type]

●
The U340F type automatic trance axle "SUPER ECT (Super Intelligent 4-speed Automatic)" has been set to a 4WD car with a 1NZ-FE engine.
●
The U340F Automatic Transxle is an automatic transxle that has been optimized for 4WD based on the U340E type used for 2WD vehicles.Basic structures such as gear trains and ECT control are followed by the U340E type set in 2WD vehicles, and some shapes such as transformers have been optimized with transfers settings.
