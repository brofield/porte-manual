Disposal processing function

●
In order to discard the airbag when scrapped, a communication and disposal function with the disposal tools have been set.
Batch operation processing connector


・
A connector for collective operation processing has been set for the center airbag sensor (computer).
・
By connecting the dedicated tool to this connector, the batch operation of the airbag system is possible.


Batch operation processing display label


・
At the top of the passenger door lock striker, an information label is attached to display the center airbag sensor ASSY mounting position.In the case of this vehicle, the center airbag sensor ASSY is attached to the floor tunnel in front of the center console.
