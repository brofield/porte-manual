FM multi -weight


・
By touching "FM multi -layer" in the information mode screen, the character information of the FM multi -fold general character broadcast is displayed.There is also information that cannot be displayed while driving.
FM multiple mode screen





FM multiple mode function list

Item
Content
table of contents
Displays up to 6 items of the FM multi -heavy general character broadcast from up to 6 items (select information from "1" to "6" on the screen).In addition, you can call the table of contents screen by touching the "table of contents" on the screen on a screen other than the table of contents.
Record / call / erasure
When you touch the "memo record" on the screen, the character information currently displayed is recorded on the navigation computer (up to 20).By touching "memo call" on the screen, the recorded character information is called and displayed.In addition, the recorded information is deleted by touching "erasure" or "all erasure" on the screen.If you touch "Erase", the character information currently displayed is deleted, and if you touch "All Erasive", you will delete all the character information remembered on the navigation computer.
Automatic sending / stop
When there are multiple pages of FM multi -heavy general character broadcasting, each page is automatically sent by touching "Automatic Send" on the screen.In addition, automatic sending can be stopped by touching "stop" on the screen.In addition, automatic sending while driving cannot be sent.
Automatic Selection Bureau
By touching the "Automatic Selection Bureau" on the screen, we will select a broadcasting station that transmits FM multiple character information.
Emergency broadcast reception
When the FM multiple emergency information is received during the character information display, the display is automatically switched.
Interlocking program
By touching "program linkage" on the screen, it switches to linked information programs.


maintenance


・
By touching "maintenance" in the information mode screen, you can migrate to the maintenance menu screen, display and input various maintenance information such as oil change time (time, distance, dealer, etc.).In addition, by touching "Personal Items" on the screen during the maintenance menu screen, it has been transferred to the personal item screen, and it is possible to display personal information such as the license renewal time and input the time.

Information mode function setting


・
By touching "Settings" in the information mode screen, you can move to the setting screen and set the continuation display of each operation screen such as audio operation screen and information screen.You can also select the background color of the setting screen (screen other than the map) from four types (green blue, gray beige).
