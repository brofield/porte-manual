Electric storage door mirror system configuration parts




Electric storage/return function


・
When the storage switch is operated with the ignition switch ACC or ON, the mirror is stored or returned.
・
When the ignition switch is turned off during storage or returning, the mirror stops operating when the ignition switch is turned off.In addition, if you re -ACC or turned on the ignition switch, the mirror will continue to operate and will be stored or returned.
・
If the status of the storage switch and the mirror are different in the ignition switch off (the storage switch is the storage switch, the mirror is the return position, etc.), the runition switch is ACC or turned on from OFF, and the mirror according to the status of the storage switch.It will be stored or returned.

Return to the door mirror


・
If the left and right mirrors are operated on the returning side at the storage position, both the left and right mirrors will return.
・
The mirror retains the state when the mirror on one side from the storage position is manually, and when the return position is set to the return position.At this time, if the storage switch is operated to the return side, only one mirror will be returned.


Door mirror storage


・
If the left and right mirrors are operated on the storage side at the return position, both the left and right mirrors are stored.
・
When the mirror on one side is manually from the return position, and the omission position is set in the front, the mirror holds the state.At this time, when the storage switch is operated to the storage side, both the left and right mirrors are stored.
・
The mirror retains the state when the mirror on one side is manually from the return position.At this time, when the storage switch is operated to the storage side, only one mirror is used to store.


Door mirror Fail Safe


・
The mirror stops with the obstacle from moving.For this reason, after removing the obstacle, operate the storage switch to the storage position or return position.


Door mirror operation





Door mirror storage operation
・
When both left and right mirrors are returned, the storage switch is returned, when the ignition switch ACC or ON, the storage switch is operated on the storage side, the contact A1 and A2 of the storage switch switch to B1 and B2.As a result, the current flows from ACC to B2 → MR terminal → Fet1 → PTC1 → D1 → Return Motor → MF terminal → B1 → Earth, and the storage / restoration motor rotates to the storage side.
・
After that, when the storage operation is completed, when the current that the transistor Q1 turned on and the storage side was turned on the MR terminal → Q1 → R1 → MF terminal → B1 → ground, the power to the storage / return motor was cut off., Storage / return motor stops.


Return of the return
・
When the left and right mirrors are stored, the storage switch is stored, when the ignition switch ACC or ON, the storage switch is operated to the return side, and the contact switch contact B1 and B2 of the storage switch switch to A1 and A2.As a result, the current flows to the ACC → A1 → MF terminal → storage / return motor → D2 → PTC2 → FET2 → MR terminal → A2 → earth, and the storage / return motor rotates to the return side.
・
After that, when the return operation is completed, the current that the transistor Q2 turned on and the return side was turned on the MF terminal → R2 → Q2 → MR terminal → A2 → ground, and the power to the storage / return motor was cut off., Storage / return motor stops.
