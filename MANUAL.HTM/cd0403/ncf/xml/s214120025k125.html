Combination meter

●
With the adoption of the VSC, each indicator lamp of VSC warning, slip, and TRC off has been set in the combination meter of TRC & VSC -equipped vehicles (manufacturer options).
●
For the combination meter, all cars are used for all vehicles with the dial and guidelines that spontaneously glow.
●
A stepper motor has been adopted for the movement movement of each meter.
●
An electronic Odo/Trip meter displayed on LCD (liquid crystal display) has been adopted.
●
Leostat is adopted to make it possible to switch the meter brightness.
●
The car speed signal shall input signals from a wheel speed sensor for ABS via a skid control computer.
