DVD navigation system major components list


Parts name
Main function
DVD navigation (made by Fujitsu Ten)
Various information and operating status are displayed by video and audio by inputting and processing the main built -in components (main computers, display panel, navigation computers) and data from components placed in various parts of the vehicle.
Main computer
Enter and control the RGB video signal from the navigation computer and output it to the display panel.
Controls each display and function of the DVD navigation according to each operation signal input.
Adjust the color balance of the display screen.
Display panel
The RGB video signal from the main computer displays a map screen and operation screen with a color LCD panel.
Output each operation screen to the main computer.
Navigation computer
The position of the vehicle is detected by the signal from the built -in GPS receiver.
The direction of travel is detected by a signal from the built -in gyro sensor.
The map information data read by the built-in DVD-ROM player and the calculated position are output to the main computer as the RGB video signal.
The mileage is calculated by the vehicle speed signal from the combination meter.
Output the navigation guidance audio signal to the right front speaker.
DVD-ROM disk for maps
One card is loaded on a built-in DVD-ROM player on the navigation computer, and the data of the map information, guidance audio, and destination search is memorized.
GPS antenna (made by Aisin Seiki)
Receive the orbital signal and signal transmission time that is constantly transmitted from the GPS satellite placed on the track surrounding the earth at an altitude of about 20,000 km, and output it to the GPS receiver.
speaker
Outputs navigation information signals and audio audio.
Radio antenna
Receive radio broadcasting radio waves in a glass antenna and output to DVD navigation.
Combination meter
Output the vehicle speed signal to the DVD navigation.
Tail lamp tray
Output the tail lamp or to the DVD navigation.
Parking brake switch
Output the parking brake on to the DVD navigation.
Neutral start switch
Output the shift position signal to the DVD navigation.
