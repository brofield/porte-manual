2003 specification "Wide Multi AV Station" operation




Panel switch operation
・
As before, the display screen of the "Wide Multi AV Station" is composed of a TV mode screen, information mode screen, navigation mode screen, etc., and each mode screen can be called from the panel switch placed at the bottom of the display.Masu.


Display panel opening and closing operation
・
As with before, the display panel part is a downturn type, and the operation of Open / Close / Tilt is performed by switching.


Touch switch operation
・
As before, the operation after calling each mode screen (TV / information, etc.) is performed by a touch switch that can be input by directly touching the switch frame displayed on the display screen with a finger.


Audio operation
・
After pressing the voice recognition utterance switch, displaying the audio recognition mark on the screen, the voice command can be performed by uttering the voice command to the audio recognition microphone, so that the operation is excellent.
・
There are "standard mode" and "expansion mode" for voice operation, and each can be switched on arbitrarily depending on the application.
