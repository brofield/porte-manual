Skid control computer

●
Outputs the operation signal to the brake actuator so that the wheel speed sensor of each wheel speed sensor is controlled according to the road surface condition.

Fail Safe function


・
If an abnormality occurs in skid control computers and brake actuators, the ABS Warnin Gramp is turned on and prohibits the operation of ABS and brake assist systems.If an abnormality that cannot be functioned by the EBD occurs, the brake warnin group is further turned on to prohibit EBD control.

Diagnostic function


・
Using a diagnostic function, we considered service.Please refer to the repair letter for details.
