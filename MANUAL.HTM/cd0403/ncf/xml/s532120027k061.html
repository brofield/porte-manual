Illuminated entry system

●
Even after each door is closed, an illuminated entry control that lights the room lamp for about 15 seconds is adopted, and has been set as standard for all vehicles.
●
If the door of the driver's seat is locked during the Illuminated entry control, the lamp on the lights is turned off immediately (fade out).
●
When the driver's seat door is unlocked by the key linked or transmitter, or the driver's seat door is unlocked by the smart door lock system*The door lock lamp control, which lights the room lamp for about 15 seconds, is set as standard for all vehicles.
●
If any door is half -door or open, and if the ignition key is not inserted, the room lamp will be turned on for more than 20 minutes, and the battery saver function will be automatically turned off.And the standard was set to all cars.
●
A circuit that controls the Illuminated entry system is built into the integration relay attached to the upper part of the passenger seat in the instrument panel.
Attending
*
Only car equipped with a smart door lock system

Door lock lamp control function


・
When all doors are closed in the ignition switch OFF, the room lamp is turned on for about 15 seconds when the driver's seat door is unlocked by the key linked, transmitter and smart door lock system.
・
If any door is opened and closed while the lamp is lit by the door lock lamp control, the door lock lamp control will be discontinued and the Illuminated entry will be transferred to control.

Illuminated entry system operation


・
Illuminated entry, door lock lamp, battery saver is performed by integration relay, which controls the lighting/turns on the room lamp, by entering each switch signal.
