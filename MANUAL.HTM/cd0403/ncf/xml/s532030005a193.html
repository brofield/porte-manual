Shift lock overview with key inter lock mechanism

●
A shift lock system has been adopted to prevent erroneous operation of the shift lever.
●
The shift lock system is a mechanical type using a link mechanism, and the structure is simplified by using the function as a key -inter lock mechanism.

Structure and operation of the key inter lock mechanism


・
If the shift position is not set in the P range, the ignition key position cannot be turned ON, ACC cannot be turned from Lock, so that the ignition key cannot be removed.Other than the shift position P range, the shift lock cam is fixed by the shift lever shaft torso, so the shift lock link is also fixed.As a result, the brake pedal side lock pin and the key cylinder side lock pin cannot act on the shift lock link.For this reason, even if you try to rotate the ignition key position from ACC to Lock, you cannot push the lock pin out, so you can not turn the key without turning the ignition key.


Structure and operation of the shift lock mechanism



Shift lock mechanism

・
When the shift position is in the P range, the shift lever position cannot be changed unless the brake pedal is stepped on.The shift lever can only move the shift lever shaft over the pile of the detent plate if both the brake pedal side lock pin and the key cylinder side lock pin are operating to the cancellation side.
