ABS Outline with EBD

●
G sensor has been set for the ABD system with EBD (Electronic BRAKE FORCE DISTRIBUTION) adopted for 4WD cars.
●
A rotor and stick type sensor similar to the front wheel speed sensor has been set as a rear wheel speed sensor of a 4WD car.
