Manual lock & unlock function


・
You can lock or unlock all doors by linking the door lock control switch (for manual operation) to the unlock side.

Doa key linked lock & unlock function


・
The key plate is inserted into the driver's door key cylinder and turned to the front or back side to lock or unlock all doors.
・
When the ignition switch ON and the driver's seat seat belt is attached, cancel the door key linked lock/unlock operation by the above operation*I will do it.

Key closed prevention function


・
When the key plate is inserted into the ignition key cylinder and the driver's seat door is open, the door lock control switch (for manual operation or key linked) and the lock by the door lock knob do not enact the lock into the car.Prevents key closing.

Door lock security function


・
When locking (including auto lock) or lock -linked lock operation using a transmitter key or a transmitter for a smart door lock, a door lock security function is set, and door lock control switch (for manual operation).Prohibit unlock operation by*I will do it.
Attending
*
When you forget to close the door glass, you can operate the door lock control switch (for manual operation) with a stick from the gap between the door sash and the door glass to prevent unlocking the door.

Electric door lock system circuit




Manual lock <unlock> operation


・
If you operate the door lock control switch (for manual operation) on the lock <anlock> side, the door lock <anlock> requirement signal will be entered in the CPU.As a result, the CPU turns TR1 <TR2> through the driver circuit and turns on the lock relay <unlock relay>.At this time, the unlock relay <lock relay> forms a ground circuit, and the current is ECU-B →+B terminal → Lock relay <unlock relay> → ACT+terminal <ACT-terminal> → Door lock control motor→ ACT-terminal <Act+terminal> → unlock relay <lock relay> → ground, all door lock control motor rotates to the lock <unlock>.

Doa key linked lock <unlock> operation


・
The key plate is inserted into the driver's door key cylinder, and when operated on the lock <unlock> side, the door lock control switch for key interlock is turned on to the lock <unlock> side, and as in the manual lock operation, all door lock control motor.Rotes to the lock <unlock> side.
