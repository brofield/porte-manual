Smart door lock system

●
A "smart door lock" that allows you to perform all door locks/unlock operations "smart" without having a key in your hand, can be performed "smart" just by wearing a transmitter for smart door locks.C package is set as standard.
●
The Smart Door Lock System is a system that controls a door lock based on the interruption result of the electronic ID code by transmitter and the vehicle wireless communication.
●
The transmitter for smart door locks has a built -in operating indicator lamp (LED) that lights up during the communication and button operation during the ID collation, and can be confirmed to be operated and out of battery.It also has a wireless arock function and can be used like a transmitter.In addition, CR2032 is used for the battery.
●
The smart receiver is located in the driver's instrument panel.In addition, weak radio waves are adopted as an identification code transmission method between transmitter and smart receiver.
●
The identification code registration function has been adopted, enabling multiple transmitter for smart door locks.
Smart Door Lock System Overview


・
Based on the ID code collation result by wireless communication between the transmitter for a smart door lock outside the car, the smart ECU is controlled by the smart ECU by sending a signal to the body ECU.
・
To match the vehicle and the transmitter for a smart door lock, the smart ECU sends out the request signal via an electrical key oscillator outside the car room.
・
The transmitter receiving the request signal sends the ID code.
・
When the smart receiver receives the ID code received by the smart receiver as an identification/collation, the door lock is controlled by sending a lock/unlock requirement signal to the body computer according to the function.



Smart door lock transmitter detection area
・
The request signal sent from the Smart ECU is sent from the electrical key oscillator and forms a transmitter detection area for a smart door lock with a radius of about 0.7 m using the driver's door -out side handle as an antenna.
・
The request signal that forms a detection area is always transmitted every 0.3 seconds to detect the approach of transmitter cell phone.
・
The detection area is formed when all the following conditions are established.

Detection area formation (request signal transmission) Conditions

・
Engine stop
・
Ignition key cylinder not inserted
・
Driving seat door lock
・
All doors "closed"



Smart door an lock function


・
All doors will unlock when you carry a transmitter for a smart door lock in the detection area and hold the driver's outside handle ASSY.
・
After the unlock, the Hazard Warnin Gramp will be changed twice as an answer back.


Smart door lock function


・
At the time of "closed", all doors are locked by carrying a transmitter for a smart door lock in the detection area and pressing the trigger switch.
・
After the lock, the Hazard Warnin Grand Lamp will be changed once as an answer back.


Wireless remote control function


・
The transmitter for the smart door lock has a lock button, unlock power slide door button, and all wireless remote control systems, including the door lock and unlock function, are possible.

Identification code registration function


・
The transmitter identification code on the smart ECU side can be added to up to 7 types of transmitter for smart door locks, transmitter keys in total (up to five types of transmitter for smart door locks and transmitter keys can be registered up to 5 types).In addition, check the identification code rewriting and the number of registration code, or if the transmitter for a smart door lock is lost, the identification code registered in the smart ECU is to be erased and the smart door lock system is prohibited.You can do it.(Please refer to the repair letter for addition, rewriting, confirmation, and prohibition of identification code.)

Smart door lock power saving function




In -vehicle battery power saving function
・
The smart door lock system forms a detection area by periodically transmitting a request signal every 0.3 seconds, so if the vehicle is left for a long time, the in -vehicle battery may increase.Therefore, the following control is set to prevent the battery rising.

In -vehicle battery power saving control

・
If there is no response from the transmitter for smart door locks for 8 days or more, the regular transmission of the request signal is extended to 0.6 seconds.
・
If there is no response from the transmitter for smart door locks for 15 days or more, the regular transmission of the request signal is suspended (the formation of the detection area).



Power saving function of transmitter for smart door lock
・
Even if you continue to press the lock or unlock button of the transmitter for the smart door lock, the transmission operation will be automatically stopped after about 20 seconds.
・
If a transmitter for a smart door lock is left for a long time in the detection area, the transmitter for the smart door lock may continue to send the ID code to the smart receiver, so the transmitter's battery may rise.Therefore, the following control is set to prevent the battery rising.

In -vehicle battery power saving control

If there is a transmitter for a smart door lock in the detected area for 10 minutes or more, stop the request signal regular transmission (the formation of the detection area).



Return conditions
・
When one of the following conditions is established, it returns from power saving control to normal smart door lock control.

Return conditions

・
Lock/unlock the door by wireless arock.
・
Engine start.
・
Press the trigger switch of the driver door out side handle ASSY.

Smart door lock system circuit




Smart door an lock operation


・
When the Ignition Switch OFF, the driver's seat door lock position switch OFF (at the time of lock), the transmitter for all door curtail slump switch OFF and smart door lock enters the detection area, the smart ECU determines the ID code of the transmitter received by the smart receiver/Cause.
・
When the correction is OK, the smart ECU will launch a touch sensor with a built -in driver's door -out side handle ASSY via an electrical key oscillator, and in an unlock standby state.
・
When the driver's door -out side handle ASSY is detected with a touch sensor, this signal is entered into the smart ECU via an electrical key oscillator.As a result, the smart ECU sends the dooran lock demanding signal to the body ECU.
・
The body ECU drives the door lock control motor and unlocks all doors, as in the operation of the electric door lock.After that, the driver's door lock position switch ON is detected and the door -an lock status is confirmed, and the answerback output is performed twice.
・
Smart ECU detects the driver's door lock position switch ON and checks the dooran lock status to stop the request signal.

Smart door lock operation


・
When a smart ECU detects the trigger switch ON in the ignition switch OFF, all door curtechi lamps, ON, a request signal is sent to the electric key oscillator, and a detection area is formed by the driver's door -out side handle ASSY built -in antenna.
・
At this time, the transmitter for the smart door lock in the detection area sends the ID code, which is received by the Smart ECU via a smart receiver.
・
Smart ECU determines and matches the ID code of the transmitter, and when OK, the door lock requirement signal is sent to the body ECU.
・
The body ECU drives the door lock control motor and locks all doors, as in the operation of the electric door lock.After that, the driver's door lock position switch OFF is detected and the door lock status is confirmed, and the answerback output is performed once.

Other smart control




Collision avoidance control with transmitter for other vehicles smart door lock
・
If two cars equipped with a smart door lock system are lined up, and if the transmitter for smart door lock continues to respond to the detection area of ​​other vehicles and can not respond to the detection area of ​​your vehicle, collision avoidance control may be executed.there is.
・
The sending request signal is stopped three times when the Smart ECU of another car has made three consecutive NG judgments to the transmitter for the smart door lock in the detection area.As a result, the transmitter for the smart door lock reacts to the detection area of ​​the vehicle after about one second, and performs a normal smart door lock control.
Collision avoidance control






Power saving control by transmitter for other vehicles smart door lock
・
If the transmitter for the smart door lock of another car is in the detection area of ​​the vehicle, the smart ECU of the car will repeat the above collision avoidance control, and will be shifted to power saving control if it is 10 consecutive times.
・
When power saving control is executed, the smart ECU changes the regular transmission of the request signal every 3 seconds.
・
Returns from power saving control by detecting your car's smart door lock transmitter, wireless remote control operation, and a transmitter for different smart door locks.
