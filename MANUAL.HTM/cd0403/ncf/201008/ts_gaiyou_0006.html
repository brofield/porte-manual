Comfort and usability

Sliding door

Sliding door

●
The sliding door was adopted on the left door to have excellent getting on and off and convenient.
●
The sliding door is used for a swing -type low roler bracket.In addition, all rollers were suppressed when the sliding door was opened and closed as a resin roll.
●
The intermediate stopper was set on the sliding door to prevent the neck and hands between the sliding door sash and the quarter pillar.
●
A lock type with excellent holding power has been adopted for the full -opening stopper of the sliding door.
●
A fuel lid interlock mechanism has been set to the sliding door that prevents the interference between fuel lid and sliding door.
●
The "sliding door normal power supply unit" is set in the sliding door, and even if the door is open, the structure can be supplied to each device in the door.
●
A double -swing door -in handle is adopted for the sliding door, and the lock assembly when the sliding door is closed and the fully opened stopper release at the time of opening can be performed with one operation handle.
●
"Slide de Ikroser", which is sure to close the sliding door with a light force, is set for all vehicles.(Slide de Iiglor System)
●
By operating the door handle, controlled switch, transmitter, a "power slide door" that automatically opens and closes sliding doors has been set in all vehicles.(Reference power slide door system)

Combination meter
