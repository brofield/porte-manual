6. Alternator rotor ASSY inspection



  1. The resistance between slip rings is measured using SST (Toyota Electrical Tester).

    SST
    09082-00030

    Reference value
    2.7-3.1Ω (20 ° C) (A/T car)
    2.1-2.5Ω (20 ° C) (M/T car)




  2. Use SST (Toyota Electrical Tester) to measure the resistance between slip rings and rotor core.

    SST
    09082-00030

    Reference value
    10mΩ or higher




  3. The outer diameter of the slip -ring is measured using a caliper.

    Reference value
    14.2-14.4mm
    limit:
    12.8mm

    □ Participation □
    In other cases other than the reference value, fix it with sandpaper ( # 400).
