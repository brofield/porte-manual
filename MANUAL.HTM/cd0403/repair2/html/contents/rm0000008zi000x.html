1. カムシャフトタイミング ギヤASSY取り付け



  1. カムシャフトのストレートピンに対して、カムシャフトタイミングギヤASSYのキー溝を図のようにずらした状態で勘合させる。

  2. 軽くカムシャフトタイミングギヤASSYを押しながら図の矢印の方向(左回転)に回転させ、キー構とストレートピンが一致したところでさらに押し込む。

    ■ 注 意 ■
    カムシャフトタイミングギヤを絶対に遅角方向(右回転)に回転させないこと。

  3. カムシャフトタイミングギヤASSYとカムシャフトのフランジ部とのすき間が無い事を確認する。

  4. カムシャフトタイミングギヤが回転しないように注意しながらフランジボルトを締め付ける。

    基準値
    T = 64 N・m{ 653 kgf・cm }

    ■ 注 意 ■
    1. 必要以上にバイスで締めすぎて、カムシャフトを傷つけない。
    2. ボルト締め付け時、カムシャフトタイミングギヤASSYを固定させない。
    3. 最遅角位置でロックしてしまった場合は、カムシャフトタイミングギヤ内のロックピンを解除してからボルトを締め付ける。

  5. 締め付け後、カムシャフトタイミングギヤASSYを遅角方向(右回転)に回転させ動くことを確認し、可動端でカムシャフトタイミングギヤASSYがロックすることを確認する。



2. カムシャフト取り付け



  1. カムシャフトカム部およびシリンダヘッドジヤーナル部にエンジンオイルを塗布する。

  2. 図のようにチェーンを手で保持し、チェーンのペイントマークがカムシャフトタイミングギヤASSYのタイミングマークと一致するように、カムシャフトおよびカムシャフトタイミングギヤASSYを取り付ける。




  3. ベアリングキャップNo.2のフロントマークおよび数字を確認し、図の順序で数回に分けて均等に締め付ける。

    基準値
    T = 12.7 N・m{ 130 kgf・cm }

    ■ 注 意 ■
    カムシャフトを水平に沈み込ませる．



3. カムシャフト NO.2取り付け



  1. 図のようにチェーンを手で保持し、チェーンのペイントマークがカムシャフトタイミングギヤのタイミングマークと一致するように、カムシャフトNo.2およびカムシャフトタイミングギヤを取り付ける。

  2. カムシャフトNo.2のノックピンとカムシャフトタイミングギヤのピン溝を合わせる。




  3. フランジボルトで仮付けする．




  4. ベアリングキャップNo.1、No.2のフロントマークおよび数字を確認し、図の順序で合う迂回に分けて均等に締め付ける。

    基準値
    ベアリングキャップNo.1
    T = 23 N・m{ 235 kgf・cm }
    ベアリングキャップNo.2
    T = 12.7 N・m{ 130 kgf・cm }




  5. カムシャフトNo.2のサービス用六角部をモンキーレンチで保持し、フランジボルトを締め付ける。

    基準値
    T = 64 N・m{ 653 kgf・cm }

  6. チェーンテンショナからφ3mmの棒または六角棒レンチを抜きとる。




  7. クランクシャフトダンパを正回転させ、クランクシャフトダンパの切り欠きを0°マークに合わせる。

  8. 各々のあわせマークが一致していることを確認する。

  9. チエーンカバーサービスホールスクリユプラグのボルト先端より、2-3ねじ山にアドヘシブ1324を塗布する。

    ■ 注 意 ■
    ボルトおよびボルト穴を清掃、脱脂する。




  10. ソケットヘキサゴンレンチ8を使用して、チエーンカバーサービスホールスクリユプラグを取り付ける。

    基準値
    T = 15 N・m{ 153 kgf・cm }



4. バルブクリアランス点検



■ 注 意 ■
バルブクリアランスの点検は冷間時に行うこと。

  1. No.1シリンダ圧縮上死点セット

    1. クランクシャフトダンパを正回転させ、タイミングマーク(切り欠き)をチェーンカバー0°の位置に合わせる。

    2. 図の位置にカムシヤフトタイミングギヤのタイミングマークがあることを確認する。

      □ 参 考 □
      マークが合っていない場合は、クランクシヤフトを回転させ再度合わせる。




  2. 図の場所のバルブクリアランスを点検する。

    1. シックネスゲージを使用して、バルブリフタとカムシヤフト間のクリアランスを測定する。

      基準値
      IN　0.15-0.25ｍｍ(冷間時)
      EX　0.25-0.35(冷間時)

      □ 参 考 □
      基準値以外の場合は、クリアランスを記録しておく。

  3. クランクシャフトを正回転方向に一回転させ、No.4シリンダを圧縮上死点にする。




  4. 図の箇所のバルブクリアランスを点検する。

    1. シックネスゲージを使用して、バルブリフタとカムシヤフト間のクリアランスを測定する。

      基準値
      IN　0.15-0.25ｍｍ(冷間時)
      EX　0.25-0.35(冷間時)

      □ 参 考 □
      基準値以外の場合は、クリアランスを記録しておく。



5. バルブクリアランス調整



■ 注 意 ■
チェーンを切り離してカムシヤフトを回転させる場合は、バルブが全開状態でピストンと干渉しないようにクランクシヤフトを左回転させBTDC40°にし、クランクシヤフトダンパの切り欠きが図の位置にあることを確認して行う。




  1. No.1シリンダ圧縮上死点セット

    1. クランクシヤフトダンパを正回転させ、タイミングマーク(切り欠き)をチェーンカバー0°の位置に合わせる。

    2. このとき、カムシヤフトタイミングギヤのタイミングマークが図の位置にあることを確認する。

      □ 参 考 □
      マークが合っていない場合は、クランクシヤフトを回転させ再度合わせる。




  2. ソケットヘキサゴンレンチ8を使用して、チエーンカバーサービスホールスクリユプラグを取りはずす。




  3. マイナスドライバーを使用して、チェーンテンショナサービスホールよりテンショナのストッパプレートを上げた状態で保持する。

    □ 参 考 □
    チェーンテンショナのストッパプレートを上にあげることにより、プランジャのロックが解除された状態となる。




  4. チェーンテンショナのストッパプレートを上げた状態からモンキーレンチを使用して、カムシャフトNo.2のサービス用六角部を少し右回転させ、チェーンテンショナのプランジャが押された状態にする。

    □ 参 考 □
    カムシャフトNo.2を少し右回転させることにより、プランジャが奥に入った状態となる。




  5. モンキーレンチを保持し、プランジャが奥に入った状態からマイナスドライバーを取りはずす。

    ■ 注 意 ■
    モンキーレンチを動かさない。

    □ 参 考 □
    マイナスドライバーを取りはずすことにより、ストッパプレートが下にさがり、ロックされた状態となる。




  6. チェーンテンショナのストッパプレートが下にさがり、ロックされた状態からストッパプレートの穴にφ3mmの棒または六角棒レンチを差し込む。

    □ 参 考 □
    ストッパプレートの穴にφ3mmの棒または六角棒レンチが差し込みにくい場合は、カムシャフトNo.2を少し左回転させ、再度少し右回転させると差し込みやすくなる。

  7. φ3mmの棒または六角棒レンチが抜け落ちないようにテープなどで固定しておく。




  8. カムシヤフトNo.2のサービス用六角部をモンキーレンチで保持し、カムシヤフトタイミングギヤ取り付けボルトを緩める。




  9. 図の順序でカムシヤフトベアリングキヤツプNo.1およびNo.2を取りはずす。

    ■ 注 意 ■
    カムシヤフトNo.2を水平に保持しながら、ボルトを均等に緩める。




  10. カムシヤフトNo.2を少し浮かした状態で、フランジボルトをはずし、カムシヤフトNo.2およびカムシヤフトタイミングギヤを取りはずす。




  11. 図の順序でカムシヤフトベアリングキヤツプNo.2を取りはずす。

    ■ 注 意 ■
    カムシヤフトを水平に保持しながら、ボルトを均等に緩める。




  12. チェーンを手で保持し、カムシヤフトを取りはずす。




  13. チェーンをロープまた針金などで吊っておく。

    ■ 注 意 ■
    チェーンカバーの中に何も入らないように注意すること。

  14. バルブリフタを取りはずす。




  15. マイクロメーターを使用して、取りはずしたバルブリフタの厚さを測定する。

  16. バルブクリアランスが基準値になるようにバルブリフタの厚さを算出する。
    A
    		選択したバルブリフタ
    B
    		取りはずしたバルブリフタの厚さ
    C
    		測定したバルブクリアランス

    基準値
    IN　A=B+　(C-0.20ｍｍ)　(冷間時)
    EX　A=B+　(C-0.30ｍｍ)　(冷間時)

  17. バルブリフタを選択する。

    □ 参 考 □
    バルブリフタは、5．06-5．74ｍｍの範囲で0．02ｍｍごとに35種類の補給がある。

  18. 選択したバルブリフタを取り付ける。

  19. カムシヤフト部およびシリンダヘツドジヤーナル部にエンジンオイルを塗布する。




  20. 図のようにチェーンを手で保持し、チェーンのペイントマークがカムシヤフトタイミングギアASSYのタイミングマークと一致するように、カムシヤフトおよびカムシヤフトタイミングギヤASSYを取り付ける。




  21. カムシヤフトベアリングキヤツプNo.2のフロントマークおよび数字を確認し、図の順序で数回に分けて均等に締め付ける。

    基準値
    T = 12.7 N・m{ 130 kgf・cm }

    ■ 注 意 ■
    カムシヤフトを水平に沈み込ませる。




  22. 図のようにチェーンを手で保持し、チェーンのペイントマークがカムシヤフトタイミングギヤのタイミングマークと一致するように、カムシヤフトNo.2およびカムシヤフトタイミングギヤを取り付ける。

  23. カムシヤフトNo.2のノックピンとカムシヤフトタイミングギヤのピン溝を合わせる。




  24. フランジボルトを仮付けする。




  25. カムシヤフトベアリングキヤツプNo.1、No.2のフロントマークおよび数字を確認し、図の順序で数回に分けて均等に締め付ける。

    基準値
    T = 25 N・m{ 255 kgf・cm } 　(ベアリングキヤツプNo.1)
    T = 12.7 N・m{ 130 kgf・cm } 　(ベアリングキヤツプNo.2)

    ■ 注 意 ■
    カムシヤフトを水平に沈み込ませる。




  26. カムシヤフトNo.2のサービス用六角部をモンキーレンチで保持し、フランジボルトを締め付ける。

    基準値
    T = 64 N・m{ 653 kgf・cm }

  27. チエーンテンシヨナから六角棒レンチを抜き取る。




  28. クランクシヤフトプーリを正回転させ、クランクシヤフトプーリの切り欠きを0°マークに合わせる。

  29. 各々の合わせマークが一致していることを確認する。

  30. チエーンカバーサービスホールスクリユプラグのボルト先端より、2-3ねじ山にアドヘシブ1324を塗布する。

    ■ 注 意 ■
    ボルトおよびボルト穴を清掃、脱脂する。




  31. ソケットヘキサゴンレンチ8を使用して、チエーンカバーサービスホールスクリユプラグを取り付ける。

    基準値
    T = 15 N・m{ 153 kgf・cm }



6. シリンダヘッド カバーSUB-ASSY取り付け



  1. 図の位置にシールパッキンブラックを塗布し、シリンダヘツドカバーを取り付ける。

    ■ 注 意 ■
    1. 取り付け面を清掃および脱脂する。
    2. シールパッキン塗布後３分以内に取り付ける。
    3. 取り付け後、2時間以内はエンジンを始動させない。




  2. ボルト９本およびナット２個で、シリンダヘツドカバーを取り付ける。

  3. 図の順序で数回に分けてボルトおよびナットを締め付ける。

    基準値
    T = 10 N・m{ 102 kgf・cm }



7. イグニッション コイル NO.1取り付け

  1. ボルトでイグニッションコイルを取り付ける。

    基準値
    T = 9 N・m{ 92 kgf・cm }

  2. コネクタを接続する。

9. ベンチレーション ホース NO.2取り付け

  1. ベンチレーションホースNo.2をベンチレーションホースクリップNo.2で取り付ける。
