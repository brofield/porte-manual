シリンダヘッド ガスケット（1NZ-FE(4WD)）  取り付け

準備品一覧



1. シリンダヘッドセット ボルト点検



  1. ノギスを使用して、ヘッドボルトの首下長さを点検する。

    基準値
    142.8-144.2ｍｍ

    限度
    147.1ｍｍ

    □ 参 考 □
    限度以上の場合はヘッドボルトを新品に交換する。



2. シリンダヘッド ガスケット取り付け



  1. ロットナンバーが上になるようにヘッドガスケットをシリンダブロックに取り付ける。

    ■ 注 意 ■
    1. 取り付け面およびボルト穴を清掃および脱脂する。
    2. 取り付け方法を確認する。
    3. へッドガスケットを傷つけないように取り付ける。



3. シリンダ ヘッドSUB-ASSY取り付け



□ 参 考 □
シリンダへッドボルトは塑性域締め付け法で締め付ける。

  1. シリンダヘッドボルトのねじ部と座面に少量のエンジンオイルを塗布する。

  2. ダブルヘキサゴンレンチ8を使用して、図の順序でシリンダへッドボルトを数回に分けて仮締めした後、規定トルクで締め付ける。

    基準値
    T = 29 N・m{ 296 kgf・cm }




  3. シリンダへッドボルト頭部のエンジンフロント側にペイントマークをつける。

  4. ペイントマークを目安にして、締め付け順序に従いシリンダへッドボルトを90°締め付ける。さらに、締め付け順序に従い90°増し締めする。

  5. 全てのペイントマークが180°締め付けられた位置にあることを確認する。




  6. シリンダへッドボルト締め付け後、図の箇所にシールパッキンブラック(直径4.5-5.5ｍｍ)を塗布する。

    ■ 注 意 ■
    1. 取り付け面を清掃および脱脂する。
    2. シールパッキン塗布後、3分以内にチェーンケースをセットし、15分以内にボルトを締め付ける。
    3. シールパッキンはチェーンダンパにかかるまで塗布すること。



4. カムシャフト取り付け



  1. カムシャフトカム部およびシリンダヘッドジャーナル部にエンジンオイルを塗布する。

  2. カムシャフトタイミングギヤのタイミングマークを上側にして、カムシャフトおよびカムシャフトNo.2をシリンダヘッドにセットする。

  3. カムシャフトベアリングキャップNo.2のフロントマークおよび数字を確認し、図の順序で数回に分けて均等に締め付ける。

    基準値
    T = 13 N・m{ 133 kgf・cm }

    ■ 注 意 ■
    カムシャフトを水平に沈み込ませる。

  4. カムシャフトベアリングキャップNo.1を取り付ける。

    基準値
    T = 23 N・m{ 235 kgf・cm }



5. エンジン ワイヤ接続

  1. エンジンワイヤを接続する。



6. ウォータバイパス パイプ NO.1取り付け

  1. 新品のガスケットを介し、ボルト2本およびナット2個でウォータバイパスパイプNo.1を取り付ける。

    基準値
    T = 9.0 N・m{ 92 kgf・cm }

8. ヒータウォータ ホース インレット A接続

  1. クランプを取り付け、ヒータウォータインレットホースAを接続する。



9. フューエル チューブSUB-ASSY接続

  1. フューエルチューブを接続する。

    ■ 注 意 ■
    1. 作業前にパイプおよびコネクターの接続部に傷および異物の付着がないことを確認する。
    2. 接続語、コネクターとパイプを引っ張り、確実に組み付けられていることを確認する。



10. ウォータ フィラSUB-ASSY取り付け

    基準値

  2. ラジエータホースNo.3およびラジエータリザーブタンクホースを接続する。



11. ラジエータ インレット ホース接続

12. ラジエータ アウトレットホース接続

  1. ラジエータアウトレットホースを接続し、クランプを取り付ける。



13. ユニオン トゥー コネクタチューブ ホース接続

14. オイルレベルゲージ ガイド取り付け

  1. 新品のOリングにエンジンオイルを塗布し、オイルレベルゲージガイドに取り付ける。

  2. ボルトでオイルレベルゲージガイドを取り付ける。

    基準値
    T = 9.0 N・m{ 92 kgf・cm }

  3. オイルレベルゲージをオイルレベルゲージガイドに差し込む。



15. インテーク マニホルド取り付け



  1. 新品のガスケットをインテークマニホールドに取り付ける。

  2. 図の順序でインテークマニホルドを仮締めした後、規定トルクで締め付ける。

    基準値
    T = 30 N・m{ 306 kgf・cm }

  3. インテークマニホールドにワイヤハーネスを接続する。

  4. ボルト2本で、ブレーキブースタ用バキュームチューブを取り付ける。

    基準値
    T = 9.0 N・m{ 92 kgf・cm }



16. スロットル ボデーASSY取り付け

  1. 新品のガスケットをインテークマニホールドに取り付ける。

  2. ボルトおよびナット2個で、スロットルボデーASSYおよびアクセルレータコントロールケーブルブラケットを取り付ける。

    基準値
    T = 30 N・m{ 306 kgf・cm } (インテークマニホールド新品時)
    T = 20 N・m{ 204 kgf・cm } (インテークマニホールド再使用時)

  3. スロットルポジションセンサおよびアイドルスピードコントロールバルブのコネクターを接続する。



17. ウォータバイパス ホース NO.2接続

  1. ウォータバイパスホースNo.2を接続する。



18. ウォータバイパス ホース接続

  1. ウォータバイパスホースを接続する。



19. アクセルレータ コントロールケーブルASSY接続

  1. アクセルレータコントロールケーブルをスロットルボデーASSYに接続する。



20. マニホルドサポート ブラケット取り付け



  1. ボルト3本で、マニホルドサポートブラケットを取り付ける。

    基準値

21. オキシジエンセンサコネクター接続



  1. コネクターを接続する。

  2. ボルトでハーネスステーを取り付ける。

    基準値
    T = 60 N・m{ 612 kgf・cm }



22. エキゾーストパイプASSY FR接続

  1. ノギスを使用して、コンプレッションスプリングの自由長を測定する。

    限度
    40.5ｍｍ

    □ 参 考 □
    限度以下の場合は、スプリングを新品と交換する。




  2. 新品のガスケットを木片およびハンマーを使用して、エキゾーストマニホルドの面一まで徐々に打ち込む。

    ■ 注 意 ■
    1. ガスケットの取り付け方向を間違えない。
    2. 一度はずしたガスケットは再使用しない。
    3. 排気管を組み付けることによって、ガスケットを押し込むことは絶対にしない。

  3. ボルト2本で、エキゾーストマニホルドにエキゾーストパイプASSY　FRを取り付ける。

    基準値
    T = 43 N・m{ 438 kgf・cm }



23. チェーンSUB-ASSY取り付け



  1. 図の位置のマークプレート(黄色)とタイミングマークを合わせて、チェーンを取り付ける。

  2. マークプレートとタイミングマークがはずれないように、クランクシャフトタイミングスプロケット付近でチェーンをひもで縛る。

    ■ 注 意 ■
    チェーンテンショナASSY　No.1、チェーンバイブレーションダンパ No.1およびチェーンテンショナスリッパ取り付け後、必ずひもを取りはずす。

    □ 参 考 □
    歯飛び防止のため行う。



24. チェーン テンショナASSY NO.1取り付け



  1. ロックが解除された状態でプランジャを奥まで押し込む。




  2. プランジャが奥まで押し込まれた状態からストッパプレートを下にさげ、プランジャをロックする。




  3. プランジャがロックされた状態からストッパプレートの穴にφ3mmの棒または六角棒レンチを差し込む。

    □ 参 考 □
    ストッパプレートが完全に下にさがらず、φ3mmの棒または六角棒レンチが差し込めない場合は、プランジャを少し出すことによりストッパプレートが完全に下にさがり、φ3mmの棒または六角棒レンチが差し込みやすくなる。

  4. ボルト2本でチェーンテンショナASSY　No.1を取り付ける。

    基準値
    T = 9.0 N・m{ 92 kgf・cm }



25. チェーンバイブレーション ダンパ NO.1取り付け

  1. ボルト2本でチェーンバイブレーションダンパNo.1を取り付ける。

    基準値

26. チェーンテンショナ スリッパ取り付け

  1. ボルトでチェーンテンショナスリッパを取り付ける。

    基準値
    T = 9.0 N・m{ 92 kgf・cm }

  2. チェーンテンショナASSY　No.1のストッパプレートから六角棒レンチを抜き取る。

  3. ひもを取りはずす。



27. オイルポンプ シール取り付け



  1. SSTを使用して、新品のオイルシールをオイルポンプの端面まで均等に打ち込む。

    SST
    09950-60010   (09951-00250, 09951-00410, 09952-06010)
    09950-70010   (09951-07100)

    ■ 注 意 ■
    オイルシールを斜めに打ち込まない。

  2. オイルシールのリップ部に少量のMPグリースNo.2を塗布する。

    ■ 注 意 ■
    リップ部に切粉および砂等の異物を付着させない。



28. オイル ポンプASSY取り付け



  1. 図の箇所に新品のOリングを取り付ける。



  2. エンジン本体側およびオイルポンプにシールパッキンを塗布する。



    ■ 注 意 ■
    1. 取り付け面を清掃および脱脂する。
    2. シールパッキン塗布後3分以内に取り付け、15分以内にボルトを締め付ける。
    3. 取り付け後、2時間以内はエンジンオイルを注入しないで放置する。
    4. 取り付け後、2時間以内はエンジンを始動させない。




  3. オイルポンプのドライブロータスプライン部をクランクシャフトの2面幅部に合わせ、オイルポンプASSYをクランクシャフトに挿入する。




  4. ボルト15本およびナットで、オイルポンプASSYを取り付ける。

    基準値
    T = 32 N・m{ 326 kgf・cm } (ボルトＡ)(Ｍ8　30mm)
    T = 11 N・m{ 113 kgf・cm } (ボルトB)(Ｍ6　35mm)
    T = 11 N・m{ 113 kgf・cm } (ボルトC)(Ｍ6　20mm)
    T = 24 N・m{ 245 kgf・cm } (ボルトD)
    T = 24 N・m{ 245 kgf・cm } (ボルトE)(Ｍ8　2段ボルト)

    ■ 注 意 ■
    1. オイルポンプASSY取り付け時、チエーン接触しないように注意する。
    2. オイルポンプASSY取り付け後、15分以内にエンジンマウンテイングブラケットRHおよびウオータポンプASSYを取り付ける。

46. ファン ＆ オルタネータ Vベルト調整



  1. ハブナットレンチまたはバーなどを使用して、オルタネータを車両フロント側に押して、ファン　&　オルタネータVベルトの張力を調整する。

    ■ 注 意 ■
    Vベルトを張る際は、オルタネータとエンジンマウンティングインシュレータの間にハブナットレンチまたはバーを差し込み、テンションをかける。オルタネータとカムシャフトタイミングオイルコントロールバルブ(インテークマニホルド側)の間にハブナットレンチまたはバーを差し込むとカムシャフトタイミングオイルコントロールバルブを破損させる恐れがあるため、絶対に行わない。




  2. 調整用ボルトAを締め付けてから、固定用ボルトBを締め付ける。

    基準値
    T = 19 N・m{ 189 kgf・cm } (ボルトA)
    T = 54 N・m{ 551 kgf・cm } (ボルトB)

    5. Vリブドベルトの張力、たわみ量はクランキング2回転後の値を示す。
    6. ベルトテンションゲージを使用する場合は、マスタゲージで精度チェックを行ってから測定する。
    7. ベルト張力は冷間時および上死点で測定する。

51. 冷却液(トヨタ純正スーパーLLC)補充

  1. ラジエータドレーンコツクプラグを閉じて、冷却水をラジエータ注入口よりあふれるまで注入する。[*1]

    □ 参 考 □
    ラジエータインレットホースおよびラジエータアウトレットホースを手で数回圧迫し、ラジエータ注入口の水位が下がる様であれば追加注入する。

  2. ラジエータキャップを締める。

  3. ラジエータリザーブタンクに冷却液を上限まで注入する。

  4. エンジンをサーモスタットが開弁するまで暖機する。

    □ 参 考 □
    暖機中にラジエータインレットホースおよびアウトレットホースを数回圧迫すること。

  5. エンジンを止め、冷却液が冷えるまで待ち、ラジエータキャップをはずして水位を確認する。

  6. 水位が下がっている場合は、[*1]より繰り返す。

  7. 水位が下がらなくなったら、ラジエータリザーバタンクの冷却液を調整する。
