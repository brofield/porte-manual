    1. TASCAN-Active test

      1. Use SST (Tascan) to perform the operation according to the screen display, display the [Active Test] screen and perform an active test.

        SST
        09991-70200

        [System selection screen: TCCS → active test]
        Item name
        		Test content
        		Restrictions
        ECT shift position
        		Can be set in any shift position
        1. Testing is possible at 50km/h or less
        2. 3 ← → 4 When performing gear gears, be sure to do it in idl -on
        3. Be sure to hold the idl ON state for 2 seconds after the gear shifting, and do not perform continuously.
        Lockup SOL
        1. Lockup solenoid ON ← → OFF is possible
        2. ON → 100% duty
        3. OFF → 0% of duty
        1. Testing is possible at 60km/h or more
        2. Only 4th speed test is possible
        Line pressure sol
        1. ON → OFF of line pressure solenoid valve is possible
        2. ON → 100%duty (line press when idle)
        3. OFF → 0%of duty ratio (line pressure at stall)
        		Throttle opening is fully closed (when idle) and testing is possible at a vehicle speed of 0km/h
        Timing sol
        		Timing solenoid ON ← → OFF is possible
        		Throttle opening is fully closed (when idle) and testing is possible at a vehicle speed of 0km/h

        □ Participation □
        The active test of the line pressure SOL is tested using oil pressure gauge.
