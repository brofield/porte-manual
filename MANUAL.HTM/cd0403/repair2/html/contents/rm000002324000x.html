        1. Bluetooth test


          1. Start the diagnostic and display the diagram test menu screen.

          2. Select "BT Hands Free Test" on the diager inspection menu screen and display the BT hands -free test menu screen.

          3. Select "Bluetooth test" on the BT hands -free test menu screen and display the Bluetooth test menu screen.

          4. Select "Version Information" on the Bluetooth test menu screen and display the version information screen.




          5. Check the version information.
            Version information screen display
            display
            		Content
            Integrated ver.
            		It is a version of the whole software of the Bluetooth module, and manages the software in the integrated version
            APL version information display
            		Indicates the version of the APL software
            UPPER STACK version information display
            		Show the UPPER STACK software version
            Lower Stack version information display
            		Indicates the version of the Lower Stack software




        2. Hands -free sound quality setting


          1. Start the diagnostic and display the diagram test menu screen.

          2. Select "BT Hands Free Test" on the diager inspection menu screen and display the BT hands -free test menu screen.

          3. Select "Hands -free sound quality setting" on the BT hands -free inspection menu screen and display the hands -free sound quality setting screen.

          4. Operate the screen switch and adjust the volume.
            Hands -free sound quality setting screen display
            item
            		Display content
            Recipient sound adjustment item
            		Adjust the receiving volume (the voice of the other person you hear) when making hands -free calls
            Dealing volume adjustment item
            		Adjust the dose volume (the voice of the other person you hear) when making hands -free calls

            ■ Attention ■
            The sound quality setting number is not changed.

            □ Participation □
            This screen is drawn by the navigation receiver ASSY.
