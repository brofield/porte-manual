1. Identification code registration

□ Participation □
  1. The additional mode is a mode that registers the registered code and registers the newly received identification code, and is used to add a door control transmitter and electrical key transmitter.In addition, up to 7 registration can be registered.However, the maximum of five for each door control transmitter and electrical key transmitter.(Examined control transmitter 5+electrical key transmitter 2 pieces)
  2. The rewriting mode is a mode that deletes all registered code and registers only the newly received identification code, and is used when replacing the door control transmitter and electrical key transmitter.In addition, up to 7 registration can be registered.However, the maximum of five for each door control transmitter and electrical key transmitter.(Examined control transmitter 5+electrical key transmitter 2 pieces)
  3. The prohibited mode is a mode that offers all registered code and prohibits the smart door lock and wireless allocation.
  4. The confirmation mode is a mode that checks how many types of identification code is registered, and is used to confirm the number of registered code at the time of additional registration.

  1. Make the vehicle in the following state.

    1. The driver's door glass is fully opened.

    2. Open only the driver's seat door.

    3. Make the driver's seat door in Unlock.

    4. The key plate is not inserted into the ignition switch rock cylinder.

  2. After the above work, perform the following work.

    1. Repeat the work of inserting the key plate into the ignition switch rock cylinder and pulling it out twice.

      ■ Attention ■
      Do it within 5 seconds.

    2. The driver's seat door is fully closed → After opening twice, the key plate is inserted into the ignition switch rock cylinder and extracted.

      ■ Attention ■
      Do it within 40 seconds.

    3. The driver's seat door is fully closed → Open twice, the key plate is inserted into the ignition switch rock cylinder, and all the doors are closed.

      ■ Attention ■
      Do it within 40 seconds.

    4. ON ← → Lock of the ignition switch is performed about 1 second for each mode shown in the figure below, and the key plate is extracted after selecting the mode.

      ■ Attention ■
      Do it within 40 seconds.




    5. After 3 seconds, the door control relay automatically performs the door lock lock ← → Unlock, and informs the worker in each response pattern as shown in the figure below.

      □ Participation □
      1. If you select a confirmation mode and prohibited mode, it will end after the response.
      2. The confirmation mode shows an example in which two types of identification code is registered in the figure because the number of cycles is activated as the registered code number.However, if the number of registration code is 0, execute 11 cycles.
      3. If the Ignition Switch is ON ← → Lock 4 or 6 or more times, it will end with no response.




  3. Registering the door control transmitter, electrical key transmitter

    1. When selecting an additional mode and rewriting mode, press the door control transmitter, the lock switch and unlock switch of the electrical key transformer, and then press the switch control transmitter and electrical key transmitter again.

      ■ Attention ■
      1. Press more than 1 second per switch operation.
      2. Submit the key to the door control transmitter and the electrical key transmitter within 45 seconds within 45 seconds.



    2. After the switch of the door control transmitter and electrical key transmitter is turned off, Lock ← → Unlock operation is automatically performed within 5 seconds if the registration of the door control train transmitter and electrical key transformer is completed.If the registration has not been completed twice, registration of the door control transmitter and the electrical key transmitter will be re -registered.

      ■ Attention ■
      After the ignition switch is turned off, Lock ← → Until the operation is automatically performed, the door control transmitter and the electrical key transmitter are not operated.However, if Lock ← → Unlock is not operated after 5 seconds, registration of the door control transmitter and electrical key transmitter will be performed again.




    3. If you continue to register, register the door control transmitter and electrical key transmitter within 45 seconds.

      □ Participation □
      Up to 4 types of identification code can be registered.

  4. End of registration mode (when one is established)

    1. Waiting for registration mode input has passed 45 seconds or more.

    2. Open any door.

    3. Insert the key into the ignition switch rock cylinder.

  5. After the registration work, perform the following work.

    1. Wireless Arrock Control System (Car with Smart Door Lock) Car Inspection

    2. Smart door lock system inspection

      ■ Attention ■
      If you do not operate wireless or smart or do not operate only smart, registration of the identification code again.
