フロントアクスル ハブ LH  取り付け

印刷

準備品一覧



1. フロントアクスルASSY　LH取り付け

  1. ロワアームASSYをフロントアクスルASSY LHに取り付け、キャッスルナットで締め付ける。

    基準値
    T = 98 N・m{ 1000 kgf・cm }

    ■ 注 意 ■
    ねじ部およびテーパ部に油脂類を付着させない。

  2. 新品のクリップを取り付ける。

    ■ 注 意 ■
    キャッスルナット締め付け後のクリップ穴合わせは60°以内の増し締め方向で行う。




  3. アクスルASSYをショックアブソーバASSYに取り付け、車両前方からボルト2本を挿入し、ナット2個を締め付ける。

    基準値
    T = 132 N・m{ 1350 kgf・cm }

    ■ 注 意 ■
    ボルトおよびナットを再使用する場合のみ、ナットのねじ部および座面にエンジンオイルを塗布する。



2. タイロッド エンドSUB-ASSY LH取り付け

  1. タイロッドエンドをステアリングナックルに取り付け、キャッスルナットで締め付ける。

    基準値
    T = 49 N・m{ 500 kgf・cm }

    ■ 注 意 ■
    ねじ部およびテーパー部に油脂類を付着させない。

  2. 新品のコッターピンを取り付ける。

    ■ 注 意 ■
    キャッスルナット締め付け後のコッターピン穴合わせは60°以内の増し締め方向で行う。



3. スタビライザ バー FR取り付け

  1. クッションリテーナNo.1を2個、クッション2個およびスタビライザバーを図の向きになる様に取り付け、ナットで締め付ける。




  2. スタビライザボルトをスパナ(10mm)で固定し、新品のナットを締め付ける。

    基準値
    T = 18 N・m{ 184 kgf・cm }



4. フロントディスク取り付け

  1. ディスクとアクスルハブに合わせマークを付け、ディスクを取りはずす。

    ■ 注 意 ■
    ディスク面に油脂を付着させない。



5. フロントデイスクブレーキキヤリパASSY LH取り付け



  1. ボルト2本でデイスクブレーキキヤリパASSYをステアリングナックルに取り付ける。

    基準値
    T = 88 N・m{ 900 kgf・cm }



6. フロントアクスル シャフト ナット LH取り付け

  1. ソケツトレンチ(30ｍｍ)を使用して、新品のハブナットを取り付ける。

    基準値
    T = 216 N・m{ 2200 kgf・cm }

    □ 参 考 □
    締め付ける際、ブレーキペダルを踏んでおく。

8. スピード センサ FR LH取り付け

  1. ボルトで、スピードセンサワイヤおよびフレキシブルホースをショックアブソーバASSYに取り付ける。

    基準値
    T = 29 N・m{ 300 kgf・cm }




  2. クリツプをショックアブソーバASSYに取り付ける。

    ■ 注 意 ■
    スピードセンサワイヤに負担がかかる状態で取り付けない。




  3. ボルトで、スピードセンサＦＲをステアリングナックルに取り付ける。

    基準値
    T = 8.0 N・m{ 82 kgf・cm }

    ■ 注 意 ■
    1. センサ本体を取り付ける際に、センサ先端部分を傷つけない。
    2. 異物をかみ込ませない。
    3. 取り付ける際に、センサワイヤがねじれないようにする。
