  1. Physical address 280 television camera controller (camera ECU))

    □ Participation □
    1. Logical address 01: communication management
    2. Logical address 5c: Camera unit


    Physical address 280 television camera controller (camera ECU))
    Logical address-Diag code
    		Diagnostic item
    		Diagnostic content
    		treatment
    01-D6 (*1)
    		Master absence
    1. The device that remembered this code had been out of the system at the time of the ACC ON, or the master device was out of memory when this code was memorized.
    1. Inspection of the power supply system and AVC-LAN communication system W/H of the navigation receiver ASSY
    2. Check the power supply system and AVC-LAN communication W/H of the television camera controller
    01-D7 (*2)
    		Connection confirmation abnormality
    1. The device that remembers this code may have been out of the system after the engine started, or the master device was out of the way when this code was memorized.
    1. Inspection of the power supply system and AVC-LAN communication system W/H of the navigation receiver ASSY
    2. Check the power supply system and AVC-LAN communication W/H of the television camera controller
    01-DC (*3)
    		Failure
    1. Sending has failed to the device shown in the auxiliary code (this diag code may not be directly linked to abnormalities).
    1. If the equipment shown in the auxiliary code is also remembered in other devices, check the power supply W/H and communication system W/H of the device indicated by the auxiliary code.
    01-DD (*4)
    		Master reset (instantaneous)
    1. After the engine started, the master equipment has come off the system
    1. Inspection of the power supply system and AVC-LAN communication system W/H of the navigation receiver ASSY
    2. If abnormalities occur frequently, replace the navigation receiver ASSY
    01-DF (*2)
    		Master abnormality
    1. The device with the display has become abnormal, and the master function has been switched to audio equipment.
    2. An abnormality occurred in the communication between the submaster (audio) device and the master
    1. Inspection of the power supply system and AVC-LAN communication system W/H of the navigation receiver ASSY
    01-E0 (*1)
    		Registration completion notification abnormality
    1. "Registration completion notification" command from the master cannot be received
    1. This diagram is an engineering diagram, which is also detected when it is not directly connected to the abnormality.
    01-E2
    		ON/OFF instruction parameter abnormality
    1. There was an abnormality in the command to control ON/OFF from the master device
    1. Replace the navigation receiver ASSY
    01-E3 (*1)
    		Registration request transmission
    1. The "registration request" command was output from the slave device
    2. A "registration request" command was output from the "connection confirmation instruction" command from the submaster device.
    1. This diagram is an engineering diagram, which is also detected when it is not directly connected to the abnormality.
    5C-40
    		Camera video abnormality
    1. There is no synchronous signal from the camera.Or the camera has not been connected
    1. Check the W/H between the television camera ASSY and the television camera controller
    2. Replace television camera ASSY
    5C-41
    		Steering sensor abnormal
    1. Fail flag was sent from the steering sensor
    1. Replace the steering sensor
    5C-42
    		Steering sensor communication line abnormality
    1. There is no signal from the steering sensor
    1. Troubleshooting Flow Characters (The predictable course is not displayed, and "cannot guide" is displayed)

    □ Participation □
    1. *1 may be memorized by the battery status and the power waveform at the start of the engine, even if an abnormality is not found.
    2. *2 remembers that the power of the navigation receiver ASSY is turned off after the engine is started and it has been 210 seconds.
    3. *3 may remember that the IG switch is Start again after 60 seconds after the engine is started.
    4. *4 may remember that the IG switch is Start again after the engine is started.
