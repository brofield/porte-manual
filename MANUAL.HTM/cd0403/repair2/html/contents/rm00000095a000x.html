EBD ABS System Diag Code Inspection / Erase

printing



  1. Diagnostic code output inspection

    1. Diagnostic code read by SST (TASCAN)


      1. Connect the SST to DLC3 in the IG switch off.

        SST
        09991-70200

      2. Turn on the IG switch and follow the Tascan display screen to read the diagnostic cord.

        ■ Attention ■
        If there is an abnormality, delete the memory of the diagnostic cord after repair.



    2. Diagnothesis code read by Warnin Gramp





      1. In the IG switch off, use the SST to shortly use the DLC3 13 (TC) ← → 4 (CG) terminal.

        SST
        09843-18040

        ■ Attention ■
        1. There is no mistake in the connector connection position because it causes a failure.
        2. Perform in the vehicle stopped.




      2. Turn on the IG switch and read the number of flashes of the ABS Warnin Gramp.

        □ Participation □
        1. In the case of normal, 0.25 seconds lit and 0.25 seconds turn off.
        2. If the abnormal code is one, output the same code at 4 seconds, output multiple code at 2.5 seconds, and once round, once a round of 4 seconds.Output.
        3. When outputting multiple code, output in order from the small code number.

      3. Open between 13 (TC) ← → 4 (CG) of DLC3.

        ■ Attention ■
        If there is an abnormality, delete the memory of the diagnostic cord after repair.



    1. Diagnostic code, freeze -frame data eraser

      1. Diagnothescode erasing method (TASCAN)


        1. Connect the SST to DLC3 with IG switch off.

          SST
          09991-70200

        2. Turn on the IG switch and delete the diagnostic code according to the Tascan display screen.



      2. Diagnothescode erasing method (DLC3 short circuit)





        1. In the state of the IG switch OFF, use the SST to shorten the DLC3 13 (TC) ← → 4 (CG) terminal.

          SST
          09843-18040

          ■ Attention ■
          There is no mistake in the connector connection position because it causes a failure.

        2. Turn on the IG switch and step on the brake pedal more than 8 times in 5 seconds.

        3. Confirm that the normal code is output.

          □ Participation □
          If the failure is continuing, it may not be possible to erase it.
