Installation of transmitsutabatsuri (wireless key)

printing


1. Transmitter battery installation



  1. Attach the battery [lithium battery (CR2016)] as shown in the figure.

    ■ Attention ■
    1. Put the battery plus upwards.
    2. Do not deform the terminal part.

  2. Attach the cover.

    ■ Attention ■
    If the claws of the cover are not ensured, the battery [lithium battery (CR2016)] may cause poor contact between the terminal.



2. Door control transmitter module set installation

  1. Return the module to the transmitter housing case, hold the transmitter housing cover and put it on.

  2. Confirm that the LED lights up when one of the switches of the door control transmitter module set is pressed on the module.
