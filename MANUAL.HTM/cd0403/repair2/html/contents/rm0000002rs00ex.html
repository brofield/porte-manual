    		Diagnostic item
    		Diagnostic content
    		treatment
    		remarks
    01 -D5
    		Registered equipment dropout
    1. The device shown in the auxiliary code may have been out of the system or out of the system when ACC turned on.
    2. The communication with the equipment shown in the auxiliary code has not been secured when starting the engine
    		Inspection of the power supply system and AVC-LAN communication system W/H of devices indicated by the auxiliary code
    		Even if an abnormality is not found, this diager may be remembered due to the battery status or the power waveform at the start of the engine.
    01 -D6
    		Master absence
    		The device that remembered this diag was out of the system at the time of ACC ON, or the master device was off when this diagram was memorized.
    		Inspection of the power supply system and AVC-LAN communication system W/H of the navigation receiver ASSY
    		Even if an abnormality is not discovered, this diagram may be remembered due to the battery status and the power waveform at the start of the engine.
    01 -D7
    		Connection confirmation abnormality
    		The device that remembers this diager has missed the system after the engine started, or the master device has come off when this diager is memorized.
    		Inspection of the power supply system and AVC-LAN communication system W/H of the navigation receiver ASSY
    		This diager remembers that the power of the navigation receiver ASSY is turned off after the engine is started and it has been 210 seconds.
    01 -D8
    		Connection confirmation without answer
    		The device shown in the auxiliary code may have been out of the system or off the system after the engine started.
    		Inspection of the power supply system and AVC-LAN communication system W/H of devices indicated by the auxiliary code
    		This diary remembers that the power of the device indicated by the auxiliary code is turned off after the engine is started, and that it has passed 180 seconds.
    01 -D9
    		Last mode abnormality
    		The device that was working (sound, video out) before the engine was turned off has been off or off the system at the time of ACC.
    		Inspection of the power supply system and AVC-LAN communication system W/H of devices indicated by the auxiliary code
    		Even if an abnormality is not discovered, this diagram may be remembered due to the battery status and the power waveform at the start of the engine.
    01
    		ON/OFF instruction without answer
    		The device shown in the auxiliary code did not respond when the sound / video mode switched operation (detected when the screen or audio did not switch due to button operation etc.)
    1. Inspection of the power supply system and AVC-LAN communication system W/H of devices indicated by the auxiliary code
    2. If the abnormality recurs, replace the equipment indicated by the auxiliary code
    		-
    01 -DB
    		Mode status abnormality
    		Detect double sound of devices indicated by auxiliary code
    		Inspection of the power supply system and AVC-LAN communication system W/H of devices indicated by the auxiliary code
    		Even if an abnormality is not discovered, this diagram may be remembered due to the battery status and the power waveform at the start of the engine.
    01 -DC
    		Failure
    		It has failed to send to the device shown in the auxiliary code (this diag code may not be directly linked to abnormalities).
    1. If the equipment shown in the auxiliary code is also remembered in other devices, check the power supply system and AVC-LAN communication W/H of the device indicated by the auxiliary code.
    2. If this diag is not recorded on other devices, delete the diag code and see for a while.
    		This diary may remember that the IG switch is Start again 60 seconds after the engine is started.
    01 -D
    		Master reset (instantaneous)
    		After the engine started, the master equipment has come off the system
    1. Inspection of the power supply system and AVC-LAN communication system W/H of the navigation receiver ASSY
    2. If abnormalities occur frequently, replace the navigation receiver ASSY
    		This diary may remember that the IG switch is Start again after the engine is started.
    In 01-
    		Sleave reset (instantaneous)
    		After the engine started, the slave device has come off the system
    		Inspection of the power supply system and AVC-LAN communication system W/H of devices indicated by the auxiliary code
    		This diary may remember that the IG switch is Start again after the engine is started.
    01 -DF
    		Master abnormality
    1. The device with the display has become abnormal, and the master function has been switched to audio equipment.
    2. An abnormality occurred in the communication between the submaster (audio) device and the master
    		Inspection of the power supply system and AVC-LAN communication system W/H of the navigation receiver ASSY
    		This diager remembers that the power of the navigation receiver ASSY is turned off after the engine is started and it has been 210 seconds.
    01 -Picture 0
    		Registration completion notification abnormality
    		"Registration completion notification" command from the master cannot be received
    		This diagram is an engineering diagram, which is also detected when it is not directly connected to the abnormality.
    		Even if an abnormality is not found, this diager may be remembered due to the battery status or the power waveform at the start of the engine.
    01 -Picture 1
    		Voice processing device on abnormality

    		Inspection of the power supply system and AVC-LAN communication system W/H of the navigation receiver ASSY
