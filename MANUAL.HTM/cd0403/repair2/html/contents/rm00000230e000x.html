1. Display color bar inspection

□ Participation □
Diagnostic screen transitionreference.

  1. Diagnostic activation

    1. Start the diagnostic and display the diagram test menu screen.(The point isreference)

    2. Select "Display Inspection" on the diagram inspection menu screen and display the display inspection menu screen.

    3. Select "Color Bar Test" on the display inspection menu screen and display the display color bar inspection screen.




  2. Display color bar inspection

    1. Confirm that the configuration color and the display color match.

      □ Participation □
      1. If you select black, red, green, blue, white or horizontal stripes, the selected colors or stripes will be displayed throughout the screen.
      2. Touch the screen and return.
      3. If the image quality is adjusted, it will be displayed according to the brightness of the set.
      4. If the display color bar inspection result is abnormal, the navigation receiver ASSY is bad.

    2. Confirm that there is no difference compared to the display color of the navigation color bar inspection.

      □ Participation □
      If the navigation color bar inspection result is normal and the display color bar inspection result is abnormal, the navigation receiver ASSY is bad.

  2. Vehicle signal & sensor inspection

    Vehicle signal & sensor inspection screen display
    item
    		Display method
    		Signal input terminal
    REV signal state
    		Display the REV signal status with "ON"/"OFF" (when IG ON)
    		Navigation receiver ASSY REV (F5) terminal
    SPD signal state
    		Display the SPD signal status (Cumulative number of input pulse after this screen and vehicle speed [km/h])
    		Navigation receiver ASSY SPD (F3) terminal

    		Displays the output status of the gyro sensor (Voltage value [V] and the amount of change in the direction after displaying this screen [degree])

    		Reset the display item of the relative orientation of the gyro sensor and the number of pulses of the SPD signal and the display contents are "0".
    		-

    □ Participation □
    *1: Display when the inclined angle sensor data becomes effective.(Included angle sensor abnormal/sensor data learning is not displayed when learning is not completed)


    1. Confirm the status of the vehicle signal (REV, SPD) incorporated into the navigation receiver ASSY.

      □ Participation □
      1. The vehicle signal is updated every second.
      2. If the REV signal state and the SPD signal condition are abnormal, the vehicle wire harness or the defective of the connector.(The point isreference)

    2. Check the output status of the gyro sensor.

      Reference value
      Gyro sensor voltage value: 2.1-2.9V

      □ Participation □
      If the output condition of the gyro senser is abnormal, the navigation receiver ASSY is bad.

    3. Drive on a slope, etc. to check the output status of the inclined angle sensor.

      Reference value
      Changes according to the inclination of the vehicle

      □ Participation □
      1. If the inclined angle sensor output condition is abnormal, the navigation receiver ASSY is bad.
      2. If the inclined angle sensor output status is not displayed, drive on a flat road for about 10-20 minutes and check the display again.If it is not displayed, the navigation receiver ASSY is bad.

  3. Smart test

    1. Select the "Smart test" on the HDD information screen and display the Smart test result.




    2. Check the Smart test result.

      standard
      "OK" displays

      □ Participation □
      1. It takes about 90 seconds to complete the test.
      2. If the SMART test result is "NG", the navigation receiver ASSY is bad.
      3. Due to low temperature or high temperature, it may be displayed as "NG" even if the HDD cannot be read or written.Therefore, if the code 13, 14, 23, or 24 are displayed in the HDD information, perform the code list treatment and perform the Smart test again.



9. Navigation guidance audio setting confirmation (guidance voice does not sound)

  1. Setting inspection

    1. Set the volume to the maximum volume in the audio settings on the menu screen.

      standard
      "I will guide you at this volume"

      □ Participation □
      In the following cases, the system does not provide voice guidance even when the system is normal.
      1. When the destination is not set.(Voices such as prefectural areas guidance are output)
      2. When it is not running on the route (there is no remaining distance display at the bottom left of the local screen).(Voices such as prefectural areas guidance are output)
      3. When "Information in other mode" is not set.(Not voice guidance is not provided other than map screen)
