    1. Freeze -frame data when abnormal diagram code occurs


      1. If the freeze -frame data when an abnormal diag code occurs, [*freeze data] is displayed on the corresponding dial code on the display screen of the diagnothescode.

        □ Participation □
        1. When a freeze -frame data when an abnormal code occurs, the freeze -frame data at the time of system operation is erased.

    2. Freeze frame data when system operation


      1. The freeze -frame data at the time of system operation is updated to new data for each system operation.

      2. Finally, the IG switch OFF → ON ON is remembered up to 31 times from the end of the system.

      3. Using the number of IG switches from the data freeze, you can check the following calculation formula as a guide to see if the display data is the data when the customer is pointed out.([Number of IG switch ON per day] × [Number of days from the occurrence occurrence] = [Ig switch ON times from data freeze])
