Trouble shooting is performed according to the following diagnosis procedure.Here, only the basic procedure is shown.For more information, the most effective method for each circuit is described in the section of each system.Before starting troubleshooting the circuit, first check the trouble shooting procedure.




Step 1Vehicle vehicle entry






    Step 2Customer analysis

    1. Ask the customer about the condition and situation when a problem occurs.






    Step 3Confirmation of defective phenomena and diag code (including freeze frame data) inspection

    1. Check the battery voltage using SST (Toyota Electrical Tester).

      09082-00030

      10-14V (when the engine is stopped)

    2. Inspection of places that can be visually confirmed, such as running out of fuse, wire harness disconnection, short circuit, and poor connectors.

    3. Warm the engine enough.

    4. Confirm the problem of the problem and state the phenomenon and state, and check the diag code according to the chart.
