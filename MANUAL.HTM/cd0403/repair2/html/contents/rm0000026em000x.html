  2. パーキングブレーキ ケーブルASSY No.2取り付け

    1. ボルトで、パーキングブレーキ ケーブルASSY No.2をバッキングプレートに取り付ける。

      基準値
      T = 8.0 N・m{ 82 kgf・cm }

    2. 図のように、ボルト4本で、パーキングブレーキ ケーブルASSY No.2の各ケーブルブラケットを取り付ける。

      基準値
      T = 6.0 N・m{ 61 kgf・cm }



2. パーキングブレーキ ケーブルASSY NO.3接続

  1. パーキングブレーキケーブルASSY　No.3をパーキングブレーキケーブルイコライザに接続する。

  1. パーキングブレーキケーブルASSY　No.2をパーキングブレーキケーブルイコライザに接続する。

  1. バッキングプレートとリヤブレーキシューASSYとの接触部に、ノンメルトルブを塗布する。



5. ブレーキシュー取り付け（LH側）

  1. リヤ側

    1. ニードルノーズプライヤを使用して、パーキングブレーキシューレバーLHをパーキングブレーキケーブルASSY　No.3に取り付ける。




    2. SSTを使用して、ブレーキシュー(リヤ)をリヤブレーキシューホールドダウンスプリングカップ、リヤブレーキシューホールドダウンスプリングおよびリヤブレーキシューホールドダウンスプリングピンで取り付ける。

      SST
      09718-00010  

    3. リヤブレーキシューリターンスプリングをブレーキシュー(リヤ側)に取り付ける。

  2. フロント側

    1. パーキングブレーキシューストラットセットLHをブレーキシュー(フロント側)に取り付ける。

    2. ブレーキシュー(フロント側)をリヤブレーキシューリターンスプリングに接続する。




    3. SSTを使用し、リヤブレーキシューホールドダウンスプリングカップ、リヤブレーキシューホールドダウンスプリングおよびリヤブレーキシューホールドダウンスプリングピンで、ブレーキシュー(フロント側)を取り付ける。

      SST
      09718-00010  

    4. SSTを使用し、テンションスプリングをブレーキシュー(リヤ)に取り付ける。

      SST
      09703-30010

7. リヤドラムブレーキ取り付け点検




  1. 各部品が正常に取り付けられていることを点検する。

    ■ 注 意 ■
    1. ホイールシリンダブーツに傷がなく、グリース等の油脂類が付着していないことを確認する。
    2. ブレ－キシューASSYのライニング表面に油分および水分等が付着していないことを確認する。



8. リヤブレーキ ドラム取り付け


9. ブレーキドラムとライニングのすき間調整（LH側）



  1. バッキングプレートのサービスホールからマイナスドライバーでアジャスターを回して拡張させ、ブレーキシューASSYをリヤブレーキドラムに接触させる。




  2. マイナスドライバーでリヤブレーキオートマチックアジャストレバーLHを押しながらもう1本のマイナスドライバーでアジャスターを回して収縮させる。

  3. ブレーキシューASSYがリヤブレーキドラムに接触しなくなった状態から更にアジャスターを1/2回転させて収縮させる。

    基準値
    T = 36.8 N・m{ 375 kgf・cm }

  3. 新品のクリップ2個を取り付ける。



25. フロントシートASSY RH取り付け

  1. シートトラックアジャスティングハンドルを操作してフロントシートASSYをスライドさせ、フロントモーストにする。

    ■ 注 意 ■
    ロックしている事を確認する。

  2. シートASSYを車両に乗せ、フロントシートレッグASSY　RH部のピンと車両側の穴を合わせる。

    ■ 注 意 ■
    シートトラックアジャスティングハンドルを持つとスライド同期位置がずれる。

  3. 車両側ワイヤハーネスとシート側ワイヤハーネスを接続させる。

  4. ボルト2本でシートASSYのリヤ側を仮締めする。

  5. シートトラックアジャスティングハンドルを操作してフロントシートASSYをスライドさせ、リヤモーストにする。

    ■ 注 意 ■
    ロックしている事を確認する。

  6. ボルト2本でシートASSYのフロント側を仮締めする。

  7. シートトラックアジャスティングハンドルを操作してフロントシートASSYをスライドさせ、フロントモーストにする。

    ■ 注 意 ■
    ロックしている事を確認する。

  8. シートASSYのリヤ側のボルト2本ををインナ→アウタの順に締め付ける。

    基準値
    T = 37 N・m{ 377 kgf・cm }

  9. シートトラックアジャスティングハンドルを操作してフロントシートASSYをスライドさせ、リヤモーストにする。

  10. シートASSYのフロント側のボルト2本をインナ→アウタの順に締め付ける。

    基準値
    T = 37 N・m{ 377 kgf・cm }

  11. フロントシートレッグカバーのツメをかん合させる。



26. スライドアジャスターロック同期点検

  1. スライド操作したとき、左右のアジャスターが同時にロックすることを点検する。

    □ 参 考 □
    スライドの同期ずれがある場合は、フロントシート固定ボルトをゆるめて調整する。



27. バッテリマイナスターミナル接続
基準値
T = 5.4 N・m{ 55 kgf・cm }


28. パーキングブレーキペダル踏みしろ点検

  1. パーキングブレーキペダルをゆっくり踏み、作動音を数える。

    基準値
    7-10ノッチ[操作力294N{30kgf} ]



29. パーキングブレーキペダル踏みしろ調整



  1. パーキングブレーキペダルを一度強く踏む。

  2. パーキングブレーキペダルを再度踏み込み、ロックを解除する。

  3. パーキングブレーキペダル部のロックナットをゆるめ、踏みしろが基準値になるようアジャスティングナットを回して調整する。

    基準値
    7-10ノッチ[操作力294N{30kgf} ]

  4. アジャスティングナットを固定し、ロックナットを締め付ける。

    基準値
    T = 5.4 N・m{ 55 kgf・cm }

  5. パーキングブレーキペダルを1回以上操作した後、踏みしろを確認する。

  6. パーキングブレーキの引きずりがないことを確認する。

  7. パーキングブレーキペダルを操作したとき、ブレーキインジゲータランプが点灯することを確認する。

    基準
    踏み始めてから1ノッチ目で確実にランプが点灯すること
