29. Parking brake pedal stepping adjustment



  1. Press the parking brake pedal once strongly.

  2. Park the parking brake pedal again and unlock.

  3. Loose the lock nut in the parking brake pedal part, and adjust the adjustment nut so that stepping is a reference value.

    Reference value
    7-10 Notch [Operation 294N {30kgf}]

  4. Fix the adjustment nut and tighten the rock nut.

    Reference value
    T = 5.4 n ・ m {55 kgf / cm}

  5. After operating the parking brake pedal at least once, check for stepping.

  6. Make sure there is no drag of the parking brake.
