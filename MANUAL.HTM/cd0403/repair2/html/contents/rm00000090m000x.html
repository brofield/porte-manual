ウォータポンプASSY  取り付け

印刷

準備品一覧



1. ウォータ ポンプASSY取り付け



  1. 新品のガスケットを介して、ボルト３本およびナット２個でウォータポンプを取り付ける。

    基準値:
    T＝11N·m｛113kgf·cm｝



2. ウォータポンプ プーリ取り付け

  1. ＳＳＴを使用して、ウォータポンププーリを固定する。




  2. ボルト３本を締め付ける。

    SST
    09960-10010   (09962-01000, 09963-00700)

    基準値:
    T＝15N·m｛153kgf·cm｝



3. エンジンマウンティング インシュレータSUB-ASSY RH取り付け



  1. ボルト５本およびナットで、エンジンマウンティングインシュレータＲＨを取り付ける。

    基準値:
    T＝45N·m｛459kgf·cm｝（ボルトＡ）
    T＝52N·m｛530kgf·cm｝（ボルトＢ、ナット）



4. オルタネータASSY取り付け



  1. 固定用ボルトBでオルタネータを仮付けする。




  2. ボルトAおよびナットでファンベルトアジャスティングバーを仮付け後、オルタネータをシリンダブロック側に寄せてナットを本締めする。

    基準値
    T = 11 N・m{ 112 kgf・cm }

  3. コネクタおよびワイヤハーネスクランプを取り付ける。

  4. ナットで+B端子ターミナルを取り付ける。

    基準値
    T = 9.8 N・m{ 100 kgf・cm }

  5. ターミナルキャップを取り付ける。



5. ファン ＆ オルタネータ Vベルト取り付け

  1. Vベルトを各プーリに仮付けする。

    ■ 注 意 ■
    1. ベルトと各プーリに、オイル等の液体及び異物等の付着無きこと。
    2. ベルトのリブ山ずれの無きこと。



6. Vベルト張力·たわみ量点検



  1. ハブナットレンチまたはバーなどを使用して、オルタネータを車両フロント側に引いて、フアン　&　オルタネータVベルトの張力を調整する。

    ■ 注 意 ■
    Vベルトを張る際は、オルタネータとエンジンマウンテイングインシユレータの間にハブナットレンチまたはバーを差し込み、テンションをかける。オルタネータとカムシャフトタイミングオイルコントロールバルブ(インテークマニホルド側)の間にハブナットレンチまたはバーを差し込むとカムシヤフトタイミングオイルコントロールバルブを破損させる恐れがあるため、絶対に行わない。




  2. 調整用ボルトAを締め付けてから、固定用ボルトBを締め付ける。

    基準値
    T = 19 N・m{ 194 kgf・cm } (ボルトA)
    T = 54 N・m{ 551 kgf・cm } (ボルトB)



7. シリンダヘッド カバー NO.2取り付け



  1. ナットA2個を締め付け後、ナットB2個でシリンダヘッドカバーNo.2を取り付ける。

    基準値
    T = 7.0 N・m{ 71 kgf・cm }



8. 冷却液(トヨタ純正スーパーLLC)補充

  1. ラジエータドレーンコツクプラグを閉じて、冷却水をラジエータ注入口よりあふれるまで注入する。［＊１］

    □ 参 考 □
    ラジエータインレツトホースおよびラジエータアウトレツトホースを手で数回圧迫し、ラジエータ注入口の水位が下がる様であれば追加注入する。

  2. ラジエータキヤツプを締める。

  3. ラジエータリザーブタンクに冷却液を上限まで注入する。

  4. エンジンをサーモスタツトが開弁するまで暖機する。

    □ 参 考 □
    暖機中にラジエータインレツトホースおよびアウトレツトホースを数回圧迫すること。

  5. エンジンを止め、冷却液が冷えるまで待ち、ラジエータキヤツプをはずして水位を確認する。

  6. 水位が下がっている場合は、［＊１］より繰り返す。

  7. 水位が下がらなくなったら、ラジエータリザーバタンクの冷却液を調整する。



9. 冷却液(トヨタ純正スーパーLLC)漏れ点検

  1. 冷却液を満水にしてテスターを取り付ける。

  2. 137ｋＰａ｛1.4ｋｇｆ/ｃｍ２｝の圧力をかけ、各部に水漏れがないことを確認する。



10. エンジンアンダ カバー RH取り付け

  1. スクリュー2個およびボルト2本で、エンジンアンダーカバーRHを取り付ける。

    基準値
    T = 5.0 N・m{ 50 kgf・cm }

  2. ナットを、締め付ける。

    □ 参 考 □
    ナットは、LH側と共締めになっている。

    基準値
    T = 5.0 N・m{ 50 kgf・cm }
