G-Book system function description

printing




  1. AVC-LAN communication system and AVC-LAN Plus communication system


    1. AVC-LAN


      1. The G-Book system communicates between devices by AVC-LAN.

      2. AVC-LAN master is the navigation receiver ASSY.

      3. Navigation receiver ASSY contains the resistance required for communication transmission (TX+← → TX-terminal 60-80Ω).

      4. In the AVC-LAN circuit, if a disconnected or short occurs, the G-Book system will not work properly due to the interruption of communication.

    2. AVC-Lan Plus


      1. The G-Book system communicates between devices by AVC-Lan Plus.

      2. In the AVC-Lan Plus circuit, if a disconnected or shorts occur, the G-Book system will not work properly due to interruption of communication.



    1. Diagnostic function

      1. The "Wide Multi-AV Station" system has a diagnostic function, and can be connected to the device connected by AVC-LAN and AVC-LAN PLUS and perform a failure diagnosis.(The test result is displayed on the display part)

      2. For devices that make up AVC-LAN and AVC-LAN Plus, the numbers (AVC-LAN connection devices) or 2 digits (AVC-LAN Plus connection equipment), called the device code (physical address), are noted in the hexadis.) Is set.

      3. Each function (unit) that makes up the interior of the device that makes AVC-LAN and AVC-LAN Plus is set with a two-digit number called a logical address (indicated in the hexadis).
