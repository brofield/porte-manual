      Use SST (Tascan) to detect a connector between the ECU-speed sensor and the wire harness instantaneous (disconnection) on the ECU data monitor.


      1. Connect SST (Tascan) to DLC3 by turning off the IG switch.

        SST
        09991-70200

      2. IG Switch ON, follow the screen display of the SST (Tascan), display the ECU data monitor, and select the item of the site to be instantaneously monitored.

        □ Participation □
        After the IG switch ON, for 3 seconds (when checking the lamp), instantaneous (disconnection) cannot be detected.

    3. Active test inspection


      1. Set the SST (Tascan) to DLC3 by turning off the IG switch.

        SST
        09991-70200

      2. The IG switch is turned on, and the active test is displayed according to the screen display of the SST (Tascan).

      3. Select the data monitor item to be confirmed during the active test and press the execution key.

      4. Select the active test item to be implemented, press the execution key, operate the specified key of the screen display, and execute the ON ← → OFF switching of each test.

        ■ Attention ■
        1. The motor is automatically stopped in about 5 seconds to protect the motor, but does not intermittently perform a few seconds.
        2. The driving of each solenoid is automatically prohibited in about 2 seconds to protect the solenoid, and is permitted at constant intervals.
        3. Do not step on the brake while only the decompression valve is turned on alone.
        4. Except by moving the holding valve of each wheel and the decompression valve, the solenoid is not driven at the same time.

        □ Participation □
        In the case of a solenoid active test, two solenoids can be driven at the same time as entering the 'All Solenoid drive selection' on the solenoid selection screen.
