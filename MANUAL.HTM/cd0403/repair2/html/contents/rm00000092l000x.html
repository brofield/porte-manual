Electronic control type automatotutran spicules Axle [ECT] System (U441E) Function description

printing



  1. Diagnostic function

    1. When an abnormality occurs in the ECT system, light the check engine warnin gland.

      ■ Attention ■
      If the abnormal item returns normally, it will not blink.

    2. The diagnosis result in the event of an abnormality is remembered by the engine control compilation.

      ■ Attention ■
      Since the computer's memory data is deleted, EFI fuse and battery terminals do not remove until the inspection is completed.



    1. Fail Safe function

      1. If one or disconnected in the solenoid valve system, the computer turns other solenoids on or off.If all solenoids are failed, only a functional hydraulic circuit will be used, and it will be a manual shift.(In the case of disconnection and shorts, the computer stops the power of the failed solenoid.)
