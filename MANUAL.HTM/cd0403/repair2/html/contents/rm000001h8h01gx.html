DTC P0705/C2 T/M Control SW Defective

printing

List of preparation


Function explanation


The shift position is detected by inputting a signal from the neutral start switch ass in the engine control computer.


DTC No.
SAE/TCCS
DTC detection conditions
  1. Diagnostic condition
  2. Abnormal state
  3. Abnormal period
  4. others
Inspection site
P0705/C2*
  1. IG ON, battery voltage 10.5V or more
  2. (a) 2 or more signals of NSW, R, D, 2, and L positions are ON at the same time
    (b) NSW, R, D, 2, L position all signals are off
  3. (a) 2 seconds or more
    (b) 60 seconds or more
  4. 2 trips
  1. Neutral Start Switch Ass
  2. Wire harness and connector
  3. Engine control computer

□ Participation □
*: The C2 (lamp code) indicates that the ten of the conventional two -digit code is 12 times and the first place flashes twice.
