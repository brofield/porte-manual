DTC P2109/B9 Throttle flow rate

printing

List of preparation


Circuit description


  1. エンジンコントロールコンピュータはアイドリング時のスロットルバルブ開度（ISC流量）を制御している。スロットルバルブ にデポジットが付着した場合、ISC流量が低下し、エンストまたはアイドル不安定につながる恐れがあるため、ISC学習および フィードバックによりアイドリングに必要なISC流量を確保している。エンジンコントロールコンピュータはISC流量学習値が制御限界に近づくとDTC を出力する。


DTC.
Tsu CS
P2109/B9
DTC detection conditions
Diagnostic condition
When all the following conditions are all available, one trip is established.
  1. 水温45℃以下で始動し、その後暖機されISC学習条件が成立、またはIG ONが1時間以上経過し、エンジンが暖機されISC学習条件が成立
  2. There is a running history of 30km/h or more.
  3. Intake airflow meter SUB-ASSY is normal
  4. Attotive pressure is 85kpa ​​or more (altitude 1400m or less)
Abnormal state
ISC流量学習値が通常の3倍程度かつ、アイドリング時の吸入空気量が正常（通常の1.5倍以内）
Abnormal period
10 seconds or more in a row
Number of trips
5トリップ
Method of detection
Att visit once in one trip.(Do not detect after the judgment.)
Sensor used for detection
  1. E.F.I. Throttle position sensor
  2. Intake airflow meter SUB-ASSY
Inspection site
Throttle body ASSY


Inspection procedure


□ Participation □
  1. SST(TaSCAN) を使用して、フリーズフレームデータを読み取る。フリーズフレームデータとして、不具合発生時のエンジン稼動状態の一部が記録されており、それらの情報がトラブルシュートを行う際に役立つ。
  1. Since this DTC does not remember the pending code, it takes time to perform the confirmation work after repair with the presence or absence of DTC output.When performing a confirmation work after repair, it can be performed in a short time by comparing the ISC flow rate learning value remembered in the freeze frame data and the ISC flow rate learning value after repair.
