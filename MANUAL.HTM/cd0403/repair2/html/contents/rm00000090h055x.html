Exhaust pipe ASSY installation

printing

List of preparation



1. Exhaust pipe ASSY FR installation



  1. Use caliper to measure the free length of the compression spring.

    limit
    40.5mm

    □ Participation □
    In the case of below the limit, replace the spring with a new one.




  2. The new gasket is gradually driven to the exhaust manifold using a wood piece and a hammer.
    Illustration instructions
    *1
    		Fragment
    *2
    		gasket
    *3
    		Exhaust manifold
    *4
    		Exhaust pipe ASSY FR

    		Front of a vehicle

    ■ Attention ■
    1. The gasket installation direction is not wrong.
    2. The gasket that has been removed once is not reused.
    3. Never push the gasket by attaching the exhaust pipe without pushing it up by hand.

  3. With two exhaust pipe support, attach the exhaust pipe ASSY FR.

  4. With two bolts and two compression springs, the exhaust pipe ASSY FR is attached to the exhaust manifold.

    Reference value
    T = 43 n ・ m {438 kgf / cm}



2. Exploist tail pipe ASSY installation

  1. Attach the exhaust tail pipe ASSY with two bolts via a new gasket.

    Reference value
    T = 19 N ・ M {194 kgf / CM}



3. Oxygen sensor installation



  1. Use SST to attach the oxygen sensor.
    Illustration instructions

    		Valid
    (30mm)

    SST

    Reference value
    T = 44 N ・ M {449 kgf / cm}
    T = 39 n ・ m {398 kgf / cm} (when using SST+torque wrench L26cm)

    □ Participation □
    1. The standard value when using SST+torque wrench L26cm is the reference value of SST and torque wrench on a straight line.
    2. If you use a different torque wrench or a different angle, you need to find a reference value.(The point isreference)

  2. Connect the clamp and draw the harness inside the vehicle.

  3. Connect an oxygen sensor connector.



4. Installation panel finish panel LWR CTR installation



  1. Attach the instrument cup holder with three screws.

  2. Connect the connector of the power outlet socket.

  3. Combine the eight claws and attach the instrument panel finish panel LWR CTR with two clips <c>.
