エキゾーストパイプASSY  取り付け

印刷

準備品一覧



1. エキゾーストパイプASSY FR取り付け



  1. ノギスを使用して、コンプレッションスプリングの自由長を測定する。

    限度
    40.5mm

    □ 参 考 □
    限度以下の場合は、スプリングを新品と交換する。




  2. 新品のガスケットを木片およびハンマーを使用して、エキゾーストマニホルドの面一まで徐々に打ち込む。
    イラスト内指示文字 
    *1
    		木片
    *2
    		ガスケット
    *3
    		エキゾーストマニホルド
    *4
    		エキゾーストパイプASSY FR

    		車両前方

    ■ 注 意 ■
    1. ガスケットの取り付け方向を間違えない。
    2. 一度はずしたガスケットは再使用しない。
    3. 手で十分押し込まず、エキゾーストパイプを取り付けることによって、ガスケットを押し込む方法は絶対にしない。

  3. エキゾーストパイプサポート2個で、エキゾーストパイプASSY FRを取り付ける。

  4. ボルト２本およびコンプレッションスプリング2個で、エキゾーストマニホルドにエキゾーストパイプASSY FRを取り付ける。

    基準値
    T = 43 N・m{ 438 kgf・cm }



2. エキゾースト テール パイプASSY取り付け

  1. 新品のガスケットを介して、ボルト2本でエキゾースト テール パイプASSYを取り付ける。

    基準値
    T = 19 N・m{ 194 kgf・cm }



3. オキシジェン センサ取り付け



  1. SSTを使用して、オキシジェン センサを取り付ける。
    イラスト内指示文字 
    *a
    		有効長
    (30mm)

    SST
    09224-00010  
    基準値
    T = 44 N・m{ 449 kgf・cm }
    T = 39 N・m{ 398 kgf・cm } (SST+トルクレンチL26cm使用時)

    □ 参 考 □
    1. SST+トルクレンチL26cm使用時の基準値はSSTとトルクレンチを一直線上で使用した基準値である。
    2. トルクレンチまたは角度を変えて使用する場合は基準値を求める必要がある。(要領は参照)

  2. クランプを接続し、ハーネスを車両内側に引き入れる。

  3. オキシジェンセンサのコネクターを接続する。



4. インストルメントパネルフィニッシュ パネル LWR CTR取り付け



  1. スクリュ3本でインストルメントカップホルダを取り付ける。

  2. パワーアウトレットソケットのコネクタを接続する。

  3. ツメ8箇所のかん合を合わせ、クリップ<ｃ>2個でインストルメントパネルフィニッシュパネル LWR CTRを取り付ける。



5. 排気ガス漏れ点検
