バックドア  調整

印刷


1. バックドア パネルSUB-ASSY点検


基準値




  1. 基準値内であることを確認する。



2. バックドア パネルSUB-ASSY調整



  1. バツクドアパネル上端上下および上端左右方向の調整は、バツクドアヒンジ取り付けボルトを緩めてから行う。

    ■ 注 意 ■
    センターリングボルトを使用しているため、調整は補給されているボルトに交換してから行う。

    基準値
    T = 19.0 N・m{ 200 kgf・cm }




  2. バツクパネル下端左右方向およびバツクドアロツクストライカASSYの調整をする場合は、トルクスソケットレンチ(T40)を使用してストライカ取り付けボルトをストライカが動く程度まで緩め、ブラスバーを使用して、ストライカを軽くたたいて調整する。

    基準値
    T = 23.0 N・m{ 240 kgf・cm }
