Back guide monitor system function description

printing




  1. Communication system ("Wide Multi -AV Station" equipped car)


    1. The back guide monitor system of the “Wide Multi-AV Station” performs communication between devices by AVC-LAN.

      □ Participation □
      In the "parallel guide mode", it is determined by vehicle angle data sent from the navigation computer through AVC-LAN (calculated by integrating a navigation gyro with navigation gyro with a navigation computer).

    2. AVC-LAN master is the navigation receiver ASSY.

    3. Navigation receiver ASSY contains the resistance required for communication transmission (TX+terminal ← → TX-terminal 60-80Ω).

    4. In the AVC-LAN circuit, if a disconnected or short is occurring, the "Wide Multi AV Station" and the back guide monitor system will not work properly due to the interruption of communication.
