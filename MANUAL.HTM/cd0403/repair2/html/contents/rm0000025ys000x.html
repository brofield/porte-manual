エンジンマウンティング インシュレータ RR（1NZ-FE(4WD)）  取り付け

印刷

準備品一覧



1. エンジンマウンティング インシュレータ RR取り付け



  1. ミッションジャッキを操作して、ボルト2本およびナット2個でエンジンマウンティングインシュレータRRを取り付ける。

    基準値
    T = 62 N・m{ 632 kgf・cm } (クロスメンバ×エンジンマウンティングインシュレータRR)
    T = 64 N・m{ 653 kgf・cm } (エンジンマウンティングインシュレータRR×エンジンマウンティングブラケットRR)



2. プロペラシャフトASSY取り付け

  1. エクステンションハウジングからSSTを取りはずす。

    ■ 注 意 ■
    オイルシールを傷つけない。

  2. エクステンションハウジングにプロペラシャフトASSYを挿入する。

    ■ 注 意 ■
    オイルシールを傷つけない。




  3. プロペラシャフトASSYおよびインタミディエイトシャフトASSYの合わせマークを合わせる。

  4. ボルト4本、ワッシャおよびナット各4個で、プロペラシャフトASSYを取り付ける。

    基準値
    T = 74 N・m{ 749 kgf・cm }

    ■ 注 意 ■
    1. ワッシャはボルト側に取り付ける。
    2. 締め付け時、ワッシャおよびナットに油脂類を付着させない。



3. エキゾースト パイプASSY CTR取り付け

  1. 新品のガスケットを介して、エキゾーストパイプASSY FRにエキゾーストセンタパイプASSYを差し込む。

  2. エキゾーストパイプサポート2個でエキゾーストセンタパイプASSYを取り付ける。




  3. 新品のクランプおよびボルトでクランプを図の位置にし、ボルトを締め付ける。

    基準値
    T = 32 N・m{ 326 kgf・cm }
