スタビライザ バー FR  取り付け

印刷

準備品一覧



1. スタビライザ バー FR取り付け



  1. 左右のフロントスタビライザバーブシユNo.1をスタビライザバーの識別マークの外側に取り付ける。

    ■ 注 意 ■
    スタビライザバーブシユの切割り方向が車両後方になるように取り付ける。




  2. ボルト4本で、フロントスタビライザバーブラケツトNo.1 ＬＨおよびRHを介して、フロントスタビライザバーをフロントサスペンションクロスメンバに取り付ける。

    基準値
    T = 36.8 N・m{ 376 kgf・cm }



2. フロントスタビライザ ボルト取り付け

  1. フロントスタビライザバークッションリテーナNo.1を4個、フロントスタビライザクッション4個、フロントスタビライザバークッションリテーナNo.2およびフロントスタビライザボルトを取り付ける。

    □ 参 考 □
    1. スタビライザボルトを図のように取り付けると、固定しやすい。
    2. ロワーアームをジャッキで少し持ち上げると締めやすい。

  2. スタビライザボルトをスパナ(10mm)で固定し、新品のナット2個で取り付ける。

    基準値
    T = 18 N・m{ 184 kgf・cm }




  3. 反対側も同様の手順で取り付ける。



3. フロントサスペンション クロスメンバSUB-ASSY取り付け



  1. ミッションジャッキで、クロスメンバを支える。

  2. ボルト4本で、クロスメンバをボデーに仮付けする。

  3. SSTをクロスメンバRH側および車両RH側の基準穴に差し込み、ボルト穴位置を調整しながらボルト2本を仮締めする。

    SST
    09670-00010  

    □ 参 考 □
    ボルトA、Bの順序で仮締めする。




  4. SSTをクロスメンバLH側および車両LH側の基準穴に差し込み、ボルト穴位置を調整しながらボルト2本を仮締めする。

    SST
    09670-00010  

    □ 参 考 □
    ボルトA、Bの順序で仮締めする。

  5. SSTをクロスメンバRH側および車両RH側の基準穴に差し込み、規定トルクで締め付ける。

    SST
    09670-00010  
    基準値
    T = 70 N・m{ 714 kgf・cm } (ボルトB(フロント側))
    T = 116 N・m{ 1183 kgf・cm } (ボルトA(リヤ側))

  6. SSTをクロスメンバLH側および車両ＬＨ側の基準穴に差し込み、規定トルクで締め付ける。

    基準値
    T = 70 N・m{ 714 kgf・cm } (ボルトB(フロント側))
    T = 116 N・m{ 1183 kgf・cm } (ボルトA(リヤ側))

    ■ 注 意 ■
    クロスメンバ取り付け後、クロスメンバ側および車両側の基準穴が一致していることを確認する。

  7. ミッションジャッキをクロスメンバから取りはずす。




  8. ボルトおよびナット2個で、エンジンマウンテイングインシユレータRRをクロスメンバに取り付ける。

    基準値
    T = 80 N・m{ 816 kgf・cm }

  9. スリングデバイスおよびチェーンブロックを取りはずす。



4. フロントサスペンション アームSUB-ASSY LWR NO.1 LH取り付け

  1. ロワーアームASSYのステアリングナックルに取り付け、キャッスルナットで締め付ける。

    基準値
    T = 98 N・m{ 1000 kgf・cm }

    ■ 注 意 ■
    ねじ部およびテーパー部に油脂類を付着させない。

  2. 新品のクリップを取り付ける。

    ■ 注 意 ■
    キャッスルナット締め付け後のクリップ穴合わせは60°以内の増し締め方向で行う。



5. フロントサスペンション アームSUB-ASSY LWR NO.1 RH取り付け

□ 参 考 □
LH側と同様に取り付ける。



6. プレッシャフィード チューブASSY取り付け



  1. プレツシヤフイードチユーブASSYをステアリングリンクASSYに仮付けする。

  2. ボルトで、プレツシヤフイードチユーブASSYのクランプをクロスメンバに取り付ける。

    基準値
    T = 7.8 N・m{ 80 kgf・cm }




  3. ユニオンナットレンチ17を使用して、プレツシヤフイードチユーブASSYを本締めする。

    基準値
    T = 44.1 N・m{ 450 kgf・cm } (規定締付けトルク)
    T = 40 N・m{ 410 kgf・cm } (ユニオンナットレンチおよびトルクレンチ460F使用時)



7. タイロッド エンドSUB-ASSY LH取り付け

  1. タイロッドエンドをステアリングナックルに取り付け、キャッスルナットで締め付ける。

    基準値
    T = 49 N・m{ 500 kgf・cm }

    ■ 注 意 ■
    ねじ部およびテーパー部に油脂類を付着させない。

  2. 新品のコッターピンを取り付ける。

    ■ 注 意 ■
    キャッスルナット締め付け後のコッターピン穴合わせは60°以内の増し締め方向で行う。



8. タイロッド エンドSUB-ASSY RH取り付け

□ 参 考 □
LH側と同様に取り付ける。



9. エンジンアンダ カバー LH取り付け

  1. スクリュー2個およびボルト2本で、エンジンアンダーカバーLHを取り付ける。

    基準値
    T = 5.0 N・m{ 50 kgf・cm }



10. エンジンアンダ カバー RH取り付け

  1. スクリュー2個およびボルト2本で、エンジンアンダーカバーRHを取り付ける。

    基準値
    T = 5.0 N・m{ 50 kgf・cm }

  2. ナットを、締め付ける。

    □ 参 考 □
    ナットは、LH側と共締めになっている。

    基準値
    T = 5.0 N・m{ 50 kgf・cm }

12. ステアリングスライディング ヨークSUB-ASSY接続



  1. クリップAを付け、ホールカバーおよびクリップBをボデーに取り付ける。

    ■ 注 意 ■
    リップ部が噛み込んでいないか確認する。




  2. 合わせマークを合わせて、ボルトでスライディングヨークを取り付ける。

    基準値
    T = 28 N・m{ 290 kgf・cm }




  3. ステアリングホイール回転防止用シートベルトを取りはずす。
