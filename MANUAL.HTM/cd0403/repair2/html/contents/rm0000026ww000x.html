リヤショックアブソーバW/コイルスプリング（4WD）  取り付け

印刷

準備品一覧



1. リヤショックアブソーバW/コイルスプリング仮締め



  1. ナット2個でリヤショックアブソーバ W/コイルスプリングをボデーに取り付ける。

    基準値
    T = 39 N・m{ 400 kgf・cm }




  2. ショックアブソーバ（ロワー側）のボルトおよびナットを仮締めする。

    ■ 注 意 ■
    ナットは回り止めナットのため、必ずボルトを回して仮締めする。




  3. ボルトで、スピードセンサをボデーに取り付ける。

    基準値
    T = 8.0 N・m{ 82 kgf・cm }



2. リヤタイヤ取り付け
基準値
T = 103 N・m{ 1050 kgf・cm }


3. 車両落ち着かせ

  1. 車両をジャッキダウンして数回上下にゆすり、リヤサスペンションを落ち着かせる。



4. リヤショックアブソーバW/コイルスプリング本締め

  1. アクスルハウジングをジャッキなどで支え、サスペンションに荷重をかける。

  2. ショックアブソーバ（ロワー側）のボルトおよびナットを本締めする。

    基準値
    T = 68 N・m{ 690 kgf・cm }

    ■ 注 意 ■
    ナットを固定して、ボルトを回す。



5. リヤアブソーバ キャップ取り付け

  1. リヤアブソーバキャップを取り付ける。
