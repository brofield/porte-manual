CAN communication system (with VSC) system description

printing



  1. overview

    1. CAN (Controller Area Network) is a multiple multiple communication for in -vehicles with excellent data communication speed and error detection capacity, a serial communication for real -time applications.

    2. CAN has one pair of two lines, Can-H and Can-L, and communicates at differential voltage.

    3. The car is equipped with many ECUs (sensors), but they are not operating alone, and they are working while shared information with other ECUs (sensors).

    4. The CAN communication main line contains two resistance required for communication transmission.



    1. Definition of terms

      1. Main line


        1. The harness between the terminal circuit located on the bus (communication line) is defined as the main line.It becomes a core bus of the control system CAN.

      2. Branch line


        1. The line that is branched into the ECU or sensor from the main line, which is the core bus of the CAN bus, is defined as a branch.

      3. Terminal circuit


        1. The circuit consisting of a resistor and a capacitor, which is placed to convert the communication current of the control system to a bus voltage, is called an terminal circuit.Two terminal circuits need to be prepared on the bus.
