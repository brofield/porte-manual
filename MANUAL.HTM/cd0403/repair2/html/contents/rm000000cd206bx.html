  1. Freeze -frame data confirmation (read by Tascan)

    1. Use SST (TASCAN) to perform the operation according to the screen display, and select a diagram code (display F) that records freeze-frame data from [Diagnosis Menu]-[Diag Code Confirmation Screen].

      SST
      09991-70200


      1. Connect SST (Tascan) to DLC3.

      2. Ig ON.

      3. Select [Diagnosis Menu]-[Diagcode Freeze Data].

      4. The detected diagcode is displayed on the diagcode display screen.

      5. Select the diag code you want to check the freeze frame data.

        □ Participation □
        There is a [F] display in the diary code, and the freeze -frame data is remembered in the diag code displayed in reverse.




    1. Time series freeze frame data confirmation

      1. Select the item that you want to check the time series freeze frame data on the freeze data display screen.

        □ Participation □
        1. The time -series freeze frame data also records the ECU data before and after the diag code occurs.
        2. The time -series freeze frame data can be confirmed when the guidance is displayed on the freeze data display screen, "Press the item to confirm the time series data].
        3. The time -series freeze frame data can be displayed up to 3 points before detection, 1 point after detection, including the diagram detection point.
