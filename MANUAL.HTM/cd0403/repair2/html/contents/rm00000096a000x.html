パーキングブレーキシステム  調整

印刷


1. リヤタイヤ取りはずし


2. リヤブレーキ ドラム取りはずし


3. ブレーキドラムとライニングのすき間調整



  1. バッキングプレートのサービスホールからマイナスドライバーでアジャスターを回して拡張させ、ブレーキシューASSYをリヤブレーキドラムに接触させる。




  2. マイナスドライバーでリヤブレーキオートマチックアジャストレバーLHを押しながらもう1本のマイナスドライバーでアジャスターを回して収縮させる。

  3. ブレーキシューASSYがリヤブレーキドラムに接触しなくなった状態から更にアジャスターを1/2回転させて収縮させる。



4. リヤブレーキ ドラム取り付け


5. リヤタイヤ取り付け
基準値
T = 103 N・m{ 1050 kgf・cm }


6. パーキングブレーキペダル踏みしろ点検

  1. パーキングブレーキペダルをゆっくり踏み、作動音を数える。

    基準値
    7-10ノッチ[操作力294N{30kgf} ]



7. パーキングブレーキペダル踏みしろ調整



  1. パーキングブレーキペダルを一度強く踏む。

  2. パーキングブレーキペダルを再度踏み込み、ロックを解除する。

  3. パーキングブレーキペダル部のロックナットをゆるめ、踏みしろが基準値になるようアジャスティングナットを回して調整する。

    基準値
    7-10ノッチ[操作力294N{30kgf} ]

  4. アジャスティングナットを固定し、ロックナットを締め付ける。

    基準値
    T = 5.4 N・m{ 55 kgf・cm }

  5. パーキングブレーキペダルを1回以上操作した後、踏みしろを確認する。

  6. パーキングブレーキの引きずりがないことを確認する。

  7. パーキングブレーキペダルを操作したとき、ブレーキインジゲータランプが点灯することを確認する。

    基準
    踏み始めてから1ノッチ目で確実にランプが点灯すること
