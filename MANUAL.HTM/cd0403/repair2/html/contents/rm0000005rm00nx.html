printing



  1. Points to keep in mind about troubles

    1. The CAN communication abnormality code is also output by the ECU of the Can Communication system, the internal abnormality of the sensor, and the power supply abnormality, so be careful if the internal abnormality of the system or the power supply abnormality is output or memorized at the same time.

    2. If the CAN communication abnormality code is confirmed, or if there is no response from the ECU to the SST (Tascan), which is output by CAN communication, first confirm that the DLC3 branch line and the CAN bath main line are normal.

      □ Participation □
      The ECU, which is output by SIL communication, can output a diagnostic code to SST (Tascan) even in the DLC3 branch line and a CAN bus abnormality.

    3. By checking the DLC3 terminal using SST (Toyota Electrical Tester), the abnormalities of the DLC3 branch line and the CAN bath main circuit can be confirmed.

      ■ Attention ■
      When checking the DLC3 connector, the tester terminal is not directly inserted, but the diagnosis check wire No.2 is connected and checked.

    4. By using SST (Tascan) to confirm the 'CAN bus connection ECU', you can check the ECU (sensor) in the bronze disconnection connected to the can No.1 bus circuit.(CAN bus main line circuit normal)

      □ Participation □
      1. When confirming the CAN bus connection ECU, the ECU (sensor) during the bronze line does not respond (screen) to SST (Tascan).
      2. CAN Bus Connection ECU When all ECUs are answered (screen display), if the CAN communication abnormal diagcode outputs, select a problem that seems to have occurred from the combination table of the diagcode,Check the reproduction method and check.
