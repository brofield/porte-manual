1. フロントサスペンション アームSUB-ASSY LWR NO.1 LH仮締め



  1. 角軸マイナスドライバーを使用して、サスペンションクロスメンバを支点にミッション下部を押してエンジンを傾け、ボルト（フロント側）を挿入する。




  2. SSTをサスペンションクロスメンバRH側および車両RH側の基準穴に差し込む。

    SST
    09670-00010  

  3. ボルトA、Bの順序で仮締めする。




  4. SSTをサスペンションクロスメンバLH側および車両LH側の基準穴に差し込む。

    SST
    09670-00010  

  5. ボルトA、Bの順序で仮締めする。

  6. SSTをサスペンションクロスメンバRH側および車両RH側の基準穴に再度差し込み、規定トルクで締め付ける。

    SST
    09670-00010  
    基準値
    T = 116 N・m{ 1180 kgf・cm } (ボルトA)
    T = 70 N・m{ 714 kgf・cm } (ボルトB)

  7. SSTをサスペンションクロスメンバLH側および車両LH側の基準穴に再度差し込み、規定トルクで締め付ける。

    SST
    09670-00010  
    基準値
    T = 116 N・m{ 1180 kgf・cm } (ボルトA)
    T = 70 N・m{ 714 kgf・cm } (ボルトB)

    ■ 注 意 ■
    サスペンションクロスメンバ取り付け後、サスペンションクロスメンバ側および車両側の基準穴が一致していることを確認する。




  8. ボルトおよびナット2個で、エンジンマウンティングインシュレータRRを取り付ける。

    基準値
    T = 80 N・m{ 816 kgf・cm }

  9. ボルト２本およびナットでロワーアームをフロントサスペンションクロスメンバに仮締めする。

    ■ 注 意 ■
    ナットは回り止めナットのため、必ずボルトを回して仮締めする。

  10. ロワーアームのボールジョイント部をステアリングナックルに取り付け、キャッスルナットで締め付ける。

    基準値
    T = 98 N・m{ 1000 kgf・cm }

    ■ 注 意 ■
    1. ネジ部およびテーパ部に油脂類を付着させない。
    2. キャッスルナットの締め付け後のクリップ穴合わせは、60°以内の増し締め方向で行う。

  11. 新品のクリップを取り付ける。

9. サイドスリップ点検

  1. サイドスリップテスターでサイドスリップを点検する。

    基準値:
    0±5mm(1ｍにつき)

    ■ 注 意 ■
    1. サイドスリップテスターへの進入は直進方向に歩く速度(4km/h程度)以下とし、ニュートラル状態で行う。
    2. サイドスリップテスター上でブレーキを踏まない。
    3. サイドスリップテスター上でステアリング操作をしない。

  2. 基準値外の場合はホイールアライメントを点検する。



10. トーイン点検

  1. 車両をゆすり、車両を安定させる。

  2. 車両を直進で5ｍ手押しして前進させる。

    ■ 注 意 ■
    車両を後退させた時は、必ず後退させた距離分前進させる。




  3. トーインゲージの指針高さをフロントホイール軸中心高さに合わせ、タイヤの後ろ側に入れる。

  4. 前輪タイヤ各々の後部にトレッドセンターをしるし、マーク間の距離(B寸法)を測定する。

  5. 車両をゆっくり押して前進させ、前輪を180°回転させる。

    ■ 注 意 ■
    180°以上回転させないように行い、オーバー回転した場合は、トーインゲージをセットするところからやり直す。




  6. 車両前部でマーク間の距離(A寸法)を測定する。

  7. トーインを求める。

    基準値
    トーイン 1±2mm

    □ 参 考 □
    トーイン＝B寸法-A寸法



11. トーイン調整



  1. 左右のラックエンドのねじ部の長さを測定する。

    基準
    左右差　1.5mm以下

  2. ラックブーツのクリップをはずす。

  3. タイロッドエンドのロックナットをゆるめる。

  4. ラックエンドねじ部の左右差が基準外の場合は調整を行う。

    1. トーインの測定値がアウト側にはずれる場合は、寸法の短い方のラックエンドを伸ばす方向に調整する。

    2. トーインの測定値がイン側にはずれる場合は、寸法の長い方のラックエンドを縮める方向に調整する。

    3. 再度トーインを点検する。

  5. 左右のラックエンドを各々の反対方向に同量ずつ回し、トーインが基準値内に入るように調整する。

    基準値
    トーイン 1±1mm

  6. タイロッドエンドのロックナットを締め付ける。

    基準値
    T = 47 N・m{ 480 kgf・cm }

    ■ 注 意 ■
    ロックナットおよびステアリングラックエンドが共回りしないようにステアリングラックエンドの六角部を固定し、ロックナットを仮締めしてから、タイロッドエンドの二面幅を固定させて締め付ける。




  7. プライヤーを使用して、クリップを図の範囲に取り付ける。

    ■ 注 意 ■
    ブーツをねじって取り付けない。



12. ホイール切れ角点検



  1. ターニングラジアスゲージをセットする。

  2. ホイールの切れ角を点検する。

    基準値
    内側(点検範囲)
    		外側(参考)
    39°±2°
    		34°

    ■ 注 意 ■
    1. 車両を空車状態(スペアタイヤおよび工具なし)にして行う。
    2. 空車状態で、前輪が回転しないように、ブレーキペダルプレッシャーなどを使用して、フットブレーキを効かせた状態で点検する。
    3. ストップランプが点灯しないように、ストップランプヒューズをはずして点検する。



13. キャンバー、キャスター、キングピンアングル点検
SST
09772-30010  

  1. ホイールキャップまたはセンターオーナメントを取りはずす。

  2. 車両を平坦な場所でリフトアップする。

  3. タイヤクランプコンペンセータ(TCK-01)を取り付ける。

    1. タイヤクランプコンペンセータのクランプダイヤルを回し、タイヤクランプアームを広げる。

      ■ 注 意 ■
      タイヤクランプアームを広げすぎるとボデー等に傷を付けるため注意する。




    2. ダイヤルを回し、下側ツメ2箇所をホイールサイズに合わせホイールリム下部に当て上側ツメをホイールリムに合わせて縮める。

      ■ 注 意 ■
      ツメがバランスウェイトに掛からず、ホイールのエアーバルブがなるべく車両上側になるように取り付ける。




    3. クランプダイヤルを回し、タイヤクランプアームを縮めタイヤに取り付ける。

      ■ 注 意 ■
      タイヤクランプアーム先端がタイヤ溝に掛かり、タイヤクランプコンペンセータがしっかりとホイールに圧着していること。

    4. 脱落防止ワイヤーをホイールのエアーバルブに取り付ける。

  4. タイヤクランプコンペンセータを補正する。




    1. タイヤクランプコンペンセータの補正ダイヤルを回してメカゼロライン(赤ライン)の範囲内に合わせる。




    2. キャンバーキャスターキングピンゲージをタイヤクランプコンペンセータの取り付けプレートに取り付け、合わせマークを合わせる。




    3. ホイールを少し回転させ、キャンバーキャスターキングピンゲージの水平気泡が中心になるように合わせ、キャンバーキャスターキングピンゲージの調整ネジでキャスター気泡を0°に合わせる。




    4. ホイールを180°回転させ、キャンバーキャスターキングピンゲージとタイヤクランプコンペンセータの合わせマークを合わせ、水平気泡を合わせる。(*1)




    5. 上記状態のキャスター気泡を読み取り、その値の1/2の目盛りになるようにタイヤクランプコンペンセータの補正ダイヤルを回し合わせる。(*2)

      ■ 注 意 ■
      キャンバーキャスターキングピンゲージの調整ネジは動かさないこと。

    6. 前記(*1)、(*2)の手順を繰り返し、キャスター気泡の目盛り位置がホイールを180°正、反転させても、一定の数値目盛りを指示すればキャンバーキャスターキングピンゲージのセットは完了。

  5. ターニングラジアスゲージをセットし、車両を降ろす。

  6. キャンバー、キャスターおよびキングピンアングルを点検する。

    基準値

    キャンバー
    		キャスター
    		キングピンアングル
    -0°40'±45'
    		2°00'±45'
    		10°15'±45'

    ■ 注 意 ■
    1. 車両を空車状態(スペアタイヤおよび工具なし)にして行う。
    2. キャンバーおよびキャスターの左右差限度は30'以内。

  7. キャンバーキャスターキングピンゲージおよびタイヤクランプコンペンセータを取りはずす。

  8. ホイールキャップまたはセンターオーナメントを取り付ける。



14. キャンバー調整

  1. 車両をジャッキアップして、フロントタイヤを取りはずす。




  2. ボルトをはずし、スピードセンサおよびフレキシブルホースを切り離す。




  3. ショックアブソーバロワー側の取り付けナット2個を緩める。




  4. ショックアブソーバのロワー側を動かして、ボルトとステアリングナックルのボルト穴のガタでキャンバーを調整する。




  5. ナット2個を締め付ける。

    基準値
    T = 132 N・m{ 1350 kgf・cm }




  6. ボルトで、フレキシブルホースおよびスピードセンサを取り付ける。

    基準値
    T = 29 N・m{ 300 kgf・cm }

    ■ 注 意 ■
    フレキシブルホースおよびスピードセンサをねじって取り付けない。

  7. フロントタイヤを取り付ける。

    基準値
    T = 103 N・m{ 1050 kgf・cm }

  8. 車両をジャッキダウンして数回上下にゆすり、サスペンションを落ち着かせる。

  9. キャンバーを点検する。

    ■ 注 意 ■
    1. 車両を空車状態(スペアタイヤおよび工具なし)にして行う。
    2. キャンバーおよびキャスターの左右差限度は30'以内。

    □ 参 考 □
    セットボルトで調整できる範囲は、0.1°から0.5°である。

  10. 基準値内の場合は、トーイン点検を行う。

  11. 基準値外の場合は、ジャッキアップしてフロントタイヤをはずし、スピードセンサおよびフレキシブルホースを切り離し以下の作業を行う。




    1. ショックアブソーバロワー側の取り付けナット2個およびボルト2本を取りはずす。





    2. アジャスティングボルト2本を選択して、車両前方から挿入する。

    3. ナット2個を取り付ける。

      ■ 注 意 ■
      ショックアブソーバロワーの締め付け面のよごれを拭き取る。

      基準値
