フューエルシステム  注意事項

印刷



    1.作業前注意

    1. フユーエルシステムでの作業前に、マイナスターミナルをバッテリからはずす。

    2. フユーエルシステムの作業は火気ののそばでは絶対にしないこと。また作業中には絶対に喫煙しない。

    3. ゴム製、または皮製の部品はガソリンから離しておく。



      2.燃料流出防止作業

      ■ 注 意 ■
      1. 燃料系部品取りはずし前に必ず燃料流出防止作業を行う。
      2. 燃料流出防止作業後もフユーエルラインに若干残圧が残るためフユーエルラインを切り離す際はウエス等で覆い、燃料の飛散を防ぐ。




      1. フユーエルポンプASSY用のコネクタを切り離す。

      2. エンジン始動


        1. エンジンを始動する。

        2. エンジンが自然に停止した後、イグニツシヨンスイツチをOFFにする。

          □ 参 考 □
          “ダイアグコードP0171/25”の燃料異常が検出される場合がある。

        3. 再度エンジンをクランキングし、始動しないことを確認する。

      3. フユーエルタンクキヤツプをはずし、フューエルタンク内の圧力を抜く。

      4. バツテリのマイナスターミナルを取りはずす。

      5. フユーエルポンプASSYのコネクタを取り付ける。



        3.フユーエルシステム


        1. 高圧燃料ラインをはずすときには、大量のガソリンが飛散するので下記の手順に従う。


          1. 燃料流出防止作業

          2. フユーエルチユーブを取りはずす。

          3. フユーエルチユーブに残っている燃料を抜き取る。




          4. はずしたフユーエルチユーブが傷ついたり、中に異物が混入しないようにビニール袋でカバーをする。




        2. インジエクタ脱着時には下記の事前注意を守る。


          1. Oリングは絶対に再使用しない。

          2. 新品のOリングをインジエクタに取り付けるとき、絶対に傷つけない。

          3. 取り付け前にスピンドルオイルまたはガソリンを新品のOリングに塗布する。エンジンオイル、ギヤオイルおよびブレーキオイルは絶対に使用しない。




        3. 図のように、デリバリパイプとシリンダヘツドにインジエクタを取り付ける。インジエクタ取り付け前に、デリバリパイプがインジエクタのOリングに触れる箇所にガソリンかスピンドルオイルを必ず塗っておく。




        4. フユーエルチユーブコネクタ(金属タイプ)の取り外し時には下記の事前注意を守る。

          □ 参 考 □
          金属性コネクタの構造は左記に示す。





          1. EFIフユーエルパイプクランプNo.2を取りはずす。




          2. フユーエルチユーブASSYの金属性コネクタを手に取り、上方に向かって引き出し、保持する。

          3. SSTを図のように接続部につける。

            SST
            09268-21010  




          4. SSTの面取りした部分をコネクタ内のリテーナ部と合わせ、SSTをコネクタに差し込む。

          5. SSTとコネクタを一緒にフユーエルチユーブ側へスライドさせて取りはずす。




        5. フユーエルチユーブコネクタ(クイックタイプ)を取りはずす場合は下記の事前注意を守る.


          1. フユーエルパイプクランプをはずす。

          2. 取りはずす前にパイプやコネクタ周りに泥などの汚れがないかを確認し、あれば拭き取る。




          3. 半かん合防止コネクタの場合は、フユーエルコネクタカバーを上方に引き上げて、ロックを解除する。

          4. 必ず手で取りはずす。




          5. コネクタとパイプが固着した場合、手でリテーナを挟み、押したり引いたりして接続をはずして引き出す。このとき、工具は使用しない。

          6. 取りはずしたパイプのシール面に泥などがついていないかを点検して、あれば拭き取る。




          7. 取りはずしたパイプとコネクタが損傷したり、異物が混入しないようにビニール袋でカバーする。




        6. フユーエルチユーブコネクタ(クイックタイプ)の接続時には下記の事前注意に従う。


          1. コネクタの軸をパイプ軸に合わせ、リテーナがカチッと音がするまでコネクタを押し込む。接続がきつい場合、少量のエンジンオイルをパイプの先に塗る。

          2. 接続後、パイプとコネクタを引っ張って、確実に接続されたかを確認する。




          3. 半かん合防止コネクタの場合は、フユーエルホースコネクタカバーをロックする。

            □ 参 考 □
            半かん合防止コネクタはフユーエルチユーブの接続が不完全な場合、フユーエルホースコネクタカバーがロックできない構造になっている。



          4.燃料漏れ点検

          1. フユーエルシステムの修理を行ったあとは、燃料漏れがないことを必ず確認する。(要領は参照)
