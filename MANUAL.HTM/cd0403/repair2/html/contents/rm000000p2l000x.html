ABS & TRC & VSC & BA System Freeze Frame Data / Diag Details Information

printing



  1. Freeze frame data output

    1. You can use Tascan to remember and display the vehicle (sensor) status when the system is operated or at the abnormal dial -scenoxiscode output.

    2. Freeze frame data when system operation


      1. The data is memorized when the system is operating, and the data is updated for each system operation.

      2. Finally, IG OFF to 31 times from when the system is activated is memorized and displayed.

        □ Participation □
        1. IG OFF → ON at the vehicle stop (7km/h less than 7km/h) is not counted up.
        2. If the number of IG switch ON exceeds 31 times, display [31].

      3. When the freeze -frame data when the abnormal diagnostic code occurs, the freeze -frame data at the time of system operation is deleted.

    3. Freeze frame data when abnormal diagnostic cord occurs


      1. When an abnormal diagnostic cord is output, the freeze -frame data is memorized when an abnormal code occurs.

      2. If you remember the freeze -frame data when an abnormal diag code occurs, the data will not be updated until the memory data is erased (deleted of the diagnostic code).

      3. In the freeze -frame data when an abnormal diagnostic cord occurs, the number of Ig ON times after the freeze frame data memory is not remembered.
