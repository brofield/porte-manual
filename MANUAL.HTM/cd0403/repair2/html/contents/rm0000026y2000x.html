アッパコントロール アームASSY RR LH（4WD）  取り付け

printing

List of preparation



1. アッパコントロール アームASSY RR LH仮締め



  1. Temporarily tighten the upper control arm with two bolts and nuts.

    ■ Attention ■
    The nut is a nut, so be sure to turn the bolt and temporarily tighten it.



2. Rear tire installation
Reference value
T = 103 n { 1050 kgf / cm }


3. Calm down the vehicle

  1. Jack the vehicle, shake up and down several times, and calm the rear suspension.



4. Appon control arm Assing RR LH LH main

  1. アクスルハウジングをジャッキなどで支え、サスペンションに荷重をかける。

  2. ナットを固定し、ボルト2本を締め付ける。

    Reference value
    T = 77 N・m{ 785 kgf・cm }

    ■ 注 意 ■
    Do not rotate the nut when tightening the bolt.
