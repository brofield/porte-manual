1. オイルポンプ シール取り付け

  1. SSTを使用して、新品のオイルシールをオイルポンプの端面まで均等に打ち込む。

    SST
    09950-60010   (09951-00250, 09951-00410, 09952-06010)
    09950-70010   (09951-07100)

    ■ 注 意 ■
    オイルシールを斜めに打ち込まない。




  2. オイルシールのリップ部に少量のMPグリースNo.2を塗布する。

    ■ 注 意 ■
    リップ部に切粉および砂等の異物を付着させない。



2. オイル ポンプASSY取り付け



  1. 図の箇所に新品のOリングを取り付ける。



  2. エンジン本体側およびオイルポンプにシールパッキンを塗布する。

    ■ 注 意 ■
    1. 取り付け面を清掃および脱脂する。
    2. シールパッキン塗布後3分以内に取り付け、15分以内にボルトを締め付ける。
    3. 取り付け後、2時間以内はエンジンオイルを注入しないで放置する。
    4. 取り付け後、2時間以内はエンジンを始動させない。







  3. オイルポンプのドライブロータスプライン部をクランクシャフトの2面幅部に合わせ、オイルポンプASSYをクランクシャフトに挿入する。




  4. ボルト15本およびナットで、オイルポンプASSYを取り付ける。

    基準値:
    T＝32N·m｛326kgf·cm｝（ボルトＡ）（Ｍ8　30mm）
    T＝11N·m｛113kgf·cm｝（ボルトB）（Ｍ6　35mm）
    T＝11N·m｛113kgf·cm｝（ボルトＣ）（Ｍ6　20mm）
    T＝24N·m｛245kgf·cm｝（ナットＤ）
    T＝24N·m｛245kgf·cm｝（ボルトＥ）（Ｍ8　2段ボルト）

    ■ 注 意 ■
    1. オイルポンプASSY取り付け時、チエーン接触しないように注意する。
    2. オイルポンプASSY取り付け後、15分以内にエンジンマウンテイングブラケットRHおよびウオータポンプASSYを取り付ける。



3. カムシャフトタイミングオイルコントロール バルブASSY取り付け

  1. ボルトで、カムシャフトタイミングオイルコントロールバルブを取り付ける。

    基準値
    T = 7.5 N・m{ 76 kgf・cm }



4. クランクポジション センサ取り付け

  1. Oリングに少量のエンジンオイルを塗布する。




  2. ボルトでクランクポジシヨンセンサを取り付ける。

    基準値
    T = 7.5 N・m{ 76 kgf・cm }

    ■ 注 意 ■
    Oリングの切れおよび噛み込みがないようにオイルポンプに取り付ける。

  3. コネクタを接続する。



5. クランクシャフト ダンパSUB-ASSY取り付け

  1. クランクシャフトダンパのストレートピン穴とクランクシャフトストレートピンを合わせ、クランクシャフトダンパを取り付ける。




  2. SSTを使用してクランクシャフトダンパを固定し、ボルトを締め付ける。

    SST
    09213-58013   (91111-50845)
    09330-00021  
    基準値
    T = 128 N・m{ 1305 kgf・cm }

    ■ 注 意 ■
    SST取り付け時、SST固定用ボルトがチェーンカバーに接触する場合があるので、取り付け位置を確認しながらSSTを取り付ける。

10. ベーンポンプVベルト(パワステベルト)取り付け

12. オルタネータASSY取り付け



  1. 固定用ボルトBでオルタネータを仮付けする。




  2. ボルトAおよびナットでファンベルトアジャスティングバーを仮付け後、オルタネータをシリンダブロック側に寄せてナットを本締めする。

    基準値

  3. コネクタおよびワイヤハーネスクランプを取り付ける。

  4. ナットで+B端子ターミナルを取り付ける。

  5. ターミナルキャップを取り付ける。



13. ファン ＆ オルタネータ Vベルト取り付け

  1. Vベルトを各プーリに仮付けする。

    ■ 注 意 ■
    1. ベルトと各プーリに、オイル等の液体及び異物等の付着無きこと。
    2. ベルトのリブ山ずれの無きこと。



14. ファン ＆ オルタネータ Vベルト調整



  1. ハブナットレンチまたはバーなどを使用して、オルタネータを車両フロント側に引いて、フアン　&　オルタネータVベルトの張力を調整する。

    ■ 注 意 ■
    Vベルトを張る際は、オルタネータとエンジンマウンテイングインシユレータの間にハブナットレンチまたはバーを差し込み、テンションをかける。オルタネータとカムシャフトタイミングオイルコントロールバルブ(インテークマニホルド側)の間にハブナットレンチまたはバーを差し込むとカムシヤフトタイミングオイルコントロールバルブを破損させる恐れがあるため、絶対に行わない。




  2. 調整用ボルトAを締め付けてから、固定用ボルトBを締め付ける。

    基準値
    T = 19 N・m{ 194 kgf・cm } (ボルトA)
    T = 54 N・m{ 551 kgf・cm } (ボルトB)

22. 冷却液(トヨタ純正スーパーLLC)補充

  1. ラジエータドレーンコツクプラグを閉じて、冷却水をラジエータ注入口よりあふれるまで注入する。［＊１］

    □ 参 考 □
    ラジエータインレツトホースおよびラジエータアウトレツトホースを手で数回圧迫し、ラジエータ注入口の水位が下がる様であれば追加注入する。

  2. ラジエータキヤツプを締める。

  3. ラジエータリザーブタンクに冷却液を上限まで注入する。

  4. エンジンをサーモスタツトが開弁するまで暖機する。

    □ 参 考 □
    暖機中にラジエータインレツトホースおよびアウトレツトホースを数回圧迫すること。

  5. エンジンを止め、冷却液が冷えるまで待ち、ラジエータキヤツプをはずして水位を確認する。

  6. 水位が下がっている場合は、［＊１］より繰り返す。

  7. 水位が下がらなくなったら、ラジエータリザーバタンクの冷却液を調整する。
