DTC C1231/31 Red angle sensor system abnormalities

printing

List of preparation


Function explanation


  1. The skid control computer (built -in actuator) enters a steering sensor signal by CAN communication.
  2. If the communication line with the steering sensor is abnormal, a diary code [U0126/63 (communication with the steering angle sensor)] will be output.

DTC detection status
DTC output -related items
C1231/31
When the IG1 terminal voltage is 9.5V or more, when an abnormal signal from the steering sensor is received
  1. Steering sensor power circuit disconnection or short
  2. Steering sensor
  3. Brake actuator ASSY

Inspection procedure

Step 1Diag code check

  1. Erase the diagnostic cord and turn off the IG switch.

  1. Turn on the IG switch again and confirm that the CAN communication diag code is not output.

  1. Run and steering at 35km/h or more, and confirm that the speed sensor abnormality and the dial cord of the yolly sensor are not output.

    □ Participation □
    1. If there is an abnormality in the speed sensor or a yolate sensor, an abnormal value may be output even if the steering sensor is normal.
    2. If the speed sensor or a yolly sensor abnormal code is output at the same time, after repairing them, check the steering sensor.
