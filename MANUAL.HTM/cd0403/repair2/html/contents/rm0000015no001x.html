ABS & BA system (4WD) inquiry table with EBD

printing






  1. Confirmation of defective status with freeze frame data

    1. Freeze -frame data when abnormal diagram code occurs


      1. If the freeze frame data in the event of an abnormal diagnostic cord occurs, "F" is displayed in the corresponding dial code on the Tascan display screen.

        □ Participation □
        1. When a freeze -frame data when an abnormal code occurs, the freeze -frame data at the time of system operation is erased.
        2. If the freeze -frame data when an abnormal code occurs, the data will not be updated until the memory data is erased (the diagnothesis code is erased).

    2. Freeze frame data when system operation


      1. The freeze -frame data at the time of system operation is updated to new data for each system operation.

      2. Finally, the Ig OFF → ON number of times from the time the system is activated is memorized and displayed.

      3. The following calculation formula can be confirmed as a guide to whether the freeze -frame data is the data at the time of the problem pointed out by the customer.(Number of IG ON times per day) x (number of days from the occurrence of defects) = (Ig ON times from data freeze)

        □ Participation □
        1. The IG OFF → ON at the vehicle stop (the vehicle speed is 7 km/h or less) is not counted.
        2. If the number of Ig ON from data freezes exceeds 31 times, "31" is displayed.
