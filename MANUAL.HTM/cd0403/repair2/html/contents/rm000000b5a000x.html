Door control transmitter registration

printing


1. Identification code registration

□ Participation □
  1. The additional mode is a mode that remains the registered code and registers the newly received identification code, and is used when the door control transmitter is added.In addition, up to 4 registration can be registered.(If you exceed 4, the first registered code will be deleted.)
  2. The rewriting mode is a mode that deletes all registered code and registers only the newly received identification code, and is used when replacing the door control transmitter.In addition, up to 4 registration can be registered.(If you exceed 4, the first registered code will be deleted.)
  3. The prohibited mode is a mode that erases all registered code and prohibits the wireless alloca function, and is used to lose the door control transmitter.
  4. The confirmation mode is a mode that checks how many types of identification code is registered, and is used to confirm the number of registered code at the time of additional registration.

  1. Make the vehicle in the following state.

    1. The driver's door glass is fully opened.

    2. Open only the driver's seat door.

    3. Make the driver's seat door in UNLOCK.

    4. The key plate is not inserted into the ignition switch rock cylinder.

  2. After the above work, perform the following work.

    1. Repeat the work of inserting the key plate into the ignition switch rock cylinder and pulling it out twice.

      ■ Attention ■
      Do it within 5 seconds.

    2. The driver's seat door is fully closed → After opening twice, the key plate is inserted into the ignition switch rock cylinder and extracted.

      ■ Attention ■
      Do it within 40 seconds.

    3. The driver's seat door is fully closed → Open twice, the key plate is inserted into the ignition switch rock cylinder, and all the doors are closed.

      ■ Attention ■
      Do it within 40 seconds.

    4. ON ← → Lock of the ignition switch is performed about 1 second for each mode shown in the figure below, and the key plate is removed after selecting the mode.

      ■ Attention ■
      Do it within 40 seconds.




    5. After the mode selection, within 5 seconds, the door control relay is automatically performed the door lock Lock ← → UNLOCK operation, and the selected mode is notified to the worker in each response pattern as shown in the figure below.

      □ Participation □
      1. If you select a confirmation mode and prohibited mode, it will end after the response.
      2. The confirmation mode shows an example in which two types of identification code is registered in the figure because the number of cycles is operated as the registered code number.However, if the number of registration code is 0, execute 11 cycles.
      3. If Ignitsushi -si -si On ← → → LOCK is 4 or 6 or more times, it will end with no response.




  3. Registration of the door control transmitter

    1. If you select an additional mode and rewriting mode, press the door control transmitter rock switch and unlock switch at the same time, and then press the switch on the door control transmitter again.

      ■ Attention ■
      1. Press more than 1 second per switch operation.
      2. After pulling out the key, the registration of the door control transmitter is done within 45 seconds.

    2. After the switch of the door control transmitter is turned off, if the registration work of the electrical key transmitter is successfully completed, Lock ← → UNLOCK operation is automatically performed once within 5 seconds, but if it is performed twice, it is registered.Since it has not been completed, registration of the door control transmitter will be registered again.

      ■ Attention ■
      After the ignition switch is turned off, do not operate the door control transmitter until Lock ← → UNLOCK is operated automatically.However, if LOCK ← → UNLOCK is not operated even after 5 seconds, registration of the door control transmitter will be performed again.


    3. If you continue to register, register the door control transmitter within 45 seconds.

      □ Participation □
      Up to 4 types of identification code can be registered.

  4. End of registration mode (when one is established)

    1. Registered mode waiting for input has passed 45 seconds or more.

    2. Open any door.

    3. Insert the key into the ignition switch rock cylinder.

  5. After the registration work, perform the following work.
