    1. Diagnostic activation and cancellation

      ■ Attention ■
      After turning on the IG switch, there are items that cannot be inspected unless the diagnodes mode is launched after confirming that the map screen appears.

      □ Participation □
      There are two types of diagnostic startup methods.



      1. Starting method 1


        1. Make the vehicle stopped.

        2. Start IG Switch ACC, ON or Engine.




        3. Navigation receiver ASSY "information · G" is pressed, and the head lamp Dimaswitch is operated as OFF → Tail → OFF → OFF → Tail → OFF.




        4. When the diagnostic is started, the "service inspection" screen is displayed, and a service inspection is performed.




        5. Select the "Inspection Menu" on the "Service Inspection" screen and display the diagram test menu screen.
          Diag inspection menu screen display
          display
          		Content
          Service inspection
          		Collect system inspection+diag memory, and display the current and past test results of each connection device from the results of both sides.
          Display inspection
          		Display the screen for performing a unique inspection of the display part
          Navigation inspection
          		Display the screen for performing unique inspections and settings in the navigation ECU part
          Camera examination
          		Display the screen to adjust the back guide monitor system
          BT Hands Free Test
          		Display the screen to inspect and configure Bluetooth hands -free functions
          Helpnet test
          		Display the screen for inspection and setting of helpnet system
          G terminal information
          		Display the screen that displays information on the G-Book device
          Program update
          		This switch can be pressed but not used

          □ Participation □
          When adjusting the back guide monitor system, start the engine and then start the diagnostic.(The point isreference)



      2. How to start 2


        1. Make the vehicle stopped.

        2. Start IG Switch ACC, ON or Engine.




        3. On the screen other than TV, press the "image quality / erasure" key of the navigation receiver ASSY and display the image quality adjustment screen.




        4. Touch the upper left and lower left of the image quality adjustment screen slowly, as in the upper left → lower left → upper left → upper left → lower left → lower left, slowly.

          □ Participation □
          Touch the four corners of the image quality adjustment screen, not the corner of the display.




        5. When the diagnostic is started, the "service inspection" screen is displayed, and a service inspection is performed.




        6. Select the "Inspection Menu" on the "Service Inspection" screen and display the diagram test menu screen.

      3. Method of unlocking


        1. The release method is turned off the IG switch.Alternatively, press the "image quality / erasure" key for 3 seconds or more.

            1. GPS information


              1. Start the diagnostic and display the diagram test menu screen.

              2. Select "Navi Inspection" on the diagram inspection menu screen and display the navigation test menu.

              3. Select "GPS information" on the navigation inspection menu screen and display the GPS information screen.

              4. Check GPS related information.
                GPS information screen display
                display
                		meaning
                Satellite information
                		Display "satellite number", "elevation angle", "signal level", and "receiving status" of satellites captured by the antenna (up to 12 satellites)
                Date and time
                		The date and time information obtained from the GPS receiver is displayed in "year", "moon", "day", "time", "second", and "seconds"
                position
                		Display the latitude and longitude information of the current location in "degree", "minutes" and "seconds"
                Receiving state
                display
                		conditions
                in use
                		When using the satellite for positioning
                Under reception
                		If the satellite is captured but is not used for positioning
                -

                Positioning
                display
                		conditions
                2D
                		At 2D position
                3D
                		At 3D position
                D -2D
                		Differential effective 2D position
                D -3D

                G
                		When the positioning data is not used
                Ero R
                		When receiving error occurs
                -
                		Other than the above

                □ Participation □
                The vehicle signal is updated every second.




              5. Select the “Date and time setting” switch and set the GPS date and time.

                □ Participation □
                1. G-Book cannot be used in places where GPS radio waves cannot be received.For this reason, the G-BOOK demonstration is performed in a place where GPS radio waves cannot be received, such as indoors, etc., using this function to set the GPS time.
                2. While receiving GPS radio waves, the GPS time is prioritized.
