ロワーコントロール アームASSY LH（4WD）  取り付け

印刷

準備品一覧



1. ロワーコントロール アームASSY LH仮締め



  1. ボルト2本およびナットで、ロワーコントロールアームASSY LHを仮締めする。

    ■ 注 意 ■
    ナットは回り止めナットのため、必ずボルト回して仮締めする。



2. 車両落ち着かせ

  1. 車両をジャッキダウンして数回上下にゆすり、リヤサスペンションを落ち着かせる。



3. ロワーコントロール アームASSY LH本締め

  1. アクスルハウジングをジャッキなどで支え、サスペンションに荷重をかける。

  2. ナットを固定し、ボルト2本を締め付ける。

    基準値
    T = 77 N・m{ 785 kgf・cm }

    ■ 注 意 ■
    ボルトを締め付ける際、ナットを回転させない。




  3. ボルトでパーキングブレーキケーブルASSY No.3をロワーコントロールアームASSY LHに取り付ける。

    基準値
    T = 5.4 N・m{ 55 kgf・cm }



4. リヤタイヤ取り付け
基準値
T = 103 N・m{ 1050 kgf・cm }
