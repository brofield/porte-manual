Lower Control Arm ASSY LH (4WD) Installation

printing

List of preparation



1. Lower Control Arm ASSY LH temporary tightening



  1. With two bolts and nuts, provisionally tighten the rower control arm Assy LH.

    ■ Attention ■
    Nut is always bolted and temporarily tightened because it is a non -nut.



2. Calm down the vehicle

  1. Jack the vehicle, shake up and down several times, and calm the rear suspension.



3. Lower control arm Assy LH main closure

  1. Axle housing is supported by jacks and the like and loading the suspension.

  2. Fix the nut and tighten the two bolts.

    Reference value
    T = 77 n ・ m {785 kgf / cm}

    ■ Attention ■
    Do not rotate the nut when tightening the bolt.




  3. Attach the parking brake cable ASSY No.3 with bolts to the rower control arm Assy LH.

    Reference value
    T = 5.4 n ・ m {55 kgf / cm}



4. Rear tire installation
Reference value
T = 103 n ・ m {1050 kgf / cm}
