22. フロントシート アウタベルトASSY RH取り付け

  1. ELRロック開始の傾斜角度点検




    1. リトラクタを取り付け状態から静かに動かしたとき、全方向に対してベルトのロックが15°以内でロックしないことを点検し、45°以上でロック状態を保持することを点検する。

      ■ 注 意 ■
      リトラクタは分解しない。

  2. ボルト2本で、フロントシートアウタベルトASSY RHのリトラクタ部を仮締めする。

  3. 上部→下部の順にフロントシートアウタベルトASSY RHのリトラクタ部を本締めする。

    基準値
    リトラクタ上部のボルト
    T = 8.5 N・m{ 87 kgf・cm }
    リトラクタ下部のボルト
    T = 42 N・m{ 428 kgf・cm }

  4. ボルトで、フロントシートアウタベルトASSY RHのショルダアンカ部を取り付ける。

    基準値
    T = 42 N・m{ 428 kgf・cm }




  5. コネクタ(プリテンショナ用)を接続し、ロッキングボタンをロックさせる。

    ■ 注 意 ■
    ロッキングボタンを確実にロックする。

  6. ELRロック点検

    1. 車両取り付け状態において、すばやくシートベルトを引き出したとき、シートベルトがロックすることを確認する。

45. リヤシートクッションASSY取り付け（リヤシート分割可倒式）

  1. リヤシートクッションASSY前側をリヤシート クッション ロック ストライカにロックさせる。

  2. リヤシートベルトを、リヤシートクッションカバー & パッド後部のゴムバンドに通す。




  3. ボルトで、リヤシートヒンジLHを取り付ける。

    基準値
    T = 37 N・m{ 377 kgf・cm }

  4. リヤシートバックヒンジＲＨにスナップリングを取り付ける。

  5. リヤシートヒンジRHを、リヤシートクッションASSY右側ブラケットの角パイプに挿入する。

    □ 参 考 □
    リヤシートヒンジRHのスナップリング溝にスナップリングがはまっていることを確認する。



46. リヤシートバックASSY LH取り付け（リヤシート分割可倒式）
