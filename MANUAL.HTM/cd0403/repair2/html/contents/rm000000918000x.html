イグニッションシステム  機能点検

印刷



  1. 火花点検

    1. ダイアグコード確認(要領は参照)

      ■ 注 意 ■
      ダイアグコードが出力されたら該当ダイアグコードのトラブルシュートを行う。

    2. 火花点検


      1. シリンダーへツドカバーNO.2を取りはずす。

        基準値
        T = 7.0 N・m{ 71 kgf・cm }

      2. イグニツシヨンコイルASSYを取りはずす。

        基準値
        T = 9.0 N・m{ 92 kgf・cm }

      3. スパークプラグを取りはずす。

        基準値
        T = 18 N・m{ 184 kgf・cm }

      4. イグニツシヨンコイルASSYにスパークプラグを取り付けてコネクタを接続する。

      5. フユーエルインジエクタASSYのコネクタを全て切り離す。

      6. スパークプラグをアースする。

      7. クランキングしたとき、火花が飛ぶことを確認する。

        ■ 注 意 ■
        1. スパークプラグの接地電極は、必ず六角部またはねじ部を接地させて点検する。
        2. 落下等、イグニツシヨンコイルに衝撃を与えた場合は、新品に交換する。
        3. 2秒以上クランキングしない。
