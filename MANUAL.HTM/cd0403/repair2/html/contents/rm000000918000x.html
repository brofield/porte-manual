    2. Spark inspection


      1. Remove the noodles No.2 from the cylinder.

        Reference value
        T = 7.0 n ・ m {71 kgf / cm}

      2. Take off Ignitsushiyon coil ASSY.

        Reference value
        T = 9.0 n ・ m {92 kgf / cm}

      3. Take off the spark plug.

        Reference value
        T = 18 n ・ m {184 kgf / cm}

      4. Attach a spark plug to Ignitsu Syon coil ASSY to connect the connector.

      5. Cut all the connectors of the Fuuel Ingiele ASSY ASSY.

      6. Earth a spark plug.

      7. When cranking, make sure that sparks fly.

        ■ Attention ■
        1. The ground electrodes of the spark plug are always checked by contacting the hexagon or the screw.
        2. If you shock the Ignitsushiyon coil, such as falling, replace it with a new one.
        3. Do not crank up for 2 seconds or more.
