  1. CD


    1. Disk used




      1. This player can only use the music CD, CD-R (CD-RECORDABLE) and CD-RW (CD-Rewritable) with the figure.



      2. Precautions on the disk

        ■ Attention ■
        1. Copy control CDs cannot be used.
        2. DualDiscs (CDs and DVDs are attached to the front and back) cannot be used.
        3. CD-R and CD-RW may not be able to be regenerated due to record status, disk characteristics, scratches, dirt, and long-term deterioration in the indoor environment.
        4. CD-R and CD-RW that are not finely processed cannot be played.
        5. The disk is handled so as not to add fingerprint, dirt, and scratches.
        6. When handling the disk, hold the center hole and end, and raise the label surface.
        7. If you press the disk removal button and leave the disk protruding for a long time, the disk may be deformed and cannot be used.
        8. If you use cellophane tape, sticker, CD-R label, rental CD label, etc., or use a disc after peeling, you may not be able to take out the disk or cause equipment failure.be.
        9. The disk is stored in avoiding direct sunlight.(Because the disk is deformed and may not be available)
        10. Disks other than circulars with a diameter of 12cm or 8cm cannot be played.
        11. Special shaped disks cannot be used because they cause equipment failure.
        12. CDs with transparent or translucent parts in the recording state are not used because they may not be able to put in and out normally.
        13. When playing a single CD (8cm), a "Toyota genuine 8cm CD adapter" is required.If a CD adapter other than the Toyota genuine product, it may break down.

        □ Participation □
        1. When it is cold or rainy, there may be dew (water drops) inside the player, so that the glass inside the car becomes cloudy.In this case, the sound may fly or the playback will stop, so if you use it after ventilation or dehumidification for a while, it may break down.
        2. If the player vibrates violently due to a bad road driving, the sound may fly.

    1. MP3/WMA

      1. About regenerated MP3 file standards
        Corresponding standard
        		MP3 (MPEG1 Layer3, MPEG2 LSF LAYER3)
        Supported sampling frequency
        1. MPEG1 Layer3: 32, 44.1, 48 (KHz)
        2. MPEG2 LSF Layer3: 16, 22.05, 24 (KHz)
        Supported bit rate
        1. MPEG1 Layer3: 64, 80, 96, 112, 128, 160, 192, 224, 320 (Kbps)
        2. MPEG2 LSF Layer3: 64, 80, 96, 112, 128, 144, 160 (Kbps)
        3. Compatible with VBR
        Compatible channel mode
        		Stereo, joint stereo, dual channel, monaural

      2. Regarding regenerated WMA file standards
        Corresponding standard
        		WMA Ver.7, 8, 9
        Supported sampling frequency
        		32, 44.1, 48 (KHz)
        Supported bit rate
        1. Ver.7, 8: CBR48, 64, 80, 96, 128, 160, 192 (Kbps)
        2. Ver.9: CBR48, 64, 80, 96, 128, 160, 256, 320 (Kbps)
        3. Supports only 2ch playback

      3. About ID3 tag/WMA tag


        1. In the MP3 file, the attached character information called ID3 tag can be entered, and the title of the song, artist, etc. can be saved.

          □ Participation □
          Compatible with ID3 tags of ID3 ver.1.0, 1.1, ID3 ver.2.2, 2.3 (the number of characters is based on ID3 ver.1.0, 1.1)

        2. In the WMA file, the attached character information called WMA tag can be entered, and the title of the song, the artist, etc. can be saved.

      4. About the media that can be used


        1. The media that can be used for MP3/WMA playback can only be CD-ROM, CD-R (CD-RECORDABLE) and CD-RW (CD-REWRITABLE).

          ■ Attention ■
          1. The CD-R and CD-RW are weaker in hot and humid environments than the disks used in regular music CDs, and some CD-R and CD-RW may not be played.
          2. If a fingerprint or scratched disk is made, it may not be possible to regenerate or fly.
          3. Some CD-Rs and CD-RWs have deteriorated in long-term inside vehicles.
          4. Since the CD-R and CD-RW are vulnerable to ultraviolet rays, it is stored in cases that do not allow light.

      5. About used media format


        1. Used media format
          Disk format
          		CD-ROM Model1, CD-ROM XA MODEL2 FORM1
          File format
          		ISO09660 Level 1, Level 2 (Joliet)

          □ Participation □
          1. MP3/WMA files written other than the above format may not be able to be played normally, or files and folder names may not be displayed correctly.
          2. Disks containing music data and MP3 or WMA format are mixed only in music data.
          3. It is compatible with multi-sessions, and can play CD-R and CD-RW with MP3/WMA files.However, only the first session is played.

        2. Standards and restrictions
          Maximum directory hierarchy
          		Eight -layer
          Maximum folder name/
          Number of file name characters
          		16 characters in full -width (32 characters when only half -width)
          Maximum folder number
          		192
          (Route folder, folder without MP3/WMA files, including empty folders)
          Maximum number of folders in the disk
          		255
          (Including files other than MP3/WMA files)

      6. About the file name


        1. The only files that are recognized and played as MP3/WMA shall be the MP3/WMA extension ".mp3"/". WMA".

        2. Save the MP3/WMA file with an extension of ". Mp3"/". Wma".

          ■ Attention ■
          If a ".mp3"/"wma" extension is attached to a file other than MP3/WMA, it will be misunderstood as an MP3/WMA file, and a loud noise may occur and the speaker may be damaged.

        1. AVC-LAN communication system

          1. The HDD navigation system will communicate between devices by AVC-LAN when installing options with a retailer.

          2. AVC-LAN master is the navigation receiver ASSY.

          3. Navigation receiver ASSY contains the resistance required for communication (TX+← → TX-terminal 60-80Ω).

          4. In the case of abnormalities such as shorts of communication lines,+B shorts and GND shorts in the AVC-LAN circuit, the HDD navigation system will not work properly due to interruption of communication.



          1. Diagnostic function

            1. The HDD navigation system has a diagnostic function, and can confirm the connection of the device connected by the AVC-LAN and perform a failure diagnosis.(The test result is displayed on the display part)

            2. The devices that make up the AVC-LAN are set with a three-digit number (notation in the hexadecimal number) called a device code (physical address).

            3. Each function (unit) that makes up the interior of the AVC-LAN device is set with a two-digit number (notation in the hexadis) called a logical address.



            1. Back guide monitor adjustment function

              1. The HDD navigation system has a back guide monitor system adjustment function, and it is necessary to adjust using the diagnostic screen when removing the back guide monitor system related parts.(The point isreference)
