リヤショックアブソーバASSY LH  取り付け

印刷


1. ショック アブソーバASSY RR LH取り付け

  1. リヤサスペンションサポートASSY LHをリヤショックアブソーバASSY LHに取り付ける。




  2. ジャッキを徐々に上げていき、リヤショックアブソーバスペーサおよびナットでリヤショックアブソーバASSY LH(ロワー側)をリヤアクスルビームに仮締めする。

  3. リヤサスペンションサポートおよびリヤショックアブソーバクツションリテーナを取り付ける。




  4. ショックアブソーバASSY　LHの二面幅を固定して、新品のナットを基準値まで締め付ける。

    基準値
    15-18mm




  5. シヨツクアブソーバASSY LHの二面幅を固定して、新品のロツクナットを締め付ける。

    基準値
    T = 24.5 N・m{ 250 kgf・cm }



2. 車両落ち着かせ

  1. リヤタイヤを取り付ける。

    基準値
    T = 103 N・m{ 1050 kgf・cm }

  2. 車両をジャッキダウして数回上下に揺すり、リヤサスペンションを落ち着かせる。



3. ショック アブソーバASSY RR LH本締め



  1. リヤショックアブソーバASSY LH(ロワー側)取り付けナットを本締めする。

    基準値
    T = 49 N・m{ 500 kgf・cm }

    ■ 注 意 ■
    車両を４柱リフトに乗せ、タイヤを接地状態にする。
