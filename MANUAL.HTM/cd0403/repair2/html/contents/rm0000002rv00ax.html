    ■ Attention ■
    After turning on the IG switch, there are items that cannot be inspected unless the diagnodes mode is launched after confirming that the map screen appears.

    □ Participation □
    There are two types of diagnostic startup methods.



    1. Starting method 1


      1. Make the vehicle stopped.

      2. Start ACC, ON or engine the IG switch.




      3. Navigation receiver ASSY "information · G" is pressed, and the head lamp Dimaswitch is operated as OFF → Tail → OFF → OFF → Tail → OFF.




      4. When the diagnostic is started, the "service inspection" screen is displayed and a service test is performed.




      5. Select the "Inspection Menu" and display the diagram inspection menu screen.

        1. Display color bar inspection


          1. Start diagnostic.

          2. Select "Inspection Menu".

          3. Select "Display Inspection".

          4. Select "Color Bar Test".

          5. Confirm that the configuration color and the display color match.

            □ Participation □
            1. When you select black, red, green, blue, white or horizontal stripes, display the selected colors or stripes throughout the screen.
            2. Touch the screen and return.
            3. If the image quality is adjusted, display it according to the brightness of the set.

          6. Confirm that there is no difference compared to the display color of the navigation color bar inspection.

            □ Participation □
            1. If the display color bar inspection result is abnormal, the navigation receiver ASSY is bad.
            2. If the display result of the display color bar inspection is normal and the navigation color bar inspection result is abnormal, the navigation computer or navigation receiver ASSY ← → Navigation Computer visual signal vehicle Wire harness is bad.




        2. Display touch switch inspection


          1. Start diagnostic.

          2. Select "Inspection Menu".

          3. Select "Display Inspection".

            □ Participation □

            2. If the display test result is abnormal, the navigation receiver ASSY is bad.

          4. Memory evacuation / reading

            □ Participation □
            By using a memory catcher when replacing the navigation computer, the memory information of the customer currently in the navigation system in use can be temporarily evacuated, and it can be loaded into the next device.


            1. Cut the connector of the navigation computer and connect the memory catcher.

            2. Start diagnostic.

            3. Select "Inspection Menu".

            4. Select "Navi Inspection".

            5. Select "Memory evacuation / reading".

            6. Select "Evacuate to external memory" to evacuate memory information.

            7. Replace with a new navigation computer.

            8. Connect a memory catcher to the navigation computer.

            9. Start diagnostic.

            10. Select "Inspection Menu".

            11. Select "Navi Inspection".

            12. Select "Memory evacuation / reading".

            13. Select "Read from external memory" and read memory information.

            14. Cut the memory catcher from the navigation computer and connect to the vehicle wire harness.

              □ Participation □
              For details on the operation procedure, refer to the instruction manual attached to the memory catcher.

            1. Screen noise automatic adjustment

              □ Participation □
              1. This function is performed to reduce noise when replacing any of the navigation receiver ASSY, navigation computer or television camera controller.
              2. When replacing the television camera controller, the noise adjustment is also performed at the same time as the camera optical axis adjustment and the steering angle adjustment after replacement.



              1. Navigation screen noise automatic adjustment

                □ Participation □
                Navigation receiver ASSY, navigation computer or television camera controller is replaced to reduce noise.


                1. Start diagnostic.

                  □ Participation □
                  After about 6 seconds or more after IG Switch ACC, launch the diagnosis.

                2. Select "Inspection Menu".




                3. Select "Navi Inspection".




                4. The correction is completed when the beep sounds and the navigation test menu screen is displayed.

                  □ Participation □
                  1. Confirm that the mark (orange) of the adjustment completion is displayed at the position of the figure.
                  2. If the navigation receiver ASSY is replaced and the navigation screen noise adjustment is not performed, the beep sounds are blown for about 3 seconds after the initial screen is displayed when the power is turned on.
                  3. If you replace the navigation computer and television camera controller, the beep sound will not blew.
                  4. If the beep sounds, automatically adjust again.

                5. Ends the diagnostic.



              2. Back guide monitor screen noise automatic adjustment

                □ Participation □
                1. Navigation receiver ASSY, navigation computer, or television camera controller is replaced to reduce noise.
                2. When replacing the television camera controller, the noise adjustment is also performed at the same time as the camera optical axis adjustment and the steering angle adjustment after replacement.


                1. Start diagnostic.

                  □ Participation □
                  After about 6 seconds or more after IG Switch ACC, launch the diagnosis.

                2. Select "Inspection Menu".




                3. Select "Camera Test".




                4. The correction is completed when the steering angle adjustment screen is displayed.
