フロント エアバッグ センサ LH  取り付け

準備品一覧



1. フロント エアバッグ センサ LH取り付け

■ 注 意 ■
  1. 落下品は使用しない。
  2. センサに衝撃や力を加えない。

  1. IGスイッチがOFF(LOCK)になっていることを確認する。

  2. バツテリのマイナスターミナルがはずしてあることを確認する。

    ■ 警 告 ■
    ターミナルをはずした場合は、その後90秒間作業を始めない。




  3. フロントエアバツグセンサLHをボルト2本で取り付ける。

    基準値
    T = 17.5 N・m{ 178 kgf・cm }

  4. フロントエアバッグセンサLHにコネクタを接続する。

  5. フロントエアバッグセンサLHにガタがないことを確認する。



2. エア クリーナASSY取り付け

  1. ボルト4本で、エアクリーナケース　W/エアクリーナインレットNo.1およびNo.2を取り付ける。

    基準値
    エアクリーナケースASSY × エアクリーナブラケット
    T = 7.8 N・m{ 80 kgf・cm }
    エアクリーナインレットNo.2 × ボデー
    T = 7.5 N・m{ 76 kgf・cm }

  2. エアクリーナフィルタエレメントを取り付ける。

  3. エアクリーナキャップ　W/エアクリーナホースNo.1を取り付ける。



3. バッテリ取り付け

  1. バッテリトレイを取り付ける。

  2. バッテリを取り付ける。

  3. ボルトおよびナットでバッテリクランプを取り付ける。



4. エアバッグウォーニングランプ点検
(要領は参照)
