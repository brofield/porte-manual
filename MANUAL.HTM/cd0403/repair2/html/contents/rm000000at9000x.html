If both smart door an locks and locks are not operated, the door -electrical oxylator key does not send the request code, the customization function has set the system, or the electrical door lock system itself is considered abnormal.

■ Attention ■
After confirming that the customization of the smart door lock system has been canceled, proceed with trouble shooting.


□ Participation □
If you select [Smart Cancellation Mode] from the Electric Key Transmitter registration, you will be smartly canceled.Again, if you select [Smart Cancel Mode], your smart function will return.
