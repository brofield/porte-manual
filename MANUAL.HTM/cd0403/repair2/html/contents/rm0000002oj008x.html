        □ Participation □
        1. After replacing the navigation computer, it is necessary to contact the G-Book Support Center and set the vehicle linking of the G-Book terminal ID.
        2. In order to use the G-Book online service, you need to wear a hands-free kit or telematic custlan saver (data communication module).




      3. Telemati Castle Seavor (data communication module) car installed car

        □ Participation □

        2. Voice calls cannot be made on Telemati Caser Seava (data communication module).
        3. After replacing the navigation computer or replacing the telematic custlan saver (data communication module), it is necessary to contact the G-Book Support Center and set the vehicle linking of the G-Book terminal ID and communication device ID.




      4. Hands free kit -mounted car

        □ Participation □
        1. Use a mobile phone for data communication and voice calls.
        2. After replacing the navigation computer, it is necessary to contact the G-Book Support Center and set the vehicle linking of the G-Book terminal ID.




      5. Telemati Castle Runber (data communication module) and hands -free kit -equipped car

        □ Participation □
        1. Use a telematic custlan saver (data communication module) for data communication.
        2. Use a mobile phone for voice calls.
        3. Voice calls cannot be made on Telemati Caser Seava (data communication module).
        4. Data communication on mobile phones is not possible.
        5. After replacing the navigation computer or replacing the telematic custlan saver (data communication module), it is necessary to contact the G-Book Support Center and set the vehicle linking of the G-Book terminal ID and communication device ID.




    2. About G-Book terminal serial ID


      1. The navigation computer has a serial No. (G-Book terminal ID) that can identify the terminal and is used for individual recognition on service (listed in the terminal label barcode).Therefore, when replacing the G-Book device, contact the G-Book Support Center and re-register the serial No. (G-Book terminal ID).




    3. About data communication modules


      1. The data communication module (data communication dedicated radio) is used by G-Book terminal to provide network services.

      2. The data communication module has a phone number, like mobile phones, etc. (listed in the label barcode of the main unit).When replacing the data communication module, you will need to contact the G-Book Support Center, re-registration of the phone number (communication module ID) and delete the old product.

        □ Participation □
        1. The supplied telemata custlan saver (data communication module) has a telephone number registered by the prescribed procedure.
        2. The support center is the registration of the old product.

    4. About mobile phones for data communication


      1. Use the mobile phone held by the user for data communication.

      2. For data communication, a model that has been confirmed by the operation has been used for PDC (general digital method) or CDMA (CDMAONE, CDMA2000 1X).

      3. If you use a mobile phone to use the G-Book online service, you need to connect to g-book.com and register your phone number.

    5. About GPS antenna


      1. It is equipped with a navigation dedicated GPS antenna in the instrument panel.
