  1. Preparation before inspection

    1. Confirm that the throttle valve is fully closed.

    2. Make the shift position in N range or P range.

    3. Turn off the air conditioner.



    1. Check wire harness and connector instantaneous check

      □ Participation □
      If a defect system is found by the diagnostic cord (check mode) inspection diagnostic code output, the defective part is narrowed down by the following method.

      1. After reading the diagnostic code in the check mode, erase the memory of the diagnostic cord.

      2. Select the check mode and start the engine.

      3. Shake the defective wire harness and connector of the defective system output in the diagnostic cord (check mode) inspection while the idle rotation state remains.

      4. If the check engine warnin gland lights up with the wire harness and connector, there is a risk of not contacting the wire harness or connector at that point.
