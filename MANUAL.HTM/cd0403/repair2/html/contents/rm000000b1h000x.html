DTC B0103/12 Ride seat Airbag Ignit circuit (+B short circuit)

printing

List of preparation


Circuit explanation


The driver's airbag spontaneous circuit consists of an airbag sensor ASSY CTR, a spiral cable and a horn button ASSY.When the operation condition is satisfied, the airbag is activated.

When a+B is detected in the driver's airbag icon (D+, D-line), a diag code B0103/12 is recorded in the airbag sensor ASSY CTR.


DTC No.
DTC detection item
Place of problem
B0103/12
  1. Airbag Sensor ASSY CTR-Horn button ASSY (driver's side airbag) D+, D-line+B short circuit
  2. Horn button ASSY (driver's airbag) failure
  3. Spiral cable failure
  4. Airbag sensor ASSY CTR failure
  1. Horn button ASSY
  2. Spiral cable
  3. Airbag Sensor ASSY CTR
  4. Instrument panel wire (between Airbag Sensor ASSY CTR-Spiral Cable)
