エンジンマウンティング インシュレータ LH（1NZ-FE(4WD)）  取り付け

印刷

準備品一覧



1. エンジンマウンティング インシュレータ LH取り付け



  1. ボルト3本で、エンジンマウンティングインシュレータLHを取り付ける。

    基準値
    T = 64 N・m{ 653 kgf・cm }

  2. アースボルトを取り付ける。

    基準値
    T = 26 N・m{ 260 kgf・cm }




  3. ジャッキを操作して、ボルト2本を取り付ける。

    基準値
    T = 49 N・m{ 500 kgf・cm }



2. エンジンマウンティング ブラケット LH取り付け



  1. ボルトで、エンジンマウンティングブラケットLHを取り付ける。

    基準値
    T = 88 N・m{ 897 kgf・cm }




  2. エンジンマウンティングブラケットの矢印が下向きの状態で取り付いていることを確認する。



3. エンジンアンダ カバー LH取り付け

  1. スクリュー2個およびボルト2本で、エンジンアンダカバーLHを取り付ける。

    基準値
    T = 5.0 N・m{ 50 kgf・cm }
