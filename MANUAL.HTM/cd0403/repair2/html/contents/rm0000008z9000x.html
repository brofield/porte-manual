7. Identification timing inspection

  1. Inspection by TASCAN

    1. Warm the engine.

    2. Connect TASCAN to DLS3.

      SST
      09991-70200

    3. Operate according to the screen display, and display the [ECU data monitor] screen.

    4. Confirm that the ignition time is within the reference value.

      Reference value
      0-14 ° CA BTDC (when Idol rotation)

    5. When the engine rotation is raised, make sure that the ignition timing is quickly progressing.

    6. The [Active Test] screen is displayed, and after the ignition time # 1 is selected, the measurement is performed in the TC terminal ON.

      Reference value
      8-12 ° CA BTDC (when idle rotation)

      ■ Attention ■
      1. Turn off all accessories switches and A/C.
      2. Inspection when electric fan is OFF.
      3. When checking the ignition timing, the shift position is set to the N range.

      □ Participation □
      Inspect that the engine speed is kept to 1000-1300R/min for 5 seconds and the idol rotation.

  2. Inspection by general instrument

    1. Warm the engine.

    2. Remove the cylinder head cover No.2.




    3. Pull out the wire harness in the figure and connect the timing light clip to the wire harness.

      ■ Attention ■
      1. Use a timing light that can detect primary signals.
      2. After the inspection, make sure that the wire harness is taped.




    4. Use the diagnodes check wire No.2 to shorten the DLC3 13 (TC) ← → 4 (CG) terminal.

      SST
      09843-18040

      ■ Attention ■
      1. If you make a mistake in the short circuit position, you will never make a mistake because it will cause a failure.
      2. Turn off all accessories switches and A/C.
      3. Inspection when electric fan is OFF.

    5. Confirm that the ignition time is within the reference value.

      Reference value
      8-12 ° BTDC (when idle rotation)

    6. Open between 13 (TC) ← → 4 (CG) of DLC3.

    7. Confirm that the ignition time is within the reference value.

      Reference value
      0-14 ° BTDC (when idle rotation)

    8. When the engine speed is increased, make sure that the ignition timing is quickly progressing.

9. Compression inspection

  1. Warm the engine.

  2. Remove the cylinder head cover No.2.

  3. Take off the ignition coil.

  4. Take off the spark plug.

  5. Check the compression.

    1. Attach a compression gauge to the plug hole.

    2. Make the throttle fully open.

    3. Crank the engine and measure the compression.

      Reference value
      1471kpa {15.0kgf/cm2}

      limit
      1079kPa {11.0kgf/cm2}

      Tube difference
      98kPa {1.0kgf/cm2}

      ■ Attention ■
      1. Use a full charging battery to the engine speed of 250R/min or more.
      2. Other cylinders are also inspected in the same way.
      3. Compression measurement is performed in a short time.




    4. If the compression pressure is low, pour a small amount of engine oil into the cylinder and measure the compression again.

      □ Participation □
      1. If the compression rises, there is a possibility of wearing a piston ring.
      2. If the compression does not change, there is a possibility of valve -related defective.
