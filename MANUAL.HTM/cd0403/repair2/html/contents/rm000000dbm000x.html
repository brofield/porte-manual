  1. Panel diagnostic function


    1. The heater and air conditioner system can check the contents shown in the table below by launching the panel diagnostic.

      Check content
      		Content
      Indica check
      		The activation indicator and the configuration display part of each switch can be flashed
      Sensor chak
      1. You can check the sensor failure status (past, present)
      2. Clearing past failure data is possible
      Actuator check
      		Output of actuator check pattern is possible





    1. Inspection with panel diagnostic (panel diagnostic activation)



      1. OFF the ignitsui sionsuitsu.

      2. Press the "auto" switch and "air conditioner" switch on the control panel at the same time to start the engine.



      1. Inspection with panel diagnostic (indica check)



        1. When the panel diagnostic is started, an indica check is performed automatically.

        2. Confirm that all indicators and configuration display portions light up and turn off four times at one second.

          □ Participation □
          After the indicator check, start automatically check and actuators.

        3. When the panel diagnostic is terminated, press the "F/D" switch.



        1. Inspection with panel diagnostic (sensor chak)

          1. Perform indicators.

          2. When the indica check is completed, the sensor chak is automatically performed.

          3. Perform all check mode in the actuator check.

            ■ Attention ■
            At the sensor chak that automatically performed from the indica check, the diagnosis of failure is not currently possible, so be sure to check the actuata after checking the actuator again and check the result.




          4. The result of the sensor chak is displayed by flashing the indicator portion.

            □ Participation □
            1. The figure shows that the code "face" (a broken insurer failure) is output.
            2. If there are two or more abnormal code, the blinking is displayed at the same time.

          5. If the sensor chaeck code is output, read the code from the diagram code list.

          6. When the panel diagnostic is terminated, press the "F/D" switch.



          1. Inspection by panel diagnostic (sensor chaeck code memory erasure)

            1. Inspect, repair or replace the problem.



            2. Perform the following work and erase the memory of the code (currently failed).





              1. Press the "Rrdef" switch (b) while pressing the "DEF" switch (A) during the sensor chak.

                □ Participation □
                Even if you remove the "ECU-B" fuse of the engine room relay block ASSY, you can erase the memory of the code.

            3. After deleting the memory, perform a sensor chak and confirm that the normal code is output.



            1. Inspection with panel diagnostic (actuator check)

              1. Start the engine and warm enough.




              2. Perform indicators.

              3. After the indicator check, automatically shifts to sensoracheck and actuator check mode.

                □ Participation □
                When performing an actuator check, be sure to perform the engine start.

              4. Since the temperature setting switch is changed to each step by changing the setting temperature of the switch, check the temperature and air volume by visible, hand.

                □ Participation □
                1. The figure shows the case where the step "4" is output.
                2. The result of the sensor chaeck is also displayed when performing each actuata.

              5. When the panel diagnostic is terminated, press the "F/D" switch.
