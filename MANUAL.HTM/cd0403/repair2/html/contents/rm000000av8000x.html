  1. Inspection by self -diagnosis mode

    □ Participation □
    1. The self -diagnostic mode function includes [operation confirmation mode], [input confirmation mode], and [abnormal operation memory mode].
    2. The operation confirmation mode is detected by a buzzer by performing a determined door operation and a buzzer.
    3. The input confirmation mode detects the current switch input and informs the buzzer.
    4. The abnormal operation memory mode remembers the switch abnormality detected during the use process, and reads the memorable abnormalities.
    5. Inspection of self -diagnosis mode is divided into [Self -Diagnosis Mode 1], [Self -Diagnosis Mode 2], and [Self -Diagnosis Mode 3].
    6. In the self -diagnosis mode 1, the operation confirmation mode is detected and the input confirmation mode function is performed.
    7. In the self -diagnosis mode 2, a confirmation function by the buzzer in the operation confirmation mode is performed.
    8. In the self -diagnosis mode 3, the failure cord is read by the buzzer output by the abnormal operation memory mode.
