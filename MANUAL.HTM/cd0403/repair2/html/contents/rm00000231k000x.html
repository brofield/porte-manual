■ Attention ■

  1. AVN

    □ Participation □
    1. Logical address 01: communication management
    2. Logical address 5c: Camera unit


    Service inspection screen display Name: AVN, parts name: Navigation receiver ASSY, physical address: 140
    Logical address-Diag code
    		Diagnostic item
    		Diagnostic content
    		treatment
    		remarks
    01-D6
    		Master absence
    		The device that remembers this code has been out of the system at the time of IG switch ACC, or the master device has come off when this code is memorized.
    		Inspection of the power supply system and AVC-LAN communication system W/H of the navigation receiver ASSY
    		This code may be memorized by the battery status and the power waveform at the start of the engine, even if no abnormality is found.
    01-D7
    		Connection confirmation abnormality
    		The device that remembers this code may have been out of the system after the engine started, or the master device was out of the way when this code was memorized.
    		Inspection of the power supply system and AVC-LAN communication system W/H of the navigation receiver ASSY
    		This code remembers that the power of the navigation receiver ASSY is turned off after the engine is started, and it will be 210 seconds.
    01-DC
    		Failure
    		Sending has failed to the device shown in the auxiliary code (this diag code may not be directly linked to abnormalities).
    		If the equipment shown in the auxiliary code is also remembered in other devices, check the power supply system and communication system W/H of the device indicated by the auxiliary code.
    		This code may remember that the IG switch is Start again 60 seconds after the engine is started.
    01-DD
    		Master reset (instantaneous)
    		After the engine started, the master equipment has come off the system
    1. Inspection of the power supply system and AVC-LAN communication system W/H of the navigation receiver ASSY
    2. If abnormalities occur frequently, replace the navigation receiver ASSY
    		This code may be remembered as the IG switch is Start again after the engine is started.
    01-E0
    		Registration completion notification abnormality
    		"Registration completion notification" command from the master cannot be received
    		This diagram is an engineering diagram, which is also detected when it is not directly connected to the abnormality.
    		This code may be memorized by the battery status and the power waveform at the start of the engine, even if no abnormality is found.
    01-E2
    		ON/OFF instruction parameter abnormality
    		There was an abnormality in the command to control ON/OFF from the master device
    		Replace the navigation receiver ASSY
    		-
    5C-40
    		Camera video abnormality
    		There is no synchronous signal from the camera
    1. Check the W/H between the television camera ASSY and the navigation receiver ASSY
    2. Replace television camera ASSY
    		-
    5C-41
    		Steering sensor abnormal
    		Detected the abnormality of the steering sensor
    		Replace the steering sensor
    		-
    5C-42
    		Steering sensor communication abnormality
    		There is no signal from the steering sensor
    		See Trouble Shooting Flow Chart [The expected course is not displayed and "cannot be guided" is displayed]
