準備品一覧



1. カムポジション センサ取り付け

  1. Oリングに少量のエンジンオイルを塗布する。

  2. ボルトでカムポジシヨンセンサを取り付ける。

    基準値
    T = 8 N・m{ 82 kgf・cm }

    ■ 注 意 ■
    Oリングの切れおよび噛み込みがないようにシリンダヘツドに取り付ける。

  3. コネクタを接続する。



2. エア クリーナASSY取り付け

  1. ボルト４本でエアクリーナケースとエアクリーナインレツトNo.1およびエアクリーナインレツトNo.2を取り付ける。

    基準値
    T = 7.8 N・m{ 80 kgf・cm } (エアクリーナケースASSY　×　エアクリーナブラケツト)
    T = 7.5 N・m{ 76 kgf・cm } (エアクリーナインレツトNo.2　×　ボデー)

  2. エアクリーナフイルタエレメントを取り付ける。

  3. エアクリーナキヤツプとエアクリーナホースNo.1を取り付ける。
