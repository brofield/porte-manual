Installation of transmission wire (U441E)

printing

List of preparation



1. Trans mission wire installation



  1. Apply auto fluid type T-IV to the O-ring of the transmission wire (with oil temperature sensor).

  2. Pass the transmission wire (with oil temperature sensor) to the automatic transxle.




  3. In the bolt, fix the transmission wire (with an oil temperature sensor).

    Reference value
    T = 5.4 n ・ m {55 kgf / cm}



2. Trans mission valve body installation



  1. Confirm that the position in the figure contains a check ball, applied gasket and each piston.




  2. Attach a manual valve link and attach a valve body ASSY.




  3. Tighten 14 bolts evenly.

    Reference value
    T = 11 N ・ M {112 kgf / cm}

  4. Attach each solenoid connector and oil temperature sensor.



3. Valve body oil strainer Assy installation



  1. Attach a new gasket to the oil strainer.

  2. Attach the oil strainer with three bolts.

    Reference value
    T = 9.8 n ・ m {100 kgf / cm}



4. Automatic transformer axle oil bread SUB-ASSY (ATM) Installation



  1. Attach the two oil cleaner magnet to the position of the figure.




  2. Attach the oil pan with 18 bolts.

    Reference value
    T = 7.0 n ・ m {71 kgf / cm}

6. Tascan-at learning value Initialization

  1. Use SST (TASCAN) to perform the operation according to the screen display, select [Work Support] in the main menu, and select Trans Mission in the work support category.

    SST
    09991-70200

  2. Select [AT learning value initialization] on the screen display and initialize the AT learning value.

    ■ Attention ■
    When replacing any of the Transxle ASSY, Valve Body ASSY, transmission wires, engine and engine control computers, the AT learning value is initialized.

    □ Participation □
    The learning value is not initialized just by removing the battery.
