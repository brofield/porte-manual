Step 1TASCAN-AT Learning value Initialization (initialization of learning)

    SST
    09991-70200

  1. Use Tascan to operate according to the screen display, and select a transmission for the work support category.

  2. Select [AT learning value initialization] on the screen display and initialize the AT learning value.

    ■ Attention ■
    When replacing either Transxle ASSY, Valve Body ASSY, transmission wire, engine and engine controlled compilation, the AT learning value is initialized.

    □ Participation □
    The learning value is not initialized just by removing the battery.
