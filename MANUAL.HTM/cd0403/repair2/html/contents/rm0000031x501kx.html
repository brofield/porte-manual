Circuit explanation





When a misfire is generated by the engine, hydrocarbons (HC) are discharged during exhaust gas.When the HC concentration rises, the exhaust emission level deteriorates.At the same time, the rise in HC concentration may cause the temperature rise of the catalyst to impair the purification ability of the catalyst.To prevent the increase in exhaust emissions and prevent the heat deterioration of the catalyst, the engine control computer monitors the misfire rate (a value of the number of misconfires by the number of ignitions).The engine control computer uses a cam position and a crank position sensor to monitor the misfire.The crank position sensor is used to measure the crankshaft rotation speed.When the change in the crankshaft rotation speed exceeds the judgment value, the engine control computer is judged to be a failure, and counts a misfire counter (multiplied by the number of misfires).

The cam position sensor is used to identify the incredible cylinder.




When the failure rate exceeds the judgment value, the engine control computer lights or flashes the check engine warnin gland.

Catalytic overtempalacha level when detection

  1. When the fire rate is determined for each engine 200 revolutions, and the accidental failure (several tens of or more) occurs three times, the engine control computer immediately blinks the check engine warnin glamp.

Emission deterioration level at the time of failure

  1. When the fire rate is determined for each engine 1000 revolutions, the engine control computers light up the check engine warning lamp when the emissions exceed the judgment value (several%or more) (2 trips).

Inspection procedure


■ Attention ■
Check the fuse of this circuit before performing trouble shooting.


□ Participation □
  1. If the diag code other than the misfire diag code is remembered at the same time, the diag code other than the misfire diag code is repaired first.
  2. Read freeze frame data using SST (Tascan).The freeze -frame data is a part of the engine operation state in the event of a defect, which is useful for troubleshooting.
  3. If the phenomenon cannot be confirmed, the state of the freeze frame data is reproduced.
  4. If either F/B Correction value Bank 1 or F/B Learn Bank 1 of the freeze-frame data is out of the range of ± 20 %, the air-fuel ratio is rich (-20 %or less) or lean (+20 %or more).It may be.
  5. If the engine cooling water temperature of the freeze -frame data is less than 80 ° C, the fire may only be generated during the engine warm -up.
  6. The reason why the misfire cannot be reproduced is the lack of fuel, the use of inappropriate fuel, and the dirt on the spark plug.
  7. After repair, check the value of the misfire counters #1- #4 and the misfire counter.
  8. If the wheel balance of the drive wheel is significantly shifted enough to cause a body vibration, the dial cord of the misfire may be detected.
