ABS & TRC & VSC & BA system test mode with EBD

printing



  1. Test mode inspection

    □ Participation □
    1. When entering the test mode, all testmoodododes are memorized once, and the memory of the code determined that the computer is normal at the time of inspection is deleted.
    2. During the test mode, VSC does not work, regardless of normal and abnormal sensor chaeck.
    3. When you return to the normal mode, all test mode code will be deleted.
    4. If the sensor is abnormal, the ABS and VSC Warnin Gramp will light up.

    1. Test mode How to start


      1. OFF the IG switch and connect SST (Tascan) to DLC3.

        SST
        09991-70200

      2. Make a shift lever P range.

      3. Turn on the IG switch and switch to test mode according to the SST (Tascan) display screen.



    2. Test mode display confirmation





      1. When the test mode starts, the ABS and VSC Warnin Gramp displays test mode.

    3. Joylete sensor & G sensor 0 check


      1. Make the shift lever in the P range with IG switch ON and stop for 2 seconds or more in a flat place.

        □ Participation □
        If the sensor is normal and the above conditions are established when the test mode is launched, it has ended normal.



    4. Yorate sensor output check





      1. Make a 180 ° turning run without wheel spinning.

        ■ Attention ■
        1. The direction of vehicle at the start and end of the turn shall be 180 ± 5 ° or less.
        2. During the turn, do not make the shift lever in the P range.Also, do not turn off the IG switch.

        □ Participation □
        1. The turn direction is good on either side.
        2. You may perform the vehicle speed, stop, or retreat during turning, but discontinue the work as soon as possible.

      2. When the shift lever is made in the P range, the buzzer sounds for 3 seconds and confirms that the sensor chak has ended.

        ■ Attention ■
        1. If the buzzer does not sound, check the sensor chaeck code.
        2. If there is no abnormality in the sensor chaeck code and the buzzer does not blew even after several sensor chaks, check the buzzer circuit.

        □ Participation □
        Even if the check of the iodide sensor is completed normally, the VSC Warnin Gramp outputs the test mode display.

    5. Master cylinder pressure sensor chaeck


      1. After opening the brake pedal for more than 1 second while stopping the vehicle, confirm that the ABS Warnin Grand Lamp lights up for 3 seconds when stepping on the pedal with a step force of 98n {10kgf} or more.

        □ Participation □
        1. While the ABS Warnin Grand Ped, the brake pedal is stepped on with the step force 98n {10kgf} or more.
        2. The ABS Warnin Grand Lamp turns for about 3 seconds and turns on every time the above pedal operation is performed.
        3. If the master cylinder pressure sensor chaeck does not end, stepping on the brake unnecessarily may further lower the negative pressure and may be less likely to end sensor chak.
        4. When negative pressure is insufficient, the master cylinder pressure sensor chaeck may not be terminated.In that case, the engine is rotated and the negative pressure is secured.
        5. If the brake is strongly stepped on when the negative pressure is insufficient, the brake warnin gland may light up due to the loss control.In this case, the engine is rotated to ensure sufficient negative pressure.



    6. Speed ​​sensor chak




      1. Start without wheel spinning on the handle straight, speed up to 45km/h or higher, and confirm that the ABS Warnin Grand Lamp will turn off.

        ■ Attention ■
        1. Before performing the speed sensor, discontinue the yolly sensor, deceleration sensor and master pressure sensoracheck.
        2. If you start a sensor chaeck with the steering state, the speed sensor chaeck may not end if the wheel spin is made.
        3. After turning off the ABS Warnin Gramp, if you drive at 80 km/h or more, you will remember the sensor chakcode again, so it will decelerate and stop before reaching 80km/h.
        4. In the case of the sensor chak, the ABS Warnin Gramp flashes while driving, and the ABS does not work.

        □ Participation □
        When the speed sensor chaeck is finished normally, the ABS Warnin Gramp will turn off while driving, and flashes test mode during the stop.



    7. End of sensor chak


      1. If the sensor chaeck is finished successfully, the ABS Warning Lamp lit state will blink during the vehicle stop (test mode display) and turn off during vehicle driving.

        ■ Attention ■
        1. The sensor chak ends when the yolly G sensor, speed sensor and master cylinder pressure sensor chaeck end normally.
        2. In the case of the sensor chak, the ABS Warnin Gramp flashes while driving, and the ABS does not work.
        3. If a diag code is detected during test mode, the ABS Warnin Grand Pu, VSC Warnin Grand Publish will light up.

    8. Sensor chakcode and diagnostic cord check


      1. Use SST (Tascan) to check the output of sensoracheck code and diagnostic cord.

        ■ Attention ■
        1. If only the diagnostic code is displayed, repair the problem point and delete the diagnostic cord.After erasing, make sure that the ABS Warnin Gramp is normal.
        2. If only the sensor chaeck code is displayed, perform the test mode test again.
        3. When the diagnostic chord and sensor chaeck code are displayed, the defective part is repaired, the diagnostic code is erased, and the test mode is inspected again.

    9. Test mode termination method


      1. Switch from test mode to normal mode according to the SST (Tascan) screen display.

      2. Turn off the IG switch and remove the SST (Tascan).
