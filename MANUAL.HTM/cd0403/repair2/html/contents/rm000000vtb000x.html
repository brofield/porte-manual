DTC C1290/66 Red angle sensor 0 points abnormal

printing

List of preparation


Function explanation


  1. The skid control computers have obtained 0 steering sensors by driving for about 5 seconds at 35km/h or higher on the IG switch ON, and remembers the past 0 points.
  2. If the alignment is adjusted, the steering wheel position adjustment, etc. are performed without separation of the battery minor stormy, or if the yolly sensor is not obtained after adjustment, the steering sensor 0 points and the memory value at the time of driving are gaps.Detects that there is a steering sensor 0 point abnormal code as a poor adjustment.
  3. The steering angle sensor 0 point abnormal code is warned off by IG switch off.


DTC No.
DTC detection conditions
DTC output -related items
C1290/66
The steering sensor 0 points acquisition position is different from the memory value (the angle is larger)
  1. Joy Rested Sensor 0 unprecedented
  2. Steering straight position poor position
  3. Front wheel alignment adjustment
