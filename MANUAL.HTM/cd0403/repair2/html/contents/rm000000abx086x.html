□ Participation □
The diag code of the power slide door system outputs a failed code detected in self -diagnosis mode by a buzzer.


(*1): Check mode

(*2): Test mode

  1. It is very important to determine whether the diagram code inspection shows the bug shown by the diag code, or that it has occurred in the past but is now normal.In addition, it is necessary to confirm whether the defect indicated by the diag code is directly related to the defective phenomenon.
  2. For this reason, it is necessary to check the diag code before and after confirming the defect phenomenon to determine the current situation.If you neglect this, you will have unnecessary trouble shooting despite the normal operation system, it will be difficult to find out where the problem is, and repairs will be performed unrelated to the problem.Therefore, it is necessary to always check the diag code in the correct work procedure.
