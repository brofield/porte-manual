  1. Diagnostic function

    1. The HDD navigation system has a diagnostic function and can perform a failure diagnosis.(The test result is displayed on the display part)

    2. The devices that make up the AVC-LAN are set with a three-digit number (notation in the hexadecimal number) called a device code (physical address).

    3. Each function (unit) that configures the device that makes AVC-LAN is set with a two-digit number (notation in hexadecimal numbers) called a logical address.
