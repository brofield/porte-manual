Control system CAN communication system CAN bus (Can-H, Can-L)+B short inspection

printing

List of preparation


Function explanation


  1. If there is an conductor between each terminal of DLC3, a short circuit between the CAN bus (Can-H or Can-L) ← →+B is considered.


Symptoms
Suspicated part
DLC3 16 (BAT) ← → 6 (Canh), 14 (CANL).
  1. (1), (2), (3), (8)+B shorts
  2. Brake actuator (VSC ECU)
  3. Steering sensor (steering angle sensor) abnormality
  4. York rate (yaw / G sensor) abnormality
