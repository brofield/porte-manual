    01-D5
    		Registered equipment dropout
    1. The device shown in the auxiliary code is out of the system during IG switch ACC
    2. The device shown in the auxiliary code has been out of the system during the IG switch ACC
    3. The communication with the equipment shown in the auxiliary code has not been secured when starting the engine
    		Inspection of the power supply system and AVC-LAN communication system W/H of devices indicated by the auxiliary code
    		Even if an abnormality is not found, this code may be remembered due to the battery status and the power waveform at the start of the engine.
    01-D8
    		Connection confirmation without answer
    1. The device shown in the auxiliary code is out of the system after starting the engine
    2. The device shown in the auxiliary code has been out of the system after starting the engine
    		Inspection of the power supply system and AVC-LAN communication system W/H of devices indicated by the auxiliary code
    		This code is remembered that the power of the device indicated by the auxiliary code turns off after the engine is started, and it will be 180 seconds.
    01-D9
    		Last mode abnormality
    1. The device that was working (sound, video out) before the engine was turned off, is out of the system during the IG switch ACC.
    2. The device that was working (sound, video out) before the engine was turned off was out of the system during the IG switch ACC.
    		Inspection of the power supply system and AVC-LAN communication system W/H of devices indicated by the auxiliary code
    		Even if an abnormality is not found, this code may be remembered due to the battery status and the power waveform at the start of the engine.
    01-DA
    		ON/OFF instruction without answer
    		The device indicated by the auxiliary code did not respond when switching the sound / video mode (detected when the screen or audio is not switched due to button operation etc.)
    1. Inspection of the power supply system and AVC-LAN communication system W/H of devices indicated by the auxiliary code
    2. If an abnormality occurs, replace the device indicated by the auxiliary code.

    01-DB

    01-DC
    		Failure
    		It has failed to send to the device shown in the auxiliary code (this code may not be directly linked to the abnormality)
    1. If the equipment shown in the auxiliary code is also remembered in other devices, check the power supply system and AVC-LAN communication W/H of the device indicated by the auxiliary code.
    2. If this code is not remembered in other devices, delete the diag code and see for a while.
    		This code may be remembered that the engine starts again 60 seconds after starting the engine.
    01-DE
    		Sleeve reset (instantaneous)
    		After the engine started, the slave device has come off the system
    		Inspection of the power supply system and AVC-LAN communication system W/H of devices indicated by the auxiliary code
