リヤラテラルコントロール ロッドASSY（4WD）  取り付け

印刷

準備品一覧



1. リヤラテラルコントロール ロッドASSY仮締め

  1. ワッシャ２枚を介してラテラルコントロールロッドASSYをリヤアクスルハウジングに取り付け、ナットを仮付けする。




  2. ボルトおよびナットでラテラルコントロールロッドASSYを仮付けする。



2. 車両落ち着かせ

  1. 車両をジャッキダウンして数回上下にゆすり、リヤサスペンションを落ち着かせる。



3. リヤラテラルコントロール ロッドASSY本締め

  1. アクスルハウジングをジャッキなどで支え、サスペンションに荷重をかける。

  2. ボデー取り付け用のボルトを固定し、ナットを締め付ける。

    基準値
    T = 127 N・m{ 1300 kgf・cm }

  3. アクスルハウジング取り付け用のナットを締め付ける。

    基準値
    T = 83 N・m{ 850 kgf・cm }
