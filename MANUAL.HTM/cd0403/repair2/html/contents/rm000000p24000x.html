DTC C1201/51 engine abnormality

printing

List of preparation


Function explanation


  1. When the engine control system is abnormal, the operation of VSC and TRC is prohibited by the fail -safe function.
  2. If the signal input from the engine returns normally, the Fail Safe is released and the diag code is not remembered.
