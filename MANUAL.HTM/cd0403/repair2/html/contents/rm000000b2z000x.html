フロントシートベルトRH  取り付け

印刷

準備品一覧



1. フロントシート インナベルトASSY RH取り付け



  1. ボルトでフロントシート インナベルトASSY RHを取り付ける。

    基準値
    T = 42.0 N・m{ 428 kgf・cm }

  2. ケーブルのクランプを取り付け、コネクタを接続する。



2. フロントシート クッション シールド INN RH取り付け

  1. クッションシールドのツメをかん合させる。

  2. スクリューでフロントシートクッションシールドINN　RHを取り付ける。



3. フロントシート アウタベルトASSY RH取り付け

  1. ELRロック開始の傾斜角度点検




    1. リトラクタを取り付け状態から静かに動かしたとき、全方向に対してベルトのロックが15°以内でロックしないことを点検し、45°以上でロック状態を保持することを点検する。

      ■ 注 意 ■
      リトラクタは分解しない。

  2. ボルト2本で、フロントシートアウタベルトASSY RHのリトラクタ部を仮締めする。

  3. 上部→下部の順にフロントシートアウタベルトASSY RHのリトラクタ部を本締めする。

    基準値
    リトラクタ上部のボルト
    T = 8.5 N・m{ 87 kgf・cm }
    リトラクタ下部のボルト
    T = 42 N・m{ 428 kgf・cm }

  4. ボルトで、フロントシートアウタベルトASSY RHのショルダアンカ部を取り付ける。

    基準値
    T = 42 N・m{ 428 kgf・cm }




  5. コネクタ(プリテンショナ用)を接続し、ロッキングボタンをロックさせる。

    ■ 注 意 ■
    ロッキングボタンを確実にロックする。

  6. ELRロック点検

    1. 車両取り付け状態において、すばやくシートベルトを引き出したとき、シートベルトがロックすることを確認する。



4. シート ベルト アンカ カバー キャップ取り付け



  1. ツメのかん合を合わせ、シートベルトアンカカバーキャップを取り付ける。



5. サイドNO.1 トリムASSY RH取り付け



  1. クリップおよびツメをかん合させサイドNo.1トリムASSYを取り付ける。



6. フロントシート アウタベルトASSY RH取り付け（フロアアンカ部）

  1. ボルトで、フロントシートアウタベルトASSY RHのフロアアンカ部を取り付ける。

    基準値
    T = 42 N・m{ 428 kgf・cm }



7. ラップベルト アウタアンカ カバー取り付け



  1. ツメのかん合を合わせ、ラップベルト アウタアンカ カバーを取り付ける。



8. リヤシート 3ポイントタイプ ベルトASSY OUT RH取り付け（フロアアンカ部）

  1. ボルトでリヤシート 3ポイントタイプ ベルトASSY OUT RH（フロアアンカ部）を締め付ける。

    基準値
    T = 42 N・m{ 428 kgf・cm }



9. フロントドア オープニングトリム ウェザストリップ RH取り付け



  1. ウェザストリップのペイントマーク(白色、どちらか1箇所)を合わせ、フロントドアオープニングトリムウエザストリツプRHを取り付ける。

    ■ 注 意 ■
    取り付け後、コーナー部が確実に取り付けてあることを確認する。



10. フロントドアスカッフ プレート RH取り付け



  1. スカッフプレート前端部および後端部のツメをかん合させる。

  2. ツメをかん合させ、フロントドアスカッフプレートRHを取り付ける。



11. リヤシートクッションASSY取り付け（リヤシート分割可倒式）

  1. リヤシートクッションASSY前側をリヤシート クッション ロック ストライカにロックさせる。

  2. リヤシートベルトを、リヤシートクッションカバー & パッド後部のゴムバンドに通す。




  3. ボルトで、リヤシートヒンジLHを取り付ける。

    基準値
    T = 37 N・m{ 377 kgf・cm }

  4. リヤシートバックヒンジＲＨにスナップリングを取り付ける。

  5. リヤシートヒンジRHを、リヤシートクッションASSY右側ブラケットの角パイプに挿入する。

    □ 参 考 □
    リヤシートヒンジRHのスナップリング溝にスナップリングがはまっていることを確認する。



12. リヤシートバックASSY LH取り付け（リヤシート分割可倒式）

  1. ボルト2本でリヤシートバックを取り付ける。

    基準値
    T = 37 N・m{ 377 kgf・cm }




  2. クリップ2個を取り付ける。



13. リヤシートバックASSY RH取り付け（リヤシート分割可倒式）

  1. ボルト2本でリヤシートバックを取り付ける。

    基準値
    T = 37 N・m{ 377 kgf・cm }

  2. クリップ2個を取り付ける。



14. リヤシートバック ヒンジSUB-ASSY RH取り付け（リヤシート一体可倒式）



  1. ボルトで、リヤシートバツクヒンジRHを締め付ける。

    基準値
    T = 42 N・m{ 428 kgf・cm }



15. リヤシートバック ヒンジSUB-ASSY LH取り付け（リヤシート一体可倒式）

  1. ボルトで、リヤシートバツクヒンジLHを締め付ける。

    基準値
    T = 42 N・m{ 428 kgf・cm }



16. リヤシートクッションASSY取り付け（リヤシート一体可倒式）

  1. リヤシートクッションASSY後部のフックをかん合させる。

  2. リヤシートベルトを、リヤシートクッションカバー & パッド後部のゴムバンドに通す。




  3. リヤシートクッションASSY前部のフックをかん合させる。



17. リヤシートバックASSY取り付け（リヤシート一体可倒式）

  1. ボルト2本をでリヤシートバックASSYを取り付ける。

    基準値
    T = 37 N・m{ 377 kgf・cm }




  2. クリップ2個を取り付ける。



18. スライドアジャスターロック同期点検

  1. スライド操作したとき、左右のアジャスターが同時にロックすることを点検する。

    □ 参 考 □
    スライドの同期ずれがある場合は、フロントシート固定ボルトをゆるめて調整する。



19. シートベルト機能点検

  1. イグニッションスイッチをONにする。

  2. フロントシートＡＳＳＹ　RHに座り、シートベルトのタングプレートをバックルに挿入していないとき、運転席シートベルトウォーニングインジケータが点滅することを点検する。

  3. シートベルトのタングプレートをバックルに挿入したとき、コンビネーションメータASSYの運転席シートベルトウォーニングインジケータが消灯することを点検する。
