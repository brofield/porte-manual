1. クランクポジション センサ取り付け

  1. Oリングに少量のエンジンオイルを塗布する。




  2. ボルトでクランクポジシヨンセンサを取り付ける。

    基準値
    T = 7.5 N・m{ 76 kgf・cm }

    ■ 注 意 ■
    Oリングの切れおよび噛み込みがないようにオイルポンプに取り付ける。

  3. コネクタを接続する。



2. エンジンアンダ カバー RH取り付け

  1. スクリュー2個およびボルト2本で、エンジンアンダーカバーRHを取り付ける。

    基準値
    T = 5.0 N・m{ 50 kgf・cm }

  2. ナットを、締め付ける。

    □ 参 考 □
    ナットは、LH側と共締めになっている。

    基準値
    T = 5.0 N・m{ 50 kgf・cm }
