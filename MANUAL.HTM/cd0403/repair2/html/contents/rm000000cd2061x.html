EFI System Freeze Frame Data

printing



  1. Freeze -frame data confirmation (read by Tascan)

    1. Use SST (TASCAN) to perform the operation according to the screen display, and select a diagram code (display F) that records freeze-frame data from the Diagnosis Cord Confirmation screen.

      SST
      09991-70201

      ■ Attention ■
      1. The freeze -frame data remembers the engine status at the time of three diameter cord detection.
      2. The freeze -frame data is classified as follows, and the freeze -frame data of one diagcode per group is memorized.


      Group classification
      		Diag code outputs misfire (P0300, P0301, P0302, P0304), rich abnormal (p0172), lean abnormality (P0171)
      Diag code with the engine check lamp lights up
      Diag code that does not light the engine check lamp

      □ Participation □
      1. In the time series freeze frame data, the engine status (ECU data) before and after the diagcode detection can be confirmed.
      2. By using chronological freeze frame data, it is useful for trouble shooting when the phenomenon cannot be reproduced.
