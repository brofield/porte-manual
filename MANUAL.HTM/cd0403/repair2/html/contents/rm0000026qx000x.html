1. リヤディファレンシャル キャリア ガスケット取り付け

  1. 白ガソリンを使用して、リヤディファレンシャルキャリアASSYおよびリヤアクスルハウジングASSYの接合面のゴミ、油脂を取り除く。

  2. 新品のリヤディファレンシャルキャリアガスケットの両面にシールパッキン1281を塗布し、リヤアクスルハウジングASSYにリヤディファレンシャルキャリアガスケットを取り付ける。



2. ディファレンシャル キャリヤASSY RR取り付け

  1. ジャッキにディファレンシャルキャリアASSYをのせジャッキアップし、リヤアクスルハウジングASSYに取り付ける。

    ■ 注 意 ■
    1. ディファレンシャルは重量物のため、ジャッキで確実に支える。
    2. ディファレンシャルキャリアASSYを取り付ける際に、取り付け面を傷つけない。




  2. 取り付けボルトのねじ部にアドヘシブ1344を塗布し、ナットおよびスプリングワッシャー各10個にてディファレンシャルキャリアASSYを取り付ける。

    基準値
    T = 31 N・m{ 316 kgf・cm }



3. プロペラシヤフトASSY RR取り付け



  1. インタミディエイトシャフトASSY およびプロペラシャフトASSY RR の合わせマークを合わせ、ボルト４本、ワッシャおよびナット各４個で、インタミディエイ トシャフトASSY を取り付ける。

    基準値
    T = 37 N・m{ 377 kgf・cm }

    ■ 注 意 ■
    1. ワッシャはボルト側に取り付ける。
    2. 締め付け時、ワッシャおよびナットに油脂類を付着させない。




  2. プロペラシャフトASSY RR およびディファレンシャルフランジ の合わせマークを合わせ、ボルト４本、ワッシャおよびナット各４個で、プロペラシャフトASSY RR を取り付ける。

    基準値
    T = 35 N・m{ 357 kgf・cm }

    ■ 注 意 ■
    1. ワッシャはボルト側に取り付ける。
    2. 締め付け時、ワッシャおよびナットに油脂類を付着させない。



4. リヤアクスル ハウジング エンド ガスケット取り付け（LH側）

  1. 白ガソリンを使用して、バッキングプレートおよびアクスルハウジングを清掃する。




  2. バッキングプレートを浮かせてエンドガスケットを取り付ける。

    ■ 注 意 ■
    1. ブレーキチューブに無理な力を加えない。
    2. ガスケットの切り欠きを下側にする。

  3. ボルト4本をセットする。

  2. ナット4個で、ベアリングリテーナOUTをバッキングプレートに取り付ける。

15. ブレーキドラムとライニングのすき間調整



  1. バッキングプレートのサービスホールからマイナスドライバーでアジャスターを回して拡張させ、ブレーキシューASSYをリヤブレーキドラムに接触させる。




  2. マイナスドライバーでリヤブレーキオートマチックアジャストレバーLHを押しながらもう1本のマイナスドライバーでアジャスターを回して収縮させる。

  3. ブレーキシューASSYがリヤブレーキドラムに接触しなくなった状態から更にアジャスターを1/2回転させて収縮させる。

    3. 取り付ける際に、センサワイヤがねじれないようにする。

18. ディファレンシャルオイル補充

  1. リヤアクスルハウジングフィラプラグおよびガスケットを取りはずす。




  2. リヤアクスルハウジングフィラプラグ穴下端から0-5mm以内にオイルがあることを確認する。

    ■ 注 意 ■
    1. オイル過多、過小はトラブルの原因になる。
    2. 車両を平坦路に停止させる。

  3. オイル量が少ない場合は、オイル漏れ点検後、オイルを補充する。

  4. 新品のガスケットを介してリヤアクスルハウジングフィラプラグを締め付ける。

    基準値
    T = 49 N・m{ 500 kgf・cm }

    ■ 注 意 ■
    オイル交換時は、走行後オイル量を再確認する。
