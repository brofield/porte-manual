    1. Precautions for exploration of failure

    1. In the case of a defective point of contact with the terminal or a defective part mounting condition, if the connector is reconnected or the relevant parts are removed, the normal return may be returned or temporarily returned.

    2. In order to clearly select a defective point, each connector is cut off, and the operation status is checked in the event of a defect in the event of a defect, such as confirming the output of the diagnostic cord, freeze -frame data output, and the record of the freeze -frame data output.。

    3. It may be possible to be affected by a problem other than the brake control system, so also check the abnormal diagnostic code of other systems.



      2. Handling, work precautions

      1. Unless otherwise specified as an inspection method, the actuator and each sensor should be removed, and the installation is always performed in the condition of the ignition switch off.

      2. When the actuator and each sensor are removed, etc., when the mounting work is performed, the test mode inspection, the diagnostic code output inspection, and the output of the normal display are confirmed.

      3. VSC -related parts may not be removed except when necessary, as the adjustment may cause incomplete adjustments.

      4. When performing VSC -related work, be sure to prepare for work and confirm the work completion according to the instructions in the text.



        3. Fail safe function

        1. If an abnormality occurs in a part of the ABS · TRC · VSC and brake assist systems, each warnin group (ABS, VSC, brake) will be turned on, and the operation of ABS, TRC, VSC, EBD and brake assists will be discontinued at the same time.
