1. トランスフアオイル点検

  1. トランスファケースプラグNo.1（フィラープラグ）を取りはずす。




  2. トランスファケースプラグNo.1（フィラープラグ）からガスケットを取りはずす。




  3. トランスファケースプラグNo.1（フィラープラグ）穴下端から0-5mm以内にオイルがあることを確認する。

    ■ 注 意 ■
    1. オイル交換時は、走行後オイル量を再確認する。
    2. オイル過多、過少の場合はトラブルの原因になる。
    3. オイル量が少ないときは、オイル漏れを点検する。

  4. 新品のガスケットを介して、トランスファケースプラグNo.1（フィラープラグ）を取り付ける。

    基準値
    T = 49 N・m{ 500 kgf・cm }



2. トランスフアオイル調整

  1. トランスファケースプラグNo.1（フィラープラグ）を取りはずす。

  2. トランスファケースプラグNo.1（フィラープラグ）からガスケットを取りはずす。




  3. ノズルをさし、トヨタ純正ギヤオイルスーパー(SAE-75W-90 GL-5)を注入する。

    ■ 注 意 ■
    1. ノズル先端を車両前方に向ける。
    2. ノズルをさす向きを間違えると、オイルが吹き出すおそれがある。
    3. オイルはゆっくり注入する。
    4. オイル注入作業は数分間隔で数回繰り返す。

  4. 5分間放置後、再度オイル量を点検する。

  5. 新品のガスケットを介して、トランスファケースプラグNo.1（フィラープラグ）を取り付ける。

    基準値
    T = 49 N・m{ 500 kgf・cm }
