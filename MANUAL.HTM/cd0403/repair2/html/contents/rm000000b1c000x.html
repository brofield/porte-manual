Circuit explanation


The front airbag sensor RH circuit consists of airbag sensor ASSY CTR and front airbag sensor RH.

The front airbag sensor RH detects the impact and sends a signal to make a ignition judgment of the airbag to the airbag sensor ASSY CTR.

When a bug is detected in the front airbag sensor RH, a diag code B1156/B1157/15 is recorded in the airbag sensor Assicy CTR.
