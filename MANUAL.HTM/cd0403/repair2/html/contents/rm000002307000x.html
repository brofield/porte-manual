        3. Notes on hard disk drive

        1. The navigation receiver ASSY has a built -in hard disk drive (HDD) to store map data, etc. used in the navigation system.For this reason, we handle it with attention to the following items.


          1. No vibration / impact

            ■ Attention ■
            Navigation receiver ASSY does not drop or hit.Do not use falling or hitting.

          2. Do not generate condensation

            ■ Attention ■
            Sudden temperature changes to the navigation receiver ASSY may cause condensation inside and cause HDD failure.For this reason, the navigation receiver ASSY, which is removed from the vehicle, is stored in the same environment in the vehicle or in the same environment.

          3. Do not add static electricity

            ■ Attention ■
            If you touch the connector terminal, static electricity may be applied to the internal circuit, causing HDD failure.

          4. Do not get closer to magnetic things

            ■ Attention ■
            Speech, motor, magnetized tools, etc. do not approach the navigation receiver ASSY.
