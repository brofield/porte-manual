スロットル ボデーASSY  取り付け

準備品一覧



1. スロットル ボデーASSY取り付け

  1. 新品のガスケットをインテークマニホールドに取り付ける。

  2. ボルトおよびナット2個で､スロツトルボデーおよびアクセルレータコントロールケーブルブラケツトを取り付ける。

    基準値
    T = 30 N・m{ 306 kgf・cm }
    (インテークマニホールド新品時)
    基準値
    T = 20 N・m{ 204 kgf・cm }
    (インテークマニホールド再使用時)

  3. スロツトルポジシヨンセンサおよびアイドルスピードコントロールバルブのコネクタを接続する。



2. ウォータバイパス ホース NO.2取り付け


3. ウォータバイパス ホース取り付け


4. アクセルレータ コントロールケーブルASSY取り付け

  1. アクセルレータコントロールケーブルASSYのナットナットでアクセルレータコントロールケーブルブラケツトに接続する。



5. トランスミッションオイルフィラ チューブSUB-ASSY取り付け

  1. 新品のOリングにオートフルードタイプT-IVを塗布し、オイルフイラチユーブに取り付ける。

  2. ボルトで、オイルフイラチユーブをトランスアクスルASSYに取り付ける。

    基準値
    T = 5.5 N・m{ 56 kgf・cm }

  3. ボルトで、オイルクーラチユーブクランプをトランスアクスルASSYに取り付ける。

    基準値
    T = 5.5 N・m{ 56 kgf・cm }

  4. ブリーザホースをオイルフイラチユーブに取り付ける。

  5. オイルレベルゲージをオイルフイラチユーブに取り付ける。



6. エア クリーナASSY取り付け

  1. ボルト4本でエアクリーナケースとエアクリーナインレツトNo.1およびエアクリーナインレツトNo.2を取り付ける。

    基準値
    T = 7.8 N・m{ 80 kgf・cm } (エアクリーナケースASSY　×　エアクリーナブラケツト)
    基準値
    T = 7.5 N・m{ 76 kgf・cm } (エアクリーナインレツトNo.2　×　ボデー)

  2. エアクリーナフイルタエレメントを取り付ける。

  3. エアクリーナキヤツプとエアクリーナホースNo.1を取り付ける。



7. フューエルベーパフィード ホース NO.1接続

  1. クラアンプをかん合させて、フユーエルベーパフイードホースNo.1をエアクリーナキヤツプに接続する。

  2. フユーエルベーパフイードホースNo.1をスロツトルボデーASSYに接続する。



8. 冷却液(トヨタ純正スーパーLLC)補充


9. 冷却液(トヨタ純正スーパーLLC)漏れ点検
