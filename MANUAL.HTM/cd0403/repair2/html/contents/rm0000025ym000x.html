1. エンジンマウンティング インシュレータSUB-ASSY RH取り付け



  1. ジャッキを操作して、ボルト5本およびナットで、エンジンマウンティングインシュレータRHを取り付ける。

    基準値
    T = 45 N・m{ 459 kgf・cm } (ボルトA)
    T = 52 N・m{ 530 kgf・cm } (ボルトB、ナット)



2. エンジンアンダ カバー RH取り付け

  1. スクリュー2個およびボルト2本で、エンジンアンダカバーRHを取り付ける。

    基準値
    T = 5.0 N・m{ 51 kgf・cm }

  2. ナットを、締め付ける。

    □ 参 考 □
    ナットは、LH側と共締めになっている。

    基準値
    T = 5.0 N・m{ 51 kgf・cm }
