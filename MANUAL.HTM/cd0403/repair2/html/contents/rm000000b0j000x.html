1. マグネット　クラッチASSY取り付け
SST
09904-00010   (09904-00050)



  1. マグネツトクラツチステータをクーラコンプレツサASSYの図の位置に合わせて取り付ける。




  2. SSTを使用して、新品のスナツプリングの面取り側を上にして取り付ける。

    ■ 注 意 ■
    スナツプリング取り付け時にベアリングのシールカバーを傷つけない。

  3. コネクタを接続し、スクリユーでクランプを取り付ける。

  4. マグネツトクラツチロータを取り付ける。

  5. SSTを使用して、新品のスナツプリングの面取り側を上にして取り付ける。

    ■ 注 意 ■
    スナツプリング取り付け時にベアリングのシールカバーを傷つけない。

  6. マグネツトクラツチハブを取り付ける。

    ■ 注 意 ■
    マグネツトクラツチワツシヤの組み合せを分解前と変えない。

  7. バイスプライヤーを使用して、マグネツトクラツチハブを固定し、ボルトで締め付ける。

    基準値
    T = 13 N・m{ 135 kgf・cm }



2. マグネットクラッチエアギャップ点検

  1. ダイヤルゲージをマグネツトクラツチハブに垂直にセットする。

  2. マグネツトクラツチコネクタ1端子にバツテリーのプラス、アースワイヤにバツテリーのマイナスを接続し、マグネツトクラツチをON←→OFFさせたときのエアギャップを測定する。

    基準値
    0.4±0.15mm

  3. 基準値外の場合はマグネツトクラツチハブをはずし、マグネツトクラツチワツシヤで調節する。



3. コンプレッサオイル量点検

  1. 新品のクーラコンプレツサASSYに交換する場合は、サービスバルブから徐々に冷媒ガスを抜いた後、新品のクーラコンプレツサASSYから次のオイルを抜き取ってから取り付ける。

    基準値
    (新品のクーラコンプレツサASSY内のオイル容量80±20ｍL)-(取りはずしたクーラコンプレツサASSY内のオイル残量)=(交換時の抜き取り量)

    ■ 注 意 ■
    1. コンプレツサオイル量点検は、クーラ脱着作業時の注意事項を厳守して行う。
    2. コンプレツサオイルが車両の配管に残っているので、新品のクーラコンプレツサASSY内のオイルを抜き取らないで取り付けると、オイル量が多くなり冷媒サイクルの熱交換を妨げ、冷房不良となる。
    3. 取りはずしたクーラコンプレツサASSYのオイル残量の少ない場合は、オイル漏れの点検を行う。
    4. コンプレツサオイルは必ずND-OIL ８を使用する。



4. クーラコンプレッサ W/マグネットクラッチASSY取り付け

  1. ボルト４本で、クーラコンプレツサ W/マグネツトクラツチASSYを取り付ける。

    基準値
    T = 25 N・m{ 255 kgf・cm }

  2. コネクタを接続する。



5. クーラリフリザラントディスチャージ ホース NO.1取り付け

  1. 配管の接合部に巻いたビニールテープを取りはずす。

  2. 新品のOリングおよびクーラコンプレツサASSYの接合部にコンプレツサオイルを十分に塗布する。

  3. ボルトで、クーラリフリザラントデイスチヤージホースNo.1をクーラコンプレツサASSYに取り付ける。

    基準値
    T = 9.8 N・m{ 100 kgf・cm }



6. クーラリフリザラントサクション ホース NO.1取り付け

  1. 配管の接合部に巻いたビニールテープを取りはずす。

  2. 新品のOリングおよびクーラコンプレツサASSYの接合部にコンプレツサオイルを十分に塗布する。

  3. ボルトで、クーラリフリザラントサクシヨンホースNo.1をクーラコンプレツサASSYに取り付ける。

    基準値
    T = 9.8 N・m{ 100 kgf・cm }



7. ファン ＆ オルタネータ Vベルト取り付け

  1. Vベルトを各プーリに仮付けする。

    ■ 注 意 ■
    1. ベルトと各プーリに、オイル等の液体及び異物等の付着無きこと。
    2. ベルトのリブ山ずれの無きこと。



8. ファン ＆ オルタネータ Vベルト調整



  1. ハブナットレンチまたはバーなどを使用して、オルタネータを車両フロント側に引いて、フアン　&　オルタネータVベルトの張力を調整する。

    ■ 注 意 ■
    Vベルトを張る際は、オルタネータとエンジンマウンテイングインシユレータの間にハブナットレンチまたはバーを差し込み、テンションをかける。オルタネータとカムシャフトタイミングオイルコントロールバルブ(インテークマニホルド側)の間にハブナットレンチまたはバーを差し込むとカムシヤフトタイミングオイルコントロールバルブを破損させる恐れがあるため、絶対に行わない。




  2. 調整用ボルトAを締め付けてから、固定用ボルトBを締め付ける。

    基準値
    T = 19 N・m{ 194 kgf・cm } (ボルトA)
    T = 54 N・m{ 551 kgf・cm } (ボルトB)



9. Vベルト張力·たわみ量点検


10. 冷媒 HFC-134a[R134a]充填

  1. 真空ポンプを使用して、真空引きを行う。



  2. 冷媒HFC-134ａ(R134ａ)を充てんする。

    基準値:
    380±30ｇ

    ■ 注 意 ■
    クーラコンプレッサは冷媒なしで運転すると潤滑不良で焼き付くおそれがあるので、冷媒充てん前に絶対運転しない。

    □ 参 考 □
    回収した冷媒ガスを再使用する場合は、フロン回収、再生機での冷媒ガス回収率が約９０％のため、再充てんにはサービス缶による補充が必要になるので、予め準備する。

12. エンジンアンダ カバー RH取り付け

  1. スクリュー2個およびボルト2本で、エンジンアンダーカバーRHを取り付ける。

    基準値
    T = 5.0 N・m{ 50 kgf・cm }

  2. ナットを、締め付ける。

    □ 参 考 □
    ナットは、LH側と共締めになっている。

    基準値
    T = 5.0 N・m{ 50 kgf・cm }
