printing



  1. TASCAN-AT Learning value Initialization (initialization of learning values)
    SST
    09991-70200

    1. Use SST (TASCAN) to operate according to the screen display, and select a transmission for the work support category.

    2. Select [AT learning value initialization] on the screen display and initialize the AT learning value.

      ■ Attention ■
      When you replace either Transxle ASSY, Valve Body ASSY, transmission wire, engine and engine controlled compilation, perform the AT learning value initialization.

      □ Participation □
      The learning value is not initialized just by removing the battery.
