1. スライドドアLH点検


  1. 基準値内であることを点検する。





2. スライドドアLH調整



  1. スライドドア後端上下左右方向およびスライドドアロックストライカプレートASSYを調整する場合は、スライドドアダウンヒメールストップのボルトをゆるめた状態で、トルクスソケットレンチ(T40)を使用して、ストライカ取り付けスクリューをストライカが動く程度までゆるめ、ブラスバーを使用してストライカを軽くたたいて調整する。

    基準値
    T = 23 N・m{ 235 kgf・cm }

  2. スライドドア後端前後上下方向を調整する場合は、ＳＳＴを使用してスライドドアヒンジASSY ＣＴＲ LHのボルトをゆるめてから行う。

    SST
    09812-00010  
    基準値
    T = 27 N・m{ 275 kgf・cm }

    ■ 注 意 ■

    センターリングボルトを使用しているため、調整は補給されているボルトに交換してから行う。




  3. スライドドアの前端上部左右方向の調整をする場合は、スライドドアローラASSY UPR LHのボルトをゆるめてから行う。

    基準値
    T = 7.5 N・m{ 76 kgf・cm }




  4. スライドドア前端下部左右方向(ボデー側)および前後上下方向(ドア側)を調整する場合は、スライドドアを全閉にして確認しながら、スライドドアローラASSY LWR LHのボルトをゆるめてから行う。

    ■ 注 意 ■
    ドア側はセンターリングボルトを使用しているため、調整は補給されているボルトに交換してから行う。

    基準値
    ドア側
    T = 27 N・m{ 275 kgf・cm }
    ボデー側
    T = 19 N・m{ 194 kgf・cm }

    □ 参 考 □
    スライドドアロワーローラベースLH(ボデー側)を調整する場合は、スライドドアフルオープンストップLHを切り離し、ボルトを仮付けする。
