  1. Check the wire harness and connector instant

    □ Participation □
    If a defective system is found by the diagcode output of the diag code (check mode) inspection, the defective part is narrowed down by the following method.

    1. Use SST (Tascan) to work according to the screen display, display the [TCCS] screen, and select Diagcode.

      SST
      09991-70200

    2. Select Mode selection (normal) and switch to check mode.

    3. After reading the diag code in the check mode, erase the memory of the diagcode.

    4. Select the check mode and start the engine.

    5. Shake the defective wire harness and connector of the defective system output in the diag code (check mode) inspection while the idle rotation state remains.

    6. If you shake the wire harness and the connector and the check engine warnin glamp lights up, there is a risk of poor contact with the wire harness or connector at that point.

    7. After the inspection, be sure to select Mode selection (check) and switch to normal mode.
