1. 燃料流出防止作業

■ 注 意 ■
  1. 燃料系部品取りはずし前に必ず燃料流出防止作業を行う。
  2. 燃料流出防止作業後もフューエルラインに若干残圧が残るためフューエルラインを切り離す際はウエス等で覆い、燃料の飛散を防ぐ。




  1. フユーエルポンプASSY用のコネクタを切り離す。

  2. エンジン始動

    1. エンジンを始動する。

    2. エンジンが自然に停止したあと、イグニッションスイッチをOFFにする。

      □ 参 考 □
      ダイアグコード“P0171/25”の燃料異常が検出される場合がある。

    3. 再度エンジンをクランキングし、始動しないことを確認する。

  3. フューエルタンクキャップをはずし、フューエルタンク内の圧力を抜く。

  4. バッテリのマイナスターミナルを取りはずす。

  5. フューエルポンプASSYのコネクタを取り付ける。



2. フロントタイヤ取りはずし


3. エンジンアンダ カバー RH取りはずし

  1. ボルト2本およびスクリュ2本をはずす。

  2. ナットをはずし、エンジンアンダーカバーRHを取りはずす。

    □ 参 考 □
    ナットは、LH側と共締めになっている。



4. エンジンアンダ カバー LH取りはずし

  1. ボルト2本およびスクリュ2本をはずし、エンジンアンダーカバーLHを取りはずす。



5. 冷却液(トヨタ純正スーパーLLC)抜き取り

■ 警 告 ■
やけどの危険を避けるため、エンジンとラジエータが熱い状態のときにはラジエータキヤツプをはずさないこと。液体と蒸気が圧力によって飛び出す。

  1. ラジエータキヤツプを開ける。

  2. ラジエータドレーンコツクプラグをゆるめ冷却液を抜き取る。



6. オートマチックトランスアクスルフルード抜き取り

  1. ドレーンプラグおよびガスケットをはずし、フルードを抜き取る。

  2. 新品のガスケットを介して、ドレーンプラグを締め付ける。

    基準値
    (U340E)
    T = 49 N・m{ 500 kgf・cm }
    (U441E)
    T = 17 N・m{ 175 kgf・cm }

13. ラジエータ グリル W/ラジエータ サポート シール UPR取りはずし

  1. クリップリムーバーを使用して、クリップ6個を取りはずす。




  2. ラジエータグリル、ラジエータサポートシールUPRの周囲および上面に保護テープを貼りつける。



  3. 保護テープを貼ったマイナスドライバーを使用して、図の3箇所のツメのかん合をサポートシールを車両後方にずらしてはずす。

    □ 参 考 □
    マイナスドライバーをサービスホールより斜めに入れ、図の矢印のようにドライバーを回し、ツメのかん合をはずす。






  4. ラジエータサポートシールUPRを持ち上げ、すき間からラジエータグリルのツメのかん合をはずす。

    □ 参 考 □
    ラジエータサポートシールUPRは片側ずつ持ち上げて、ツメのかん合がはずれたら元に戻す。




  5. ツメのかん合がはずれているのを確認しラジエータグリルを取りはずす。

    ■ 注 意 ■
    ラジエータグリル下部のツメは破損しやすいので注意する。

  6. ラジエータ サポート シール UPRを取りはずす。

25. フューエル チューブSUB-ASSY切り離し



  1. SSTを使用して、フユーエルチユーブを切り離す。

    SST
    09268-21010  

    ■ 注 意 ■
    1. フユーエルチユーブ切り離し作業前にパイプおよびコネクタの接続部に異物の付着がないことを確認する。
    2. ナイロンチューブを無理に曲げたり、折ったり、回転させない。
    3. フユーエルチユーブコネクタとパイプが固着している場合は、押したり、引いたりして固着を解除して切り離す。
    4. 取りはずしたパイプとコネクタが損傷したり、異物が混ざったりしないようにビニール袋でカバーする。

  3. エンジンワイヤをエンジンルーム内に引き抜く。

  4. エンジンワイヤをエンジンルームJ/Bより切り離す。

  5. アース線を切り離す。

  3. スライディングヨークのギヤ側(B)のボルトを取りはずす。

  4. スライディングヨークとインタミディエイトシャフトに合わせマークを付け、スライディングヨークを切り離す。




  5. クリップAをはずし、ホールカバーをボデーから切り離す。

    ■ 注 意 ■
