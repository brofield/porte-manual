Circuit description


  1. If there is an abnormality in the power circuit of the brake actaba ASSY (Skitsudo Control Companator), the skitt control compiler outputs a diagnostic code and prohibits operation by failed.
  2. This code is memorized when the voltage to the Ig1 terminal is out of the DTC detection condition range due to a power supply such as a cross, an alternator circuit, or an abnormality of the charging circuit.
  3. Diags are lifted due to normal return of voltage to Ig1 terminals.
