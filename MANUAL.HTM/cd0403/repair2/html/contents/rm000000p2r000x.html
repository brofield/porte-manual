1. ブレーキ アクチュエータASSY取り付け

  1. ブレーキアクチュエータASSYをナット2個でブレーキアクチュエータブラケットASSYに取り付ける。

    基準値
    T = 5.4 N・m{ 55 kgf・cm }

    ■ 注 意 ■
    新品のブレーキアクチュエータASSYにはブレーキフルードが充填してあるので、ブレーキチューブを接続するまでホールプラグを外さない。



2. ブレーキアクチュエータASSY　W/ブラケット取り付け

  1. 図の順にボルト3本で、ブレーキアクチュエータASSYを仮付けする。




  2. 図の順にボルト3本を本締めする。

    基準値
    T = 19 N・m{ 195 kgf・cm }

  3. コネクタをブレーキアクチュエータASSYに接続する。




  4. ユニオンナットレンチ10を使用して、ブレーキチューブ6本をブレーキアクチュエータASSYに接続する。

    基準値
    T=15N·m {155kgf·cm} (規定締付けトルク)
    T’=14N·m{143kgf·cm}(ユニオンナットレンチおよびトルクレンチ230F使用時の読み)

  5. ボルト1本を締め、ワイヤハーネスクランプブラケットを接続する。



3. エア クリーナASSY取り付け

  1. ボルト4本で、エアクリーナケース　W/エアクリーナインレットNo.1およびNo.2を取り付ける。

    基準値
    エアクリーナケースASSY × エアクリーナブラケット
    T = 7.8 N・m{ 80 kgf・cm }
    エアクリーナインレットNo.2 × ボデー
    T = 7.5 N・m{ 76 kgf・cm }

  2. エアクリーナフィルタエレメントを取り付ける。

  3. エアクリーナキャップ　W/エアクリーナホースNo.1を取り付ける。



4. バッテリ取り付け

  1. バッテリトレイを取り付ける。

  2. バッテリを取り付ける。

  3. ボルトおよびナットでバッテリクランプを取り付ける。
