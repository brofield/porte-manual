Circuit description


DTC is output when the engine speed is reduced from the set speed.

The situation when DTC is output can be used to check freeze -frame data using SST (Tascan).The freeze -frame data is a part of the engine operation state in the event of a defect, which is useful for troubleshooting.

In addition, since the idol instability due to gas missing is also detected, it is necessary to check whether there is a missing gas before work.

Inspection procedure


□ Participation □
  1. The DTC P1605 is different from the conventional parts, circuit, and system failure diags, and estimates a failure part from a defective phenomenon and freeze frame data in response to the user's indications such as "idling unstable".Since the diag code may be detected by the user operation, even if the diag code is output, the DTC will be erased and the vehicle will return the vehicle without consulting on the problem from the user.
  2. If another diag code is displayed, perform troubleshooting the corresponding diagcode.
  3. If there is a situation (such as the situation at the time of restarting) in the case of a questionnaire result, refer to it.

    Symptoms
    		Predicted defect
    Can be started with the accelerator pedal stepped on
    		Lack of air
    After starting, rough idol
    		Anomalous air -fuel ratio
  4. Read freeze frame data using SST (Tascan).The freeze -frame data is a part of the engine operation state in the event of a defect, which is useful for troubleshooting.
  5. When confirming the time series freeze frame data, check the 5 data in the event of a defectation recorded in the freeze frame data.(The point isreference)
  6. If there are multiple items corresponding to the problem when confirming the time series freeze frame data, proceed with all the corresponding diagnostic procedures.
  7. Drive and stop the situation in the event of a defectation recorded in the freeze -frame data under the same conditions as possible.Check the data at that time and the data at the time of idling (after warming, no load, D and N position) and compare it with freeze -frame data.
  8. Considering that the inspection is not reproduced, the emphasis is on checking signs and situations in the event of a problem.
  9. When performing inspection, shake the applicable wire harness and connector to make the defective detection that does not occur at all times.
  10. The following procedure shows the No. 1 cylinder inspection method.No. 1 When checking a non -cylinder, check the corresponding cylinder circuit with reference to the following procedure.
