1. サーモスタット取り付け

  1. 新品のガスケットをサーモスタットに取り付ける。




  2. 図のようにジグル弁を上側にしてサーモスタットを取り付ける。

    □ 参 考 □
    ジグル弁の向きは真上から左右１０°以内の範囲にする。



2. ウォータ インレット取り付け

  1. ナット２個で、ウォータインレット　Ｗ／ラジエータアウトレットホースを取り付ける。

    基準値:
    T＝9.0N·m｛92kgf·cm｝



3. 冷却液(トヨタ純正スーパーLLC)補充

  1. ラジエータドレーンコツクプラグを閉じて、冷却水をラジエータ注入口よりあふれるまで注入する。［＊１］

    □ 参 考 □
    ラジエータインレツトホースおよびラジエータアウトレツトホースを手で数回圧迫し、ラジエータ注入口の水位が下がる様であれば追加注入する。

  2. ラジエータキヤツプを締める。

  3. ラジエータリザーブタンクに冷却液を上限まで注入する。

  4. エンジンをサーモスタツトが開弁するまで暖機する。

    □ 参 考 □
    暖機中にラジエータインレツトホースおよびアウトレツトホースを数回圧迫すること。

  5. エンジンを止め、冷却液が冷えるまで待ち、ラジエータキヤツプをはずして水位を確認する。

  6. 水位が下がっている場合は、［＊１］より繰り返す。

  7. 水位が下がらなくなったら、ラジエータリザーバタンクの冷却液を調整する。



4. 冷却液(トヨタ純正スーパーLLC)漏れ点検

  1. 冷却液を満水にしてテスターを取り付ける。

  2. 137ｋＰａ｛1.4ｋｇｆ/ｃｍ２｝の圧力をかけ、各部に水漏れがないことを確認する。



5. エンジンアンダ カバー RH取り付け

  1. スクリュー2個およびボルト2本で、エンジンアンダーカバーRHを取り付ける。

    基準値
    T = 5.0 N・m{ 50 kgf・cm }

  2. ナットを、締め付ける。

    □ 参 考 □
    ナットは、LH側と共締めになっている。

    基準値
    T = 5.0 N・m{ 50 kgf・cm }
