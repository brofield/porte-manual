Back guide monitor system function description

printing



  1. Can communication system


    1. The back guide monitor system uses CAN communication for signal transmission and receiving between the steering sensor and the navigation receiver ASSY.



      □ Participation □
      Navigation receiver ASSY is connected to CAN communication, but cannot be read with SST (Tascan).



    1. Diagnostic function

      1. Adjustment of the back guide monitor system is made on the diagnostic screen of the navigation receiver ASSY.

        □ Participation □
        When the work shown in the table below occurs, adjustment and confirmation on the diagnostic screen are required.

        Work implementation site
        		Adjustment item
        		Reference destination
        1. When the body mounting angle of the television camera ASSY shifts due to some reason (such as removing the television camera ASSY)
        2. Replacement of television camera ASSY
        		Camera optical axis adjustment
        		(The point isreference)
        Replacement of navigation receiver ASSY
        1. Memories of the neutral point of the rudder angle
        2. Memories of left and right rudder angle
        3. Camera optical axis adjustment
        		(The point isreference)
        1. When the battery terminal is removed
        2. When replacing or removing the steering sensor
        3. When the expected course in the course display mode is not displayed and there is no abnormality in the steering sensor
        4. When the expected course shifts for some reason
        		Memories of the neutral point of the rudder angle
        		(The point isreference)

      2. The back guide monitor system has a diagnostic function.(Displayed in the navigation receiver ASSY)

      3. The “HDD navigation” system will communicate between devices by AVC-LAN when installing options with a dealer.

      4. Each device that makes AVC-LAN has a three-digit number (notation in hexadecimal numbers) called a device code (physical address).

      5. Each function (unit) that makes up the interior of the AVC-LAN device is set with a two-digit number (notation in the hexadecimal number) called a logical address.
