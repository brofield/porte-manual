リヤシートASSY(4WD)  取り付け

印刷

準備品一覧



1. チャイルドレストレイント シート アンカ ブラケット RH取り付け



  1. ボルト4本を仮締めする。

  2. ボルト4本を１→2→3→4の順に締め付け、チャイルドレストレイントシートアンカブラケットRHを取り付ける。

    基準値
    T = 42 N・m{ 420 kgf・cm }

  3. フロアカーペットを取り付ける。



2. リヤシートバック ヒンジSUB-ASSY RH取り付け



  1. ボルトで、リヤシートバックヒンジRHを取り付ける。

    基準値
    T = 42 N・m{ 420 kgf・cm }



3. リヤシートバック ヒンジSUB-ASSY LH取り付け

  1. ボルトで、リヤシートバックヒンジLHを取り付ける。

    基準値
    T = 42 N・m{ 420 kgf・cm }



4. リヤシートクッションASSY取り付け



  1. シートベルトを通し、フックをかん合させる。

  2. リヤシートクッションASSY前部のクリップをかん合させる。



5. リヤシートバツクASSY取り付け



  1. リヤシートバックASSYを車両に乗せ、リヤシートバックフレームの穴とヒンジ側のロケーションを合わせボルト2本を仮付けする。

  2. ボルト2本を本締めし、リヤシートバックASSYを取り付ける。

    基準値
    T = 36.8 N・m{ 375 kgf・cm }

  3. 新品のクリップ2個を取り付ける。
