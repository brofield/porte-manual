準備品一覧



1. コンデンサASSY (ウィズ レシーバ)取り付け



  1. ボルト2本で、コンデンサASSY(ウィズレシーバ)を取り付ける。

    基準値
    T = 9.8 N・m{ 100 kgf・cm }



2. リキッド チューブSUB-ASSY A取り付け

  1. リキツドチユーブＡおよびコンデンサASSY(ウィズレシーバ)の接合部に巻いたビニールテープを取りはずす。

  2. 新品のOリングおよびリキツドチユーブＡの接合部にコンプレツサオイル(ND-OIL8)を充分に塗布する。

  3. OリングをリキツドチユーブAに取り付ける。




  4. ボルトで、リキツドチユーブAを取り付ける。

    基準値
    T = 5.4 N・m{ 55 kgf・cm }



3. クーラリフリザラントディスチャージ ホース NO.1取り付け

  1. クーラリフリザラントディスチヤージホースNo.１およびコンデンサASSY(ウイズレシーバ)の接合部に巻いたビニールテープを取りはずす。

  2. 新品のOリングおよびクーラリフリザラントディスチヤージホースNo.１の接合部にコンプレツサオイル(ND-OIL８)を充分に塗布する。

  3. OリングをクーラリフリザラントデイスチヤージホースNo.１に取り付ける。




  4. ボルトで、クーラリフリザラントデイスチヤージホースNo.１を取り付ける。

    基準値
    T = 5.4 N・m{ 55 kgf・cm }



4. コネクタおよびクランプ取り付け（ホーン　）


5. 冷媒HFC-134a(R134a)充てん

  1. 真空ポンプを使用して、真空引きを行う。



  2. 冷媒HFC-134ａ(R134ａ)を充てんする。

    基準値:
    380±30ｇ

    ■ 注 意 ■
    クーラコンプレッサは冷媒なしで運転すると潤滑不良で焼き付くおそれがあるので、冷媒充てん前に絶対運転しない。

    □ 参 考 □
    回収した冷媒ガスを再使用する場合は、フロン回収、再生機での冷媒ガス回収率が約９０％のため、再充てんにはサービス缶による補充が必要になるので、予め準備する。

7. 冷媒ガスもれ点検


8. フロントバンパ カバー取り付け

  1. フォグランプのコネクタを接続する。




  2. フロントバンパサイドのツメのかん合を合わせ、クリップ2個およびスクリュ6本ボルト2本でフロントバンパカバーを取り付ける。

  3. スクリュ2本でフロントバンパサイドブラケットRHおよびLHを取り付ける。



9. フロントフェンダ ライナ RH取り付け

  1. スクリュ3本でフロントフェンダライナ RH(前部)を取り付ける。



10. フロントフェンダ ライナ LH取り付け


11. ラジエータ グリル W/ラジエータ サポート シール UPR取り付け

  1. ラジエータ グリルとラジエータ サポート シール UPRを組み付ける。




  2. ラジエータグリル下部のツメを合わせ、クリップ6個でラジエータ グリル W/ラジエータ サポート シール UPRを取り付ける。
