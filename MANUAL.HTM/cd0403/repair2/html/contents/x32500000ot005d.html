  1. overview

    1. CAN (Controller Area Network) is a multiple communication for inexcities that have excellent data communication speed (500 kbps) and error detection capacity for serial communication for real -time applications.

    2. CAN has one pair of two lines, Can-H and Can-L, and communicates at differential voltage.

    3. The car is equipped with many ECUs (sensors), but they are not operating alone, and they are working while shared information with other ECUs (sensors).

    4. In CAN, the main line contains two resistance (120Ω) necessary for communication transmission.



    1. Definition of terms

      1. Main line


        1. The harness between the terminal circuit located on the bus (communication line) is defined as the main line.It becomes a core bus of the control system CAN.

      2. Branch line


        1. A line that branches into the ECU or sensor is defined as a branch line as a branch line.

      3. Terminal circuit


        1. A consisting of a resistor and a capacitor placed to convert the communication current of the control system CAN to a bus voltage is called an terminal circuit.Two terminal circuits need to be prepared on the bus.



      1. CAN Communications ECU, sensor

        1. Brake actuator ASSY (VSC ECU)

        2. Steering sensor (steering angle sensor)

        3. York -rate sensor (yaw / G sensor)



        1. CAN communication diagram code

          1. U0121/94 (ABS), U0123/62 (VSC), U0124/95 (ABS), and U0126/63 (VSC) are CAN communication diagram code.



          1. Points to keep in mind about troubles

            1. CAN Communications diagram codes can be checked on Tascan.

            2. DLC3 is also connected to CAN communication, but the abnormalities of the DLC3 branch wire cannot be read.



            1. How to read CAN Communication Diag Code

              1. CAN communication troubles identify the problem by combining the output CAN communication diag code (the diagram code combination table isreference)

              2. Diagcodes other than U0121/94 (ABS), U0123/62 (VSC), U0124/95 (ABS), and U0126/63 (VSC) may output simultaneously.

              3. If U0121/94 (ABS), U0123/62 (VSC), U0124/95 (ABS), U0126/63 (VSC) are not output, it is not a problem with CAN communication (to the troubles of each system).

              4. If U0121/94 (ABS), U0123/62 (VSC), U0124/95 (ABS), and U0126/63 (VSC) are output, there is a possibility of a CAN communication system.



              1. Can Communication Diag Cord Output ECU

                1. Brake actuator Ass (VSC ECU) output U0121/94 (ABS), U0123/62 (VSC), U0124/95 (ABS), U0126/63 (VSC).

                  1. Fail safe function

                    1. VSC controlled fail -safe


                      1. If any of any of the buses (communication lines) is ferted in a short or the like, it is designed to shift to a fail -safe process and prevent the system malfunction.

                      2. Brake actuator ASSY (VSC ECU), steering sensor (steering angle sensor) and yorrate sensor (yaw / G sensor), brake actuator ASSY (VSC ECU) enters the Fail Safe and stops the VSC function.
