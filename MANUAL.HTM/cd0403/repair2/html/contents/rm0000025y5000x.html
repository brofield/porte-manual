1. フロントトランスアクスルケース オイル シール NO.1取り付け



  1. SSTおよびハンマーを使用して、新品のオイルシールを取り付ける。

    SST
    09726-27012   (09726-02041)
    09950-70010   (09951-07200)

    基準値
    オイルシ－ル打ち込み量(A寸法) 5.9±0.5mm

  2. オイルシールのリップ部にMPグリースNo.2を塗布する。



2. フロントドライブ シャフトASSY LH取り付け

  1. インボードジョイントASSY LHのスプライン部にオートフルードタイプT-IVを塗布する。




  2. インボードジョイントASSY LHのスプラインをかん合させ、ブラスバーおよびハンマーを使用して、フロントドライブシャフトASSY LHを挿入する。

    ■ 注 意 ■
    1. スナップリングLHの切り口を下側へ向ける。
    2. オイルシールを傷つけない。
    3. ドライブシャフトブーツを傷つけない。

    □ 参 考 □
    フロントドライブシャフトASSY LHが完全に打ち込まれていることを、ブラスバーの反力の変化あるいは音質の変化で判断する。



3. フロントアクスルASSY　LH取り付け



  1. フロントアクスルASSY LHを車両外側に押して、フロントアクスルASSY LHにフロントドライブシャフトASSY LHのスプラインをかん合させ、挿入する。

    ■ 注 意 ■
    1. 必要以上にフロントアクスルASSY LHを車両外側へ押し出さない。
    2. ドライブシャフトアウトボードジョイントブーツを傷つけない。
    3. スピードセンサロータおよび挿入部に異物がないことを確認する。
    4. スピードセンサロータを傷つけない。
    5. オイルシールを傷つけない。




  2. フロントアクスルASSYをショックアブソーバに取り付け、車両前方からボルト２本を挿入し、ナット２個を締め付ける。

    基準値
    T = 132 N・m{ 1350 kgf・cm }

    ■ 注 意 ■
    ボルトおよびナットを再使用する場合のみ、ナットのねじ部および座面にエンジンオイルを塗布する。



4. タイロッドエンドLH取り付け

  1. タイロッドエンドLHをステアリングナックルに取り付け、キャッスルナットで締め付ける。

    基準値
    T = 49 N・m{ 500 kgf・cm }

    ■ 注 意 ■
    1. キャッスルナット締め付け後のコッターピン穴合わせは60°以内の増し締め方向で行う。
    2. ネジ部およびテーパー部に油脂類を付着させない。

  2. 新品のコッターピンを取り付ける。



5. スピード センサ FR LH取り付け



  1. ボルトで、スピードセンサFR LHをステアリングナックルに取り付ける。

    基準値
    T = 8.0 N・m{ 82 kgf・cm }

    ■ 注 意 ■
    1. スピードセンサFR LHの先端および取り付け部に異物が付着していないことを確認する。
    2. スピードセンサFR LHをねじって取り付けない。




  2. スピードセンサFR LHのクランプおよびフレキシブルホースをフロントショックアブソーバASSY LHに取り付ける。

    基準値
    T = 29 N・m{ 300 kgf・cm }

    ■ 注 意 ■
    スピードセンサワイヤに負担がかかる状態で取り付けない。



6. フロントディスクブレーキキャリパASSY LH取り付け

  1. 合わせマークをあわせてフロントディスクを取り付ける。

    ■ 注 意 ■
    ディスク面に油脂を付着させない。




  2. ボルト2本でディスクブレーキキャリパASSYをステアリングナックルに取り付ける。

    基準値
    T = 88 N・m{ 900 kgf・cm }



7. フロントアクスル シャフト ナット LH取り付け

  1. ソケットレンチ(30mm)を使用して、新品のフロントアクスルハブナットLHを取り付ける。

    基準値
    T = 216 N・m{ 2200 kgf・cm }




  2. タガネおよびハンマーを使用して、フロントアクスルハブナットLHをかしめる。



8. エンジンアンダ カバー LH取り付け

  1. スクリュー2個およびボルト2本で、エンジンアンダカバーLHを取り付ける。

    基準値
    T = 5.0 N・m{ 50 kgf・cm }

11. オートマチックトランスアクスルフルード量点検

■ 注 意 ■
  1. オイル過多、過少はトラブルの原因となる。
  2. アイドル回転数を確認後、点検する。
  3. 車両を平坦路に停止させる。
  4. トランスアクスル完全暖機状態(AT油温70-80°C)で点検する。
  5. 注入するフルードはトヨタ純正オートフルードタイプT-IVを使用する。

  1. パーキングブレーキを作用させる。

  2. ブレーキペダルを踏み、エンジンを始動する。

  3. アイドル回転状態でシフトレバーをPからLまでの各レンジにゆっくりシフトした後、Pレンジに戻す。




  4. アイドル回転状態でレベルゲージを抜き、ウエスなどでオイルを拭き取り、再度挿入してオイルがレベルゲージの[HOT]の範囲にあることを確認する。

    ■ 注 意 ■
    1. オイル交換時など低い油温(AT油温20-30°C)の場合は、レベルゲージの[COOL]の範囲内に調整した後、完全暖機状態で再確認する。
    2. レベルゲージの裏表でレベルが異なるときは、低い方で点検する。

  5. オイル量の少ないときは、オイル漏れを点検する。

17. サイドスリップ点検

  1. サイドスリップテスターでサイドスリップを点検する。

    基準値:
    0±5mm(1ｍにつき)

    ■ 注 意 ■
    1. サイドスリップテスターへの進入は直進方向に歩く速度(4km/h程度)以下とし、ニュートラル状態で行う。
    2. サイドスリップテスター上でブレーキを踏まない。
    3. サイドスリップテスター上でステアリング操作をしない。

  2. 基準値外の場合はホイールアライメントを点検する。



18. トーイン点検

  1. 車両をゆすり、車両を安定させる。

  2. 車両を直進で5ｍ手押しして前進させる。

    ■ 注 意 ■
    車両を後退させた時は、必ず後退させた距離分前進させる。




  3. トーインゲージの指針高さをフロントホイール軸中心高さに合わせ、タイヤの後ろ側に入れる。

  4. 前輪タイヤ各々の後部にトレッドセンターをしるし、マーク間の距離(B寸法)を測定する。

  5. 車両をゆっくり押して前進させ、前輪を180°回転させる。

    ■ 注 意 ■
    180°以上回転させないように行い、オーバー回転した場合は、トーインゲージをセットするところからやり直す。




  6. 車両前部でマーク間の距離(A寸法)を測定する。

  7. トーインを求める。

    基準値
    トーイン 1±2mm

    □ 参 考 □
    トーイン＝B寸法-A寸法



19. トーイン調整



  1. 左右のラックエンドのねじ部の長さを測定する。

    基準
    左右差　1.5mm以下

  2. ラックブーツのクリップをはずす。

  3. タイロッドエンドのロックナットをゆるめる。

  4. ラックエンドねじ部の左右差が基準外の場合は調整を行う。

    1. トーインの測定値がアウト側にはずれる場合は、寸法の短い方のラックエンドを伸ばす方向に調整する。

    2. トーインの測定値がイン側にはずれる場合は、寸法の長い方のラックエンドを縮める方向に調整する。

    3. 再度トーインを点検する。

  5. 左右のラックエンドを各々の反対方向に同量ずつ回し、トーインが基準値内に入るように調整する。

    基準値
    トーイン 1±1mm

  6. タイロッドエンドのロックナットを締め付ける。

    基準値
    T = 47 N・m{ 480 kgf・cm }

    ■ 注 意 ■
    ロックナットおよびステアリングラックエンドが共回りしないようにステアリングラックエンドの六角部を固定し、ロックナットを仮締めしてから、タイロッドエンドの二面幅を固定させて締め付ける。




  7. プライヤーを使用して、クリップを図の範囲に取り付ける。

    ■ 注 意 ■
    ブーツをねじって取り付けない。



20. ホイール切れ角点検



  1. ターニングラジアスゲージをセットする。

  2. ホイールの切れ角を点検する。

    基準値
    内側(点検範囲)
    		外側(参考)
    39°±2°
    		34°

    ■ 注 意 ■
    1. 車両を空車状態(スペアタイヤおよび工具なし)にして行う。
    2. 空車状態で、前輪が回転しないように、ブレーキペダルプレッシャーなどを使用して、フットブレーキを効かせた状態で点検する。
    3. ストップランプが点灯しないように、ストップランプヒューズをはずして点検する。



21. キャンバー、キャスター、キングピンアングル点検
SST
09772-30010  

  1. ホイールキャップまたはセンターオーナメントを取りはずす。

  2. 車両を平坦な場所でリフトアップする。

  3. タイヤクランプコンペンセータ(TCK-01)を取り付ける。

    1. タイヤクランプコンペンセータのクランプダイヤルを回し、タイヤクランプアームを広げる。

      ■ 注 意 ■
      タイヤクランプアームを広げすぎるとボデー等に傷を付けるため注意する。




    2. ダイヤルを回し、下側ツメ2箇所をホイールサイズに合わせホイールリム下部に当て上側ツメをホイールリムに合わせて縮める。

      ■ 注 意 ■
      ツメがバランスウェイトに掛からず、ホイールのエアーバルブがなるべく車両上側になるように取り付ける。




    3. クランプダイヤルを回し、タイヤクランプアームを縮めタイヤに取り付ける。

      ■ 注 意 ■
      タイヤクランプアーム先端がタイヤ溝に掛かり、タイヤクランプコンペンセータがしっかりとホイールに圧着していること。

    4. 脱落防止ワイヤーをホイールのエアーバルブに取り付ける。

  4. タイヤクランプコンペンセータを補正する。




    1. タイヤクランプコンペンセータの補正ダイヤルを回してメカゼロライン(赤ライン)の範囲内に合わせる。




    2. キャンバーキャスターキングピンゲージをタイヤクランプコンペンセータの取り付けプレートに取り付け、合わせマークを合わせる。




    3. ホイールを少し回転させ、キャンバーキャスターキングピンゲージの水平気泡が中心になるように合わせ、キャンバーキャスターキングピンゲージの調整ネジでキャスター気泡を0°に合わせる。




    4. ホイールを180°回転させ、キャンバーキャスターキングピンゲージとタイヤクランプコンペンセータの合わせマークを合わせ、水平気泡を合わせる。(*1)




    5. 上記状態のキャスター気泡を読み取り、その値の1/2の目盛りになるようにタイヤクランプコンペンセータの補正ダイヤルを回し合わせる。(*2)

      ■ 注 意 ■
      キャンバーキャスターキングピンゲージの調整ネジは動かさないこと。

    6. 前記(*1)、(*2)の手順を繰り返し、キャスター気泡の目盛り位置がホイールを180°正、反転させても、一定の数値目盛りを指示すればキャンバーキャスターキングピンゲージのセットは完了。

  5. ターニングラジアスゲージをセットし、車両を降ろす。

  6. キャンバー、キャスターおよびキングピンアングルを点検する。

    基準値

    キャンバー
    		キャスター
    		キングピンアングル
    -0°40'±45'
    		2°00'±45'
    		10°15'±45'

    ■ 注 意 ■
    1. 車両を空車状態(スペアタイヤおよび工具なし)にして行う。
    2. キャンバーおよびキャスターの左右差限度は30'以内。

  7. キャンバーキャスターキングピンゲージおよびタイヤクランプコンペンセータを取りはずす。

  8. ホイールキャップまたはセンターオーナメントを取り付ける。



22. キャンバー調整

  1. 車両をジャッキアップして、フロントタイヤを取りはずす。




  2. ボルトをはずし、スピードセンサおよびフレキシブルホースを切り離す。




  3. ショックアブソーバロワー側の取り付けナット2個を緩める。




  4. ショックアブソーバのロワー側を動かして、ボルトとステアリングナックルのボルト穴のガタでキャンバーを調整する。




  5. ナット2個を締め付ける。

    基準値
    T = 132 N・m{ 1350 kgf・cm }




  6. ボルトで、フレキシブルホースおよびスピードセンサを取り付ける。

    基準値
    T = 29 N・m{ 300 kgf・cm }

    ■ 注 意 ■
    フレキシブルホースおよびスピードセンサをねじって取り付けない。

  7. フロントタイヤを取り付ける。

    基準値
    T = 103 N・m{ 1050 kgf・cm }

  8. 車両をジャッキダウンして数回上下にゆすり、サスペンションを落ち着かせる。

  9. キャンバーを点検する。

    ■ 注 意 ■
    1. 車両を空車状態(スペアタイヤおよび工具なし)にして行う。
    2. キャンバーおよびキャスターの左右差限度は30'以内。

    □ 参 考 □
    セットボルトで調整できる範囲は、0.1°から0.5°である。

  10. 基準値内の場合は、トーイン点検を行う。

  11. 基準値外の場合は、ジャッキアップしてフロントタイヤをはずし、スピードセンサおよびフレキシブルホースを切り離し以下の作業を行う。




    1. ショックアブソーバロワー側の取り付けナット2個およびボルト2本を取りはずす。





    2. アジャスティングボルト2本を選択して、車両前方から挿入する。

    3. ナット2個を取り付ける。

      基準値

      ■ 注 意 ■
      ショックアブソーバロワーの締め付け面のよごれを拭き取る。

      基準値
      T = 29 N・m{ 300 kgf・cm }

      ■ 注 意 ■
      フレキシブルホースおよびスピードセンサをねじって取り付けない。
