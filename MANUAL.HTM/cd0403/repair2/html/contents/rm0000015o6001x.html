ABS & BA system with EBD (4WD) Freeze frame data

printing



  1. Freeze frame data output

    1. You can use SST (Tascan) to memorize and display the vehicle (sensor) status when the system is operated or when an abnormal diagnostic code occurs.

    2. Freeze frame data when system operation


      1. The data is memorized when the system is operated, and new data is updated for each system operation.

      2. Finally, IG OFF to 31 times from when the system is activated is memorized and displayed.

        □ Participation □
        1. When the vehicle is stopped or the vehicle speed is less than 7 km/h, the number of Ig off → ON is not counted up.
        2. If the number of IG switch ON exceeds 31 times, display '31'.

      3. When the freeze -frame data when the abnormal diagnostic code occurs, the freeze -frame data at the time of system operation is deleted.

    3. Freeze frame data when abnormal diagnostic cord occurs


      1. When an abnormal diagnostic cord is output, the freeze -frame data is memorized when an abnormal code occurs.

      2. If you remember the freeze frame data in the event of an abnormal diagnostic cord, the data will not be updated until the memory data is erased (deleted of the diagnostic code).

      3. In the freeze -frame data at the time of abnormal diagnostic cord, the number of IG switch ON after the freeze frame data memory is not remembered.
