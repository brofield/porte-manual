1. ニュートラルスタート スイッチASSY取り付け



  1. ニユートラルスタートスイツチを取り付け、ボルト2本を仮付けする。

  2. ナットおよびナットストッパを取り付ける。

    基準値
    T = 6.9 N・m{ 70 kgf・cm }

  3. コントロールシヤフトレバーを仮付けする。




  4. コントロールシヤフトレバーを反時計方向に止まるまで回転させ、その位置から時計方向に２ノッチ回転させＮレンジにし、コントロールシヤフトレバーを取りはずす。




  5. ニュートラル基準線とナットストッパの突部を合わせて、ボルト2本を本締めする。

    基準値
    T = 5.4 N・m{ 55 kgf・cm }




  6. マイナスドライバーを使用して、ナットストッパのツメをかしめる。




  7. コントロールシャフトレバーおよびスプリングワッシャを取り付け、ナットを締め付ける。

    基準値
    T = 12.7 N・m{ 130 kgf・cm }



2. トランスミッション コントロールケーブルASSY取り付け



  1. ナットで、コントロールケーブルをコントロールシヤフトレバーに取り付ける。

    基準値
    T = 12 N・m{ 122 kgf・cm }

  2. クリップで、コントロールケーブルをブラケットに固定する。



3. シフトレバー位置点検

■ 注 意 ■
PからRレンジにシフトするときは、イグニツシヨンスイツチをACCまたはONにし、ブレーキペダルを踏んでシフトする。

  1. シフトレバーをNレンジから各レンジにシフトする。このときシフトレバーが円滑に操作でき、各レンジに節度よく動き、ポジションインジケータが表示しシフトレバー位置が一致していることを確認する。

  2. シフトレバーを手前に引いたときのみP、R、Lの各レンジにシフトできることを確認する。

  3. エンジンを始動し、Dレンジにシフトしたとき車両が前進し、Rレンジにシフトしたときブザー音がして車両が後退することを確認する。



4. シフトレバー位置調整

  1. コントロールシャフトレバーのナットをはずし、コントロールケーブルを切り離す。

  2. コントロールシャフトレバーを反時計方向へ止まるまで回し、そこから2段階戻した位置(Nレンジ)にする。




  3. 図のように、シフトレバーをNレンジにシフトし、Rレンジ側に押し付けた状態で、コントロールケーブルを取り付け、ナットで締付ける。

    基準値
    T = 12 N・m{ 120 kgf・cm }

  4. 調整後、操作具合および作動を点検する。
