ABS & BA system (4WD) test mode with EBD

printing



  1. Test mode inspection

    □ Participation □
    1. When entering the test mode, the all -test mode code is memorized once, and the memory of the code determined that the computer is normal at the time of inspection is deleted.
    2. When you return to the normal mode, all test mode code will be deleted.
    3. If there is an abnormality in the sensor, the ABS Warnin Gramp will light up and the ABS will not work.

    1. Test mode startup method (TASCAN)


      1. IG Switch off.

      2. Connect SST (Tascan) to DLC3.

        SST
        09991-70200

      3. Turn on IG switch.

      4. Follow the SST (Tascan) display screen to switch to test mode.




    2. Test mode startup method (DLC3 short circuit)


      1. IG Switch off.

      2. Use SST to short circuit between 12 (TS) ← → 4 (CG) of DLC3.

        SST
        09843-18040

        ■ Attention ■
        There is no mistake in the connector connection position because it causes a failure.

      3. Turn on IG switch.




    3. Test mode display confirmation


      1. After the ABS Warnin Grand Lamp is turned on for a few seconds, the transition to the test mode by switching to pulse blinking.

        Test mode code continuation conditions
        DLC3
        		IG switch ON
        		IG switch off
        		IG switch OFF → ON
        TS-CG terminal short circuit
        		Maintaining memory
        		Maintaining memory
        		Maintaining memory
        Opening between TS-CG terminals
        		Maintaining memory
        		Maintaining memory
        		Memory erasure

    4. Deceleration sensor chaeck


      1. Turn on IG switch.

      2. Make the shift lever in a P range and stop at a flat place for 1 second or more.

        ■ Attention ■
        Be sure to end the deceleration sensor chaeck before performing the speed sensor chaeck.

        □ Participation □
        Warnin Grand Pamps do not change.If the sensor is normal and the conditions are established when the test mode is started, it is closed.



    5. Speed ​​sensor chak




      1. Start without wheel spinning on the handle straight, speed up to 45 km/h or higher, and confirm that the ABS Warning Lamp will turn off.

        ■ Attention ■
        1. Before performing the speed sensor chaeck, discontinue the deceleration sensor chaeck.
        2. If you start a sensor chak while steering, the speed sensor chaeck may not end if the wheel spin is made.
        3. After turning off the ABS Warnin Gramp, if you drive at 80 km/h or more per hour, the test mode code will be remembered again, so the deceleration and stop will be stopped before reaching 80 km/h.
        4. In the case of an abnormal sensor, the ABS Warning Lamp lights up, and the ABS does not work.

        □ Participation □
        When the speed sensor chaeck is finished normally, the ABS Warnin Gramp will turn off while driving, and flashes test mode during the stop.



    6. End of sensor chak


      1. If the sensor chaeck is finished successfully, the lighting state of the ABS Warnin Grand lamp will blink while the vehicle is stopped (test mode display) and turn off during vehicle driving.

        ■ Attention ■
        1. If the sensor chaeck is not terminated, it will remain while running.
        2. If there is an abnormality in the sensor, it will be lit.
        3. If the sensor chaeck is finished successfully (the lamp turns off while driving), the ABS can be activated.ABS will not work if there is no end or abnormalities.

    7. Test mode code reading (Tascan)


      1. Read the test mode code according to the SST (Tascan) display screen in the test mode.

        SST
        09991-70200




    8. Test mode code reading (DLC3 short circuit)


      1. IG Switch off.

      2. With a SST, the DLC3 12 (TS) ← → 4 (CG) terminal is short -circuited, and the 13 (TC) ← → 4 (CG) terminal is short -circuited.

        SST
        09843-18040

        ■ Attention ■
        1. There is no mistake in the connector connection position because it causes a failure.
        2. Perform in the vehicle stopped.

      3. Turn on IG switch.

      4. Read the number of flashes of ABS Warnin Grand lamps.

        ■ Attention ■
        1. If only the diagnostic code is displayed, repair the problem and delete the diagnostic cord.After erasing, make sure that the ABS Warnin Gramp is normal.
        2. When only the test mode code is displayed, the defective part is repaired, the diagnostic code is erased, and the test mode is inspected again.
        3. When the diagnostic code and test mode code are displayed, the defective part is repaired, the diagnostic code is erased, and the test mode inspection is performed again.

        □ Participation □
        1. The test mode code and the diagnostic code are displayed.
        2. In the case of normal, 0.24 seconds lit and 0.24 seconds are repeated.
        3. If one abnormal code is one, the same code is displayed every 4 seconds.
        4. If the abnormal code is multiple, a different code is displayed at 2.5 seconds, and once a round, it is output again after 4 seconds.
        5. When outputting multiple code, output in order from the small code number.

      5. Open between 12 (TS) ← → 4 (CG) of DLC3.

    9. Test mode termination method (TASCAN)


      1. Switch from test mode to normal mode according to the SST (Tascan) screen display.

      2. Turn off the SST (Tascan) by turning off the IG switch.

    10. Test mode termination method (DLC3 short circuit)


      1. IG Switch off.

      2. Open between 12 (TS), 13 (TC) ← → 4 (CG) of DLC3.

      3. Turn on IG switch.
