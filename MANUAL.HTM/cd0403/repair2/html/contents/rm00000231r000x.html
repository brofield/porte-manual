      1. Diagnostic screen transition





        1. Inspection / adjustment

          1. Preparation before adjustment

            □ Participation □
            Install a target for adjustment only when adjusting the camera optical axis.


            1. Stop the vehicle with the steering wheel in a straight line.

            2. Install the adjustment target bar at the position of the figure.

              Reference value
              1250mm (a)

              □ Participation □
              1. The adjustment target bar is a tape of about 2.5 cm wide and 2m long on the floor.
              2. Before stopping the vehicle, move the vehicle back and forth to make sure that not only the steering wheels but also the tires are straight.




          2. Diag activation


            1. Make the engine ON (engine start state).

            2. Start diagnostic.(The point isreference)

            3. Select "Camera Inspection" on the diagram test menu screen.




          3. Vehicle signal input confirmation


            1. Confirm the status of the vehicle speed signal (part A) entered in the navigation receiver asser.

            2. If the part A is "car speed input OK" (blue display), proceed to the steering angle adjustment for normal.

              □ Participation □
              If you have been entered at least once in the car speed signal, it will be blue.

            3. In the case of "vehicle speed input checking" (red display), check in accordance with the instructions of the symptom -specific list [vehicle speed signal inspection result].(The point isreference)

              □ Participation □
              1. If the signal is not entered, the icon will be displayed in red along with the display in the "Vehicle Speed ​​Input Check".
              2. If the "vehicle speed input check" is red, you will not be able to proceed to the next screen.
              3. In the case of "Vehicle speed input checking" (red display), touch it in "Vehicle speed input check" (change from red to orange) to "next", and you will not proceed to the next screen.




          4. Red angle adjustment


            1. Confirm that the steering wheels of the vehicle are in a straight position (almost straight ± 5 degrees or less), and press "Red angle in the middle of the steering angle".

              □ Participation □
              1. Since there is no need to adjust the steering angle during the television camera ASSY ASSY and replacement, select "Next" on the steering angle adjustment screen.
              2. Even if only the memory of the rudder angle neutral point is performed, it is not memorable without the "next" ... "completed" and the diag inspection screen is completed.

            2. After remembering the standing point, turn the steering to the left, then turn it to the right and press the "maximum steering angle memory".When it is pressed, the left and right rudder angle is remembered.(Even if the right is ahead)

              □ Participation □
              "Next" is not accepted even if you touch it until you have a maximum steering angle memory and the maximum steering angle memory.




          5. Camera optical axis adjustment


            1. Press A or B switch to rotate C so that it is parallel to the adjustment target bar.

            2. Press the left and right, right and left switch to move the C to the left and right, left, right, up and down, so that the adjustment target bar is in the center of the middle (red frame).

            3. Select "Next".




            4. A (indicated 1250mm from the bumper edge) and confirm that the adjustment target bar is almost overlapped.Also, when the steering is straight, make sure that the expected course is straight (overlapped with the extension of the vehicle width).

            5. Select "Finish" and return to the diager inspection menu.

          6. Diag end


            1. The release method is turned off the IG switch.Or press the “image quality · erasing” key for 3 seconds or more.

          7. Adjustment confirmation


            1. When the maximum rudder angle memory is performed, the screen is confirmed on the screen that the predictable course is linked to the end of the steering wheel after the diag test is completed.

              □ Participation □
              When the predicted course does not move before the steering is cut to the left and right, the maximum steering angle is not remembered correctly, so adjustment is required again.
