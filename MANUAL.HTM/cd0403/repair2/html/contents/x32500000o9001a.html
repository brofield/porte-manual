ディファレンシャルオイル（4WD）  調整

印刷

準備品一覧



1. ディファレンシャルオイル点検

  1. リヤアクスルハウジングフィラプラグおよびガスケットを取りはずす。




  2. リヤアクスルハウジングフィラプラグ穴下端から0-5mm以内にオイルがあることを確認する。

    ■ 注 意 ■
    1. オイル過多、過小はトラブルの原因になる。
    2. 車両を平坦路に停止させる。

  3. オイル量が少ない場合は、オイル漏れ点検後、オイルを補充する。

  4. 新品のガスケットを介してリヤアクスルハウジングフィラプラグを締め付ける。

    基準値
    T = 49 N・m{ 500 kgf・cm }

    ■ 注 意 ■
    オイル交換時は、走行後オイル量を再確認する。



2. ディファレンシャルオイル補充

  1. リヤアクスルハウジングフィラプラグおよびガスケットを取りはずす。

  2. オイルを補充する。

    参考値:
    リヤアクスルハウジングフィラプラグ穴下端から0-5mm以内

  3. オイル量を点検する。

  4. 新品のガスケットを介してリヤアクスルハウジングフィラプラグを締め付ける。

    基準値
    T = 49 N・m{ 500 kgf・cm }

    ■ 注 意 ■
    オイル交換時は、走行後オイル量を再確認する。
