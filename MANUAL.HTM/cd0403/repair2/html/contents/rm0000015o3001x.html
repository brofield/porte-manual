EBD付きABS&BAシステム（4WD）  ダイアグコード点検／消去

印刷



  1. ダイアグノーシスコード出力点検

    1. SST(TaSCAN)によるダイアグノーシスコード読み取り


      1. IGスイッチOFFにする。

      2. SST(TaSCAN)をDLC3に接続する。

        SST
        09991-70200  

      3. IGスイッチONにする。

      4. SST(TaSCAN)の表示画面に従って、ダイアグノーシスコードを読み取る。

        ■ 注 意 ■
        異常がある場合は、修理後にダイアグノーシスコード記憶を消去する。




    2. ウォーニングランプによるダイアグノーシスコード読み取り


      1. IGスイッチOFFにする。

      2. SSTを使用して、DLC3の13(TC)←→4(CG)端子間を短絡する。

        SST
        09843-18040  

        ■ 注 意 ■
        1. 故障の原因となるため、コネクターの接続位置を間違えない。
        2. 車両停止状態で行う。

      3. IGスイッチONにする。




      4. ABSウォーニングランプの点滅回数を読み取る。

        □ 参 考 □
        1. 正常な場合は0.24秒点灯、0.24秒消灯を繰り返す。
        2. 異常コードが1つの場合、同一コードが4秒間隔で表示される。
        3. 異常コードが複数の場合、異なるコードが2.5秒間隔で表示され、一巡すると4秒後に再度出力される。
        4. 複数のコードを出力する場合は、コード番号の小さいものから順に出力する。

      5. DLC3の13(TC)←→4(CG)端子間を開放する。

        ■ 注 意 ■
        異常がある場合は、修理後にダイアグノーシスコードの記憶を消去する。



    1. ダイアグノーシスコード、フリーズフレームデータ消去

      1. ダイアグノーシスコード消去方法(TaSCAN)


        1. IGスイッチOFFにする。

        2. SST(TaSCAN)をDLC3に接続する。

          SST
          09991-70200  

        3. IGスイッチONにする。

        4. SST(TaSCAN)の表示画面に従って、ダイアグノーシスコードを消去する。




      2. ダイアグノーシスコード消去方法(DLC3短絡)


        1. IGスイッチOFFにする。

        2. SSTを使用して、DLC3の13(TC)←→4(CG)端子間を短絡する。

          SST
          09843-18040  

          ■ 注 意 ■
          故障の原因となるため、コネクターの接続位置を間違えない。

        3. IGスイッチONにして、5秒以内に8回以上ブレーキペダルを踏み込む。

        4. 正常コードが出力されていることを確認する。

          □ 参 考 □
          故障が継続しているときは、消去できない場合がある。
