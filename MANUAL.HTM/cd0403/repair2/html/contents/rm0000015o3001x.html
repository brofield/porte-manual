EBD ABS & BA system (4WD) Diag code inspection / erasure

printing



  1. Diagnostic code output inspection

    1. Diagnostic code read by SST (TASCAN)


      1. IG Switch off.

      2. Connect SST (Tascan) to DLC3.

        SST
        09991-70200

      3. Turn on IG switch.

      4. According to the SST (Tascan) display screen, you can read the diagnostic cord.

        ■ Attention ■
        If there is an abnormality, delete the diagnostic cord memory after repair.




    2. Diagnothesis code read by Warnin Gramp


      1. IG Switch off.

      2. Use SST to short circuit between 13 (TC) ← → 4 (CG) of DLC3.

        SST
        09843-18040

        ■ Attention ■
        1. There is no mistake in the connector connection position because it causes a failure.
        2. Perform in the vehicle stopped.

      3. Turn on IG switch.




      4. Read the number of flashes of ABS Warnin Grand lamps.

        □ Participation □
        1. In the case of normal, 0.24 seconds lit and 0.24 seconds are repeated.
        2. If one abnormal code is one, the same code is displayed every 4 seconds.
        3. If the abnormal code is multiple, a different code is displayed at 2.5 seconds, and once a round, it is output again in 4 seconds.
        4. When outputting multiple code, output in order from the small code number.

      5. Open between 13 (TC) ← → 4 (CG) of DLC3.

        ■ Attention ■
        If there is an abnormality, delete the memory of the diagnostic cord after repair.



    1. Diagnostic code, freeze -frame data eraser

      1. Diagnothescode erasing method (TASCAN)


        1. IG Switch off.

        2. Connect SST (Tascan) to DLC3.

          SST
          09991-70200

        3. Turn on IG switch.

        4. Follow the SST (Tascan) display screen to delete the diagnostic code.




      2. Diagnothescode erasing method (DLC3 short circuit)


        1. IG Switch off.

        2. Use SST to short circuit between 13 (TC) ← → 4 (CG) of DLC3.

          SST
          09843-18040

          ■ Attention ■
          There is no mistake in the connector connection position because it causes a failure.

        3. Set the IG switch on and step on the brake pedal or more within 5 seconds.

        4. Confirm that the normal code is output.

          □ Participation □
          If the failure is continuing, it may not be possible to erase it.
