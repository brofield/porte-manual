2. パーキングブレーキ スイッチASSY取り付け

  1. スクリュ1本で、パーキングブレーキスイッチASSYを取り付ける。



3. パーキングブレーキ ケーブルASSY NO.1取り付け



  1. パーキングブレーキケーブルASSY　No.１をパーキングブレーキペダルに通し、ツメを確実に折り曲げる。

  2. クリップで、パーキングブレーキケーブルASSY　No.１をパーキングブレーキペダルに固定する。




  3. 図の位置でロックナットおよびアジャスティングナットを固定する。

    基準値
    T = 5.4 N・m{ 55 kgf・cm }



4. パーキングブレーキコントロール ペダルASSY取り付け



  1. クリップを取り付け、パーキング ブレーキ ペダルASSYを固定する。

  2. 上部ボルトを取り付ける。

    基準値
    T = 12.5 N・m{ 127 kgf・cm }

  3. エンジンルーム側からナット２個を取り付ける。

    基準値
    T = 12.5 N・m{ 127 kgf・cm }

  4. パーキングブレーキスイッチASSYのコネクタを接続する。



5. パーキングブレーキ ケーブルASSY NO.1接続



  1. パーキングブレーキインタミディエイトレバーに、パーキングブレーキケーブルASSY　No.１を接続し、クリップを取り付けて固定する。。

  2. パーキングブレーキケーブルASSY　No.２および、パーキングブレーキケーブルASSY　No.３をパーキングブレーキイコライザに接続する。

  3. ボルトで、パーキングブレーキケーブルASSY　No.１のクランプをフロアに取り付ける。

    基準値
    T = 12.5 N・m{ 127 kgf・cm }



6. ワイヤアジャスティング ナット NO.1仮締め



  1. パーキングブレーキワイヤアジャスティングナット　Ｎｏ．１および、ロックナットを仮締めする。

    ■ 注 意 ■
    イコライザは、取りはずし時と同じ位置に取り付ける。



7. パーキングブレーキペダル踏みしろ調整
（要領は 参照）

■ 注 意 ■
  1. 踏みしろ調整は、パーキングブレーキインタミディエイトレバー部で行う。
  2. 踏みしろ調整を行った後、パーキングブレーキインタミディエイトレバー部のパーキングブレーキワイヤアジャスティングナットＮｏ．１および、ロックナットを締め付ける。

基準値
T = 6.0 N・m{ 61 kgf・cm }

34. フロントシートASSY RH取り付け

  1. シートトラックアジャスティングハンドルを操作してフロントシートASSYをスライドさせ、フロントモーストにする。

    ■ 注 意 ■
    ロックしている事を確認する。

  2. シートASSYを車両に乗せ、フロントシートレッグASSY　RH部のピンと車両側の穴を合わせる。

    ■ 注 意 ■
    シートトラックアジャスティングハンドルを持つとスライド同期位置がずれる。

  3. 車両側ワイヤハーネスとシート側ワイヤハーネスを接続させる。

  4. ボルト2本でシートASSYのリヤ側を仮締めする。

  5. シートトラックアジャスティングハンドルを操作してフロントシートASSYをスライドさせ、リヤモーストにする。

    ■ 注 意 ■
    ロックしている事を確認する。

  6. ボルト2本でシートASSYのフロント側を仮締めする。

  7. シートトラックアジャスティングハンドルを操作してフロントシートASSYをスライドさせ、フロントモーストにする。

    ■ 注 意 ■
    ロックしている事を確認する。

  8. シートASSYのリヤ側のボルト2本ををインナ→アウタの順に締め付ける。

    基準値
    T = 37 N・m{ 377 kgf・cm }

  9. シートトラックアジャスティングハンドルを操作してフロントシートASSYをスライドさせ、リヤモーストにする。

  10. シートASSYのフロント側のボルト2本をインナ→アウタの順に締め付ける。

    基準値
    T = 37 N・m{ 377 kgf・cm }

  11. フロントシートレッグカバーのツメをかん合させる。



35. リヤシートクッションASSY取り付け（リヤシート分割可倒式）

  1. リヤシートクッションASSY前側をリヤシート クッション ロック ストライカにロックさせる。

  2. リヤシートベルトを、リヤシートクッションカバー & パッド後部のゴムバンドに通す。




  3. ボルトで、リヤシートヒンジLHを取り付ける。

    基準値
    T = 37 N・m{ 377 kgf・cm }

  4. リヤシートバックヒンジＲＨにスナップリングを取り付ける。

  5. リヤシートヒンジRHを、リヤシートクッションASSY右側ブラケットの角パイプに挿入する。

    □ 参 考 □
    リヤシートヒンジRHのスナップリング溝にスナップリングがはまっていることを確認する。
