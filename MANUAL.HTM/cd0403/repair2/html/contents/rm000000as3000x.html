1. ラジエータASSY取り付け

  1. ラジエータASSYにサブラジエータサポートクツシヨンおよびラジエータサポートLWRを取り付ける。




  2. ボルト2本で、ウオータフィラを取り付ける。

    基準値
    T = 7.5 N・m{ 76 kgf・cm }




  3. ボルト3本で、ファンシユラウド　W/ファンを取り付ける。

    基準値
    T = 7.5 N・m{ 76 kgf・cm }

  4. 車両にラジエータASSYを取り付け、クーリングフアンモータのコネクタおよびワイヤハーネスクランプを接続する。



2. ラジエータ サポートSUB-ASSY UPR取り付け

  1. ボルト4本およびクリップ1個で、ラジエータサポートUPRを取り付ける。

    基準値
    T = 5.0 N・m{ 51 kgf・cm }

  2. ホーンASSYのコネクタおよびクランプを接続する。



3. フードロックサポート ブレース取り付け

  1. ボルト4本で、ボンネット(フード)ロックサポートブレースを取り付ける。

    基準値
    T = 5.0 N・m{ 51 kgf・cm }



4. フードロック サポートSUB-ASSY取り付け

  1. ボルト3本で、ボンネット(フード)ロックサポートを取り付ける。

    基準値
    T = 5.0 N・m{ 51 kgf・cm }



5. フード ロックASSY取り付け

  1. ボンネツトロツクASSYにボンネツトロツクコントロールケーブルASSYを接続する。




  2. ボルト3本で、ボンネツトロツクASSYを仮付けする。

    □ 参 考 □
    ボルト3本は、ボンネツト(フード)調整時に規定トルクで締め付ける。



6. ラジエータ グリル W/ラジエータ サポート シール UPR取り付け

  1. ラジエータ グリルとラジエータ サポート シール UPRを組み付ける。




  2. ラジエータグリル下部のツメを合わせ、クリップ6個でラジエータ グリル W/ラジエータ サポート シール UPRを取り付ける。



7. オイルクーラアウトレットホース接続

  1. オイルクーラアウトレットホースを接続し、クランプを取り付ける。



8. オイルクーラインレットホース接続

  1. オイルクーラインレットホースを接続し、クランプを取り付ける。



9. ラジエータ アウトレットホース接続

  1. ラジエータアウトレットホースを接続し、クランプを取り付ける。



10. ラジエータ インレット ホース接続

  1. ラジエータインレットホースを接続し、クランプを取り付ける。



11. エンジンアンダ カバー RH取り付け

  1. スクリュー2個およびボルト2本で、エンジンアンダーカバーRHを取り付ける。

    基準値
    T = 5.0 N・m{ 50 kgf・cm }

  2. ナットを、締め付ける。

    □ 参 考 □
    ナットは、LH側と共締めになっている。

    基準値
    T = 5.0 N・m{ 50 kgf・cm }



12. エンジンアンダ カバー LH取り付け

  1. スクリュー2個およびボルト2本で、エンジンアンダーカバーLHを取り付ける。

    基準値
    T = 5.0 N・m{ 50 kgf・cm }



13. 冷却液(トヨタ純正スーパーLLC)補充

  1. ラジエータドレーンコツクプラグを閉じて、冷却水をラジエータ注入口よりあふれるまで注入する。［＊１］

    □ 参 考 □
    ラジエータインレツトホースおよびラジエータアウトレツトホースを手で数回圧迫し、ラジエータ注入口の水位が下がる様であれば追加注入する。

  2. ラジエータキヤツプを締める。

  3. ラジエータリザーブタンクに冷却液を上限まで注入する。

  4. エンジンをサーモスタツトが開弁するまで暖機する。

    □ 参 考 □
    暖機中にラジエータインレツトホースおよびアウトレツトホースを数回圧迫すること。

  5. エンジンを止め、冷却液が冷えるまで待ち、ラジエータキヤツプをはずして水位を確認する。

  6. 水位が下がっている場合は、［＊１］より繰り返す。

  7. 水位が下がらなくなったら、ラジエータリザーバタンクの冷却液を調整する。



14. 冷却液(トヨタ純正スーパーLLC)漏れ点検

  1. 冷却液を満水にしてテスターを取り付ける。

  2. 137ｋＰａ｛1.4ｋｇｆ/ｃｍ２｝の圧力をかけ、各部に水漏れがないことを確認する。



15. フードSUB-ASSY調整



  1. フード前端方向および後端左右方向の調整をする場合は、ボルトを緩めてから行う。

    ■ 注 意 ■
    センターリングボルトを使用しているため、調整は補給されているボルトに交換してから行う。

    基準値
    T = 13 N・m{ 133 kgf・cm }




  2. フード前端の上下方向を調整をする場合は、左右のクッションをまわして調整する。




  3. フード前端上下左右方向の調整をする場合は、ボルトを緩めてから行う。

    ■ 注 意 ■
    センターリングボルトを使用しているため、調整は補給されているボルトに交換してから行う。

    基準値
    T = 7 N・m{ 71 kgf・cm }
