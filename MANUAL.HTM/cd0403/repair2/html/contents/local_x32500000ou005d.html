How to proceed with control system CAN communication system troubles



  1. Vehicle entry

  2. Examination and phenomenon confirmation

  3. Diag code check

  4. DLC3 inspection (Canh ← → CANL)

  5. DLC3 inspection (BAT ← → CANH, CANL)

  6. DLC3 inspection (CG ← → CANH, CANL)

  7. To the diagcode combination table

  8. Diag code combination check

  9. Adjustment, repair or replacement

  10. Confirmation test
