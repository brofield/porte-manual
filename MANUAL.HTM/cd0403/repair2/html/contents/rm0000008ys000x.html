EFI System Diag Code Inspection / Erase

printing


□ Participation □
  1. With Tascan, you can select not only normal mode but also check mode, improving the abnormal detection sensitivity compared to normal mode.
  2. Inspection by check mode is performed when outputting normal code in normal mode, despite the considerations of the signal system of each sensor.



  1. Preparation before inspection

    1. Confirm that the throttle valve is fully closed.

    2. Make the shift position in N range or P range.

    3. Turn off the air conditioner.



    1. Diagnostic code inspection (read by Tascan)

      1. Connect Tascan to DLC3.

        SST
        09991 -70200

      2. The operation is performed according to the screen display, displayed the [Diagnosis] → [Mode] screen, selects normal mode or check mode, and checks the diagnostic code.

        □ Participation □
        1. If the code is not displayed (the lamp does not blink), the TC terminal system may be disconnected or a computer defect.
        2. If the check engine warnin grand lamp is always lit, a wire harness short (such as biting) or a computer defect may be possible.
        3. When outputting a meaningless code, computer failure may be considered.
        4. If the check engine warnin gland lights up with 1000R/min or higher and does not output the code, turn off the ignition switch once and then check again.




      1. Diagnothescode inspection (read by check engine warnin grand lamp)

        ■ Attention ■
        1. Turn on the ignition switch before reading the diagnostic cord and confirm that the check engine warnin glamp lights up.
        2. Check mode cannot be checked.

        1. Turn off the ignition switch.

        2. Use the diagnodes check wire No.2 to shorten the DLC3 13 (TC) ← → 4 (CG) terminal.

          SST
          09843-18040

          ■ Attention ■
          A mistake in the connection position of the connector will cause a failure to make a mistake.



        3. Turn on the ignition switch and read the number of flashing of the check engine warnin glamp.



          □ Participation □
          1. If the code is not displayed (the lamp does not blink), the TC terminal system may be disconnected or a computer defect.
          2. If the check engine warnin grand lamp is always lit, a wire harness short (such as biting) or a computer defect may be possible.
          3. When outputting a meaningless code, computer failure may be considered.
          4. If the check engine warnin gland lights up with 1000R/min or higher and does not output the code, turn off the ignition switch once and then check again.

        4. Remove the diagnodes check wire No.2.



        1. Diagnostic code memory erasure (erased by TASCAN)

          1. The operation is performed according to the screen display, and the [Diagnosis] → [Diag Code Freeze Data] screen is displayed and the diagnostic code is erased.

            SST
            09991-70200

            ■ Attention ■
            1. If you cannot erase it, turn off the ignition switch once and start over again.
            2. Until the cause is elucidated, Tascan does not delete the diag code.




          1. Diagnothescode erasure (erasure due to fuse removal)

            1. E.F.I. Hughes (15A) remove and connect the fuse after 15 seconds or more.

              ■ Attention ■
              1. After the inspection and repair of the EFI system, be sure to delete the memory of the diagnostic cord once, and then confirm that the normal code is output.
              2. Until the cause is elucidated, the diagnostic cord is not to be erased by the battery clear (without a fuse).
