1. クウネンヒ センサ取り付け



  1. SSTを使用して、クウネンヒ センサを取り付ける。

    SST
    09224-00010  
    基準値
    T = 44 N・m{ 449 kgf・cm }
    T = 39 N・m{ 398 kgf・cm } (SST+ トルクレンチL26cm 使用時)

    □ 参 考 □
    1. SST+ トルクレンチL26cm使用時の基準値はSSTとトルクレンチを一直線上で使用した基準値である。
    2. トルクレンチまたは角度を変えて使用する場合は基準値を求める必要がある。(要領は参照)

  2. クウネンヒ センサのコネクターを接続する。



2. エア クリーナASSY取り付け

  1. ボルト4本でエアクリーナケースとエアクリーナインレツトNo.1およびエアクリーナインレツトNo.2を取り付ける。

    基準値
    T = 7.8 N・m{ 80 kgf・cm } (エアクリーナケースASSY　×　エアクリーナブラケツト)
    基準値
    T = 7.5 N・m{ 76 kgf・cm } (エアクリーナインレツトNo.2　×　ボデー)

  2. エアクリーナフイルタエレメントを取り付ける。

  3. エアクリーナキヤツプとエアクリーナホースNo.1を取り付ける。
