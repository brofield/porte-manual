SRS Airbag System Diag Code Inspection / Erase

printing



  1. Diag code reading by Tascan (normal mode)

    1. Set the IG switch to OFF (Lock).

    2. Connect SST to DLC3.

      SST
      09991-70200

    3. After the IG switch is turned on, perform the operation according to the SST screen display, display the [Diagcode Confirm] screen, select a normal mode, and read the diag code.

      SST
      09991-70200



    1. Diag code reading by Tascan (check mode)

      1. Set the IG switch to OFF (Lock).

      2. Connect SST to DLC3.

        SST
        09991-70200

      3. After turning on the IG switch, perform the operation according to the SST screen display, display the Diagcode Confirm screen, select the check mode, and read the diag code.

        SST
        09991-70200

        ■ Attention ■
        If you display the SST Diagcode Confirm screen and select a check mode, the diag code (present and past) output so far will be deleted.

        □ Participation □
        1. The check mode can improve the detection sensitivity compared to the normal mode.
        2. Inspection by check mode is performed when the normal code is output when the dial code is read in normal mode, despite the concept of a signal failure (abnormality) of each sensor.
        3. When performing troubleshooting, check in the diag code list that the diag code is compatible with the check mode.



      1. Diag cord eraser by Tascan

        1. Set the IG switch to OFF (Lock).

        2. Connect SST to DLC3.

          SST
          09991-70200

        3. After the IG switch is turned on, the diag code is erased according to the SST screen display.



        1. Diagcode reading by Warnin Grand Lamp

          1. Currently failure


            1. Turn on the IG switch and wait for 60 seconds.

              □ Participation □
              It takes about 60 seconds for the airbag sensor ASSY CTR to complete the self -diagnosis of all systems.

            2. Use SST to short -circuited between DLC3 13 (TC) ← → 4 (CG) terminals and read the number of flashing of the warnin group.

              SST
              09843-18040

              □ Participation □
              If there are multiple diary codes, output them in order of the numbers, and the second display is displayed after all code output.



          2. Past failure


            1. Use SST to short circuit between 13 (TC) ← → 4 (CG) of DLC3.

              SST
              09843-18040




            2. Turn on the IG switch and read the number of blinks after the warnin Grand lamp is turned off.

              □ Participation □
              1. If there are multiple diary codes, output them in order of the numbers, and the second display is displayed after all code output.
              2. If there is currently a failure, the current and past failures will be output at the same time.






          1. Diag cord erasing by warnin grand lamp



            1. Use SST to short -circuited between the 13 (TC) terminals ← → 4 (CG) of the DLC3 and turn on the IG switch.

              SST
              09843-18040

            2. Open the 13 (TC) terminal of DLC3 within 3 seconds to 10 seconds after the diag code begins to be output, and check if the warnin gland will light up 3 seconds.

            3. Within 2.0 seconds-4.0 seconds after the warnin glander starts to light up, the DLC3 13 (TC) ← → 4 (CG) terminal is short-circuited.

            4. The Warnin Grand Lamp turns off 2.0 seconds after short circuit between the DLC3 (TC) ← → (CG) terminals, and the 13 (TC) terminal of DLC3 is opened within 2.0-after -40 seconds after the lights are turned off.。

            5. 2.0 seconds-4.0 seconds after opening the 13 (TC) terminal of DLC3, the warnin glamp turns on again, and within 2.0 seconds-4.0 seconds after lighting, DLC3 13 (TC) ← → 4 (CG) between terminals.Is short -circuited.

            6. After short circulation between the 13 (TC) terminal ← → 4 (CG) of the DLC3, the warnin glander turns off 2.0-4.0 seconds after the terminals are turned off, and the normal code is output within 1.0 seconds after the lights are turned.

              □ Participation □
              If the normal code is not output, do it again because the diag code has not been erased.
