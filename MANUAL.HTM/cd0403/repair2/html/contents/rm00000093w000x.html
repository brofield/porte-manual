フロントサスペンションロワーアームNo.1　LH  取り付け

準備品一覧



1. フロントサスペンション アームSUB-ASSY LWR NO.1 LH仮締め

  1. ボルト2本およびナットでサスペンションロワーアームＮｏ．1をフロントサスペンションクロスメンバに仮締めする。

    ■ 注 意 ■
    ナットは回り止めナットのため、必ずボルトを回して取り付ける。



2. フロントサスペンション クロスメンバSUB-ASSY取り付け



  1. ミッションジャッキで、クロスメンバを支える。

  2. ボルト4本で、クロスメンバをボデーに仮付けする。

  3. SSTをクロスメンバRH側および車両RH側の基準穴に差し込み、ボルト穴位置を調整しながらボルト2本を仮締めする。

    SST
    09670-00010  

    □ 参 考 □
    ボルトA、Bの順序で仮締めする。




  4. SSTをクロスメンバLH側および車両LH側の基準穴に差し込み、ボルト穴位置を調整しながらボルト2本を仮締めする。

    SST
    09670-00010  

    □ 参 考 □
    ボルトA、Bの順序で仮締めする。

  5. SSTをクロスメンバRH側および車両RH側の基準穴に差し込み、規定トルクで締め付ける。

    SST
    09670-00010  
    基準値
    T = 70 N・m{ 714 kgf・cm } (ボルトB(フロント側))
    T = 116 N・m{ 1183 kgf・cm } (ボルトA(リヤ側))

  6. SSTをクロスメンバLH側および車両ＬＨ側の基準穴に差し込み、規定トルクで締め付ける。

    基準値
    T = 70 N・m{ 714 kgf・cm } (ボルトB(フロント側))
    T = 116 N・m{ 1183 kgf・cm } (ボルトA(リヤ側))

    ■ 注 意 ■
    クロスメンバ取り付け後、クロスメンバ側および車両側の基準穴が一致していることを確認する。

  7. ミッションジャッキをクロスメンバから取りはずす。




  8. ボルトおよびナット2個で、エンジンマウンテイングインシユレータRRをクロスメンバに取り付ける。

    基準値
    T = 80 N・m{ 816 kgf・cm }

  9. スリングデバイスおよびチェーンブロックを取りはずす。



3. フロントサスペンション アームSUB-ASSY LWR NO.1 LH接続

  1. サスペンションロワーアームNo.1のボールジョイント部をステアリングナックルに取り付け、新品のキャッスルナットで締め付ける。

    基準値
    T = 98 N・m{ 1000 kgf・cm }

    ■ 注 意 ■
    1. ネジ部およびテーパー部に油脂類を付着させない。
    2. キャッスルナットの締め付け後のクリップ穴合わせは、60°以内の増し締め方向で行う。

  2. 新品のクリップを取り付ける。



4. フロントサスペンション アームSUB-ASSY LWR NO.1 RH接続

□ 参 考 □
RH側はLH側と同様の手順で行う。



5. タイロッド エンドSUB-ASSY LH接続

  1. タイロッドエンドをステアリングナックルに取り付け、キャッスルナットで締め付ける。

    基準値
    T = 49 N・m{ 500 kgf・cm }

    ■ 注 意 ■
    ねじ部およびテーパー部に油脂類を付着させない。

  2. 新品のコッターピンを取り付ける。

    ■ 注 意 ■
    キャッスルナット締め付け後のコッターピン穴合わせは60°以内の増し締め方向で行う。



6. タイロッド エンドSUB-ASSY RH接続

□ 参 考 □
RH側はLH側と同様の手順で行う。



7. プレッシャフィード チューブASSY取り付け



  1. プレツシヤフイードチユーブASSYをステアリングリンクASSYに仮付けする。

  2. ボルトで、プレツシヤフイードチユーブASSYのクランプをクロスメンバに取り付ける。

    基準値
    T = 7.8 N・m{ 80 kgf・cm }




  3. ユニオンナットレンチ17を使用して、プレツシヤフイードチユーブASSYを本締めする。

    基準値
    T = 44.1 N・m{ 450 kgf・cm } (規定締付けトルク)
    T = 40 N・m{ 410 kgf・cm } (ユニオンナットレンチおよびトルクレンチ460F使用時)



8. フロントスタビライザ ボルト取り付け

  1. フロントスタビライザバークッションリテーナNo.1を4個、フロントスタビライザクッション4個、フロントスタビライザバークッションリテーナNo.2およびフロントスタビライザボルトを取り付ける。

    □ 参 考 □
    1. スタビライザボルトを図のように取り付けると、固定しやすい。
    2. ロワーアームをジャッキで少し持ち上げると締めやすい。

  2. スタビライザボルトをスパナ(10mm)で固定し、新品のナット2個で取り付ける。

    基準値
    T = 18 N・m{ 184 kgf・cm }




  3. 反対側も同様の手順で取り付ける。



9. ステアリングスライディング ヨークSUB-ASSY接続



  1. クリップAを付け、ホールカバーおよびクリップBをボデーに取り付ける。

    ■ 注 意 ■
    リップ部が噛み込んでいないか確認する。




  2. 合わせマークを合わせて、ボルトでスライディングヨークを取り付ける。

    基準値
    T = 28 N・m{ 290 kgf・cm }




  3. ステアリングホイール回転防止用シートベルトを取りはずす。



10. ステアリングコラムホールカバー プレート取り付け

  1. クリップ2個でホールカバープレートを取り付ける。



11. フロントタイヤ取り付け
基準値
T = 103 N・m{ 1050 kgf・cm }


12. フロントサスペンション アームSUB-ASSY LWR NO.1 LH本締め

  1. 車両をジャッキダウンして数回上下にゆすり、フロントサスペンションを落ち着かせる。

  2. ボルト(Ａ)を本締めする。

    基準値
    T = 137 N・m{ 1400 kgf・cm }




  3. ナットを固定し、ボルト(Ｂ)を本締めする。

    基準値
    T = 132 N・m{ 1350 kgf・cm }

    ■ 注 意 ■
    ボルトを締め付ける際、ナットを回転させない。



13. エンジンアンダ カバー RH取り付け

  1. スクリュー2個およびボルト2本で、エンジンアンダーカバーRHを取り付ける。

    基準値
    T = 5.0 N・m{ 50 kgf・cm }

  2. ナットを、締め付ける。

    □ 参 考 □
    ナットは、LH側と共締めになっている。

    基準値
    T = 5.0 N・m{ 50 kgf・cm }



14. エンジンアンダ カバー LH取り付け

  1. スクリュー2個およびボルト2本で、エンジンアンダーカバーLHを取り付ける。

    基準値
    T = 5.0 N・m{ 50 kgf・cm }
