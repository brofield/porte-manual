1. ブレーキ ブースタASSY取り付け

  1. 新品のブレーキブースタガスケツトを介して、ブレーキブースタASSYを取り付ける。

  2. クレビスロックナットおよびプッシュロッドクレビスをブレーキブースタASSYに取り付ける。




  3. ナット4個を取り付ける。

    基準値
    T = 12.7 N・m{ 130 kgf・cm } ナットA
    基準値
    T = 9.0 N・m{ 92 kgf・cm } ナットB

    ■ 注 意 ■
    1. ナット補給は図Aタイプのみ。
    2. ブレーキブースタASSY取り付けは必ず各締め付けトルクで締め付けること。

  4. クリップで、バキュームホースをバキュームチエックバルブに接続する。



2. ブレーキ マスタシリンダ リザーバ ブラケット取り付け

  1. ボルト3本で、ブレーキマスタシリンダリザーバブラケツトをボデーに取り付ける。

    基準値
    T = 4.7 N・m{ 50 kgf・cm }

  2. アクセルレータコントロールケーブルのクランプを取り付ける。



3. ブレーキブースタプッシュロッド点検·調整

  1. エンジン停止状態でブレーキペダルを数回踏み、ブレーキブースタ内の負圧を抜く。




  2. シツクネスゲージ(0.3mm)を介して、SSTをマスタシリンダに当てる。

    SST
    09737-00013  

  3. SSTのロッドをマスタシリンダのピストンに軽く当て、固定する。

    SST
    09737-00013  

  4. SSTのロッド平面にチヨークを塗布し、SSTを反転させてブレーキブースタASSYに取り付け、すき間を点検する。

    SST
    09737-00013  

    基準値
    0mm

    □ 参 考 □
    SSTのロッドをブレーキブースタASSYのプツシユロツドに当てた時、SST本体とブレーキブースタASSYのシエルの間にすき間がある場合はすき間過小、ブレーキブースタASSYのブツシユロツドにチヨークが付着しない場合はすき間過大と判断する。




  5. 基準値外の場合はSSTを使用してロツドを固定し、ボツクスレンチ(7mm)を使用してロツド先端を回し、長さを調整する。

    SST
    09737-00020  

  6. さらにボックスレンチ(7mm)を0.5回転(180°)回転させ、ロッド先端を0.5mm奥へ押し込む。(VSCなし車)
    (参考) 
    車両型式
    		フレームNo.
    NNP10-AGSGK
    		0017402以降
    NNP11-AGSGK
    		0019648以降

    ■ 注 意 ■
    調整後、プツシユロツドのすき間を再点検する。



4. ブレーキマスタ シリンダSUB-ASSY取り付け



  1. ナット2個で、ブレーキマスタシリンダASSYをブレーキブースタASSYに取り付ける。

    基準値
    T = 13 N・m{ 130 kgf・cm }

  2. ブレーキマスタシリンダリザーバのつめ部をマスタシリンダリザーバブラケツトに合わせて、ボルトを取り付ける。

    基準値
    T = 4.7 N・m{ 50 kgf・cm }




  3. ユニオンナットレンチ１０を使用して、ブレーキチユーブ２本をブレーキマスタシリンダASSYに接続する。

    基準値

    T=15N·m {155kgf·cm} (規定締め付けトルク)
    T’=14N·m {143kgf·cm} (ユニオンナットレンチおよびトルクレンチ230F使用時の読み)

  4. ブレーキフルードレベルウオーニングスイッチコネクタをブレーキマスタシリンダリザーバに接続する。



5. フロントブレーキ チューブ NO.4接続



  1. ユニオンナットレンチ１０を使用して、フロントブレーキチユーブNo.4をフロントブレーキチユーブNo.3および、フロントフレキシブルホースNo.1に接続する。

    基準値

    T=15N·m {155kgf·cm} (規定締め付けトルク)
    T’=14N·m {143kgf·cm} (ユニオンナットレンチおよびトルクレンチ230F使用時の読み)

  2. グロメットをボデーに取り付ける。



6. ブレーキマスタシリンダプッシュロッド クレビス接続

  1. プッシュロッドクレビスピンで、プッシュロッドクレビスをブレーキペダルに接続する。

  2. クリップをプッシュロッドクレビスピンに取り付ける。

    □ 参 考 □
    クレビスロックナットはブレーキペダル高さ調整時に締め付ける。

  3. ニードルノーズプライヤーを使用して、ブレーキペダルリターンスプリングを取り付ける。



7. ブレーキペダル高さ点検·調整

■ 注 意 ■
カーペットをめくった状態で点検·調整を行う。

  1. ブレーキペダル高さ点検

    基準値:
    181.9±5mm

  2. ブレーキペダル高さ調整

    1. コネクターをストツプランプスイツチから切り離す。

    2. ストツプランプスイツチのロツクナットをゆるめ、ストツプランプスイッチを回してペダルを遊ばせる。

    3. クレビスロックナットをゆるめ、プツシユロッドを回してブレーキペダル高さを調整する。

      基準値:
      181.9±5mm

    4. ストツプランプスイツチのねじ端部とブレーキペダルのクツシヨンの隙間が0.5-2.4mmになるようにストツプランプスイツチを調整し、ロツクナットおよびクレビスロツクナットを締め付ける。

      基準値:
      Ｔ＝26N·m｛265kgf·cm｝

    5. コネクタをストツプランプスイツチに接続する。




    6. ブレーキペダルを踏まない状態でストツプランプが点灯しないことを確認する。



8. ブレーキペダル遊び点検

  1. エンジン停止後ブレーキペダルを数回強く踏み込み、ブレーキブースタ内の負圧がない状態にする。




  2. ブレーキペダルを軽く指で押して、ブレーキペダルの遊びを点検する。

    基準値:
    1-6mm



9. ブレーキペダル踏み残りしろ点検



  1. ブレーキペダルを踏力500N {51kgf} で踏込んだ時の踏み残り代を点検する。

    基準値
    107mm以上

    ■ 注 意 ■
    カーペツトをめくった状態で点検を行う。

13. カウルトップ パネル OUT取り付け

  1. ボルト１０本で、カウルトップパネルOUTを取り付ける。

    基準値
    T = 5.0 N・m{ 51 kgf・cm }



14. ウインドシールド ワイパ リンクASSY取り付け



  1. ウインドシールドワイパリンクASSYをウインドシールドワイパモータASSYが付いた状態でピンのかん合にはめ込む。

  2. ボルト2本で、ウインドシールドワイパリンクASSYを取り付ける。

    基準値
    T = 5.5 N・m{ 56 kgf・cm }

  3. コネクタを接続する。



15. カウルトップベンチレータ ルーバ LH取り付け



  1. クリップとツメ8個をかん合させ、カウルトップベンチレータルーバLHを取り付ける。



16. カウルトップベンチレータ ルーバ RH取り付け



  1. クリップ2個とツメ7箇所をかん合させ、カウルトップベンチレータルーバRHを取り付ける。



17. フード トゥー カウルトップ シール取り付け



  1. クリップ6個をかん合させ、ボンネット(フ-ド)ツウカウルトップシールを取り付ける。



18. フロントワイパアームRH取り付け



  1. ワイパピボットセレーション部をワイヤブラシで清掃する。

  2. ワイパアームセレーション部を丸やすり等で清掃する。(再組付け時)




  3. 図の位置にフロントワイパブレードがくるようにフロントワイパアームRHをナットで取り付ける。

    基準値
    T = 20.5 N・m{ 209 kgf・cm }

    □ 参 考 □
    アームヒンジ部を手で押さえながらナットを締め付ける。



19. フロントワイパアームLH取り付け

  1. ワイパピボットセレーション部をワイヤブラシで清掃する。

  2. ワイパアームセレーション部を丸やすり等で清掃する。(再組付け時)




  3. 図の位置にフロントワイパブレードがくるようにフロントワイパアームLHをナットで取り付ける。

    基準値
    T = 20.5 N・m{ 209 kgf・cm }

    □ 参 考 □
    アームヒンジ部を手で押さえながらナットを締め付ける。

  4. 水またはウォッシャ液をガラスにかけながらフロントワイパを作動させ、ブレードの払拭状態およびボデーと干渉しないことを確認する。
