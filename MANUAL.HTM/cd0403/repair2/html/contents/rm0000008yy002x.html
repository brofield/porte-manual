準備品一覧



1. ノックコントロール センサ取り付け



  1. ナットで、ノックコントロールセンサを取り付ける。

    基準値
    T = 20 N・m{ 204 kgf・cm }

    ■ 注 意 ■
    ノックコントロールセンサの取り付け角度は、垂直ラインに対し上向き30°以内の範囲とする。

  2. ノックコントロールセンサのコネクターを接続する。



2. インテーク マニホルド取り付け

  1. 新品のインテークマニホルドツウヘッドガスケットをインテークマニホールドに取り付ける。




  2. 図の順序でインテークマニホルドを仮締めした後、規定トルクで締め付ける。

    基準値
    T = 30 N・m{ 306 kgf・cm }

  3. インテークマニホールドにワイヤハーネスを接続する。

  4. ボルト2本で、ブレーキブースタ用バキュームチューブを取り付ける。

    基準値
    T = 9.0 N・m{ 92 kgf・cm }



3. ユニオン トゥー コネクタチューブ ホース接続

  1. インテークマニホールドにユニオン トゥー コネクタチューブ ホースを接続する。



4. ベンチレーション ホース接続

  1. インテークマニホールドにベンチレーシヨンホースを接続する。



5. ウォータ フィラSUB-ASSY取り付け



  1. ボルト2本で、ウオータフイラを取り付ける。

    基準値
    T = 7.5 N・m{ 76 kgf・cm }

  2. ラジエータホースNo.3およびラジエータリザーブタンクホースを接続する。



6. スロットル ボデーASSY取り付け

  1. 新品のガスケットをインテークマニホールドに取り付ける。

  2. ボルトおよびナット2個で､スロツトルボデーおよびアクセルレータコントロールケーブルブラケツトを取り付ける。

    基準値
    T = 30 N・m{ 306 kgf・cm }
    (インテークマニホールド新品時)
    基準値
    T = 20 N・m{ 204 kgf・cm }
    (インテークマニホールド再使用時)

  3. スロツトルポジシヨンセンサおよびアイドルスピードコントロールバルブのコネクタを接続する。



7. ウォータバイパス ホース NO.2接続


8. ウォータバイパス ホース接続


9. アクセルレータ コントロールケーブルASSY取り付け

  1. アクセルレータコントロールケーブルASSYのナットナットでアクセルレータコントロールケーブルブラケツトに接続する。



10. トランスミッションオイルフィラ チューブSUB-ASSY取り付け

  1. 新品のOリングにオートフルードタイプT-IVを塗布し、オイルフイラチユーブに取り付ける。

  2. ボルトで、オイルフイラチユーブをトランスアクスルASSYに取り付ける。

    基準値
    T = 5.5 N・m{ 56 kgf・cm }

  3. ボルトで、オイルクーラチユーブクランプをトランスアクスルASSYに取り付ける。

    基準値
    T = 5.5 N・m{ 56 kgf・cm }

  4. ブリーザホースをオイルフイラチユーブに取り付ける。

  5. オイルレベルゲージをオイルフイラチユーブに取り付ける。



11. シリンダヘッド カバー NO.2取り付け



  1. ナットA 2個を締め付け後、ナットB 2個でシリンダヘッドカバーNO.2を取り付ける。

    基準値
    T = 7.0 N・m{ 71 kgf・cm }



12. エア クリーナASSY取り付け

  1. ボルト4本でエアクリーナケースとエアクリーナインレツトNo.1およびエアクリーナインレツトNo.2を取り付ける。

    基準値
    T = 7.8 N・m{ 80 kgf・cm } (エアクリーナケースASSY　×　エアクリーナブラケツト)
    基準値
    T = 7.5 N・m{ 76 kgf・cm } (エアクリーナインレツトNo.2　×　ボデー)

  2. エアクリーナフイルタエレメントを取り付ける。

  3. エアクリーナキヤツプとエアクリーナホースNo.1を取り付ける。



13. 冷却液(トヨタ純正スーパーLLC)補充

  1. ラジエータドレーンコツクプラグを閉じて、冷却水をラジエータ注入口よりあふれるまで注入する。［＊１］

    □ 参 考 □
    ラジエータインレツトホースおよびラジエータアウトレツトホースを手で数回圧迫し、ラジエータ注入口の水位が下がる様であれば追加注入する。

  2. ラジエータキヤツプを締める。

  3. ラジエータリザーブタンクに冷却液を上限まで注入する。

  4. エンジンをサーモスタツトが開弁するまで暖機する。

    □ 参 考 □
    暖機中にラジエータインレツトホースおよびアウトレツトホースを数回圧迫すること。

  5. エンジンを止め、冷却液が冷えるまで待ち、ラジエータキヤツプをはずして水位を確認する。

  6. 水位が下がっている場合は、［＊１］より繰り返す。

  7. 水位が下がらなくなったら、ラジエータリザーバタンクの冷却液を調整する。



14. 冷却液(トヨタ純正スーパーLLC)漏れ点検

  1. 冷却液を満水にしてテスターを取り付ける。

  2. 137ｋＰａ｛1.4ｋｇｆ/ｃｍ２｝の圧力をかけ、各部に水漏れがないことを確認する。
