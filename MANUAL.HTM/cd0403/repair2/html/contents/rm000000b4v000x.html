リヤワイパ モータASSY  取り付け

印刷

準備品一覧



1. リヤワイパ モータASSY取り付け



  1. ボルト３本で、リヤワイパモータＡＳＳＹを取り付ける。

    基準値
    T = 5.5 N・m{ 55 kgf・cm }

  2. コネクタを接続する。



2. バックドアトリム ボード取り付け



  1. クリップ9箇所をかん合させ、バツクドアトリムボードを取り付ける。



3. リヤワイパモータ グロメット取り付け



  1. リヤワイパモータグロメツトのマークを上にして取り付ける。



4. リヤ ワイパ アーム取り付け



  1. リヤワイパピボットセレーション部をワイヤブラシで清掃する。

  2. リヤワイパアームセレーション部を丸やすり等で清掃する。(再組み付け時)




  3. リヤワイパモータASSYを自動停止位置に停止させ、ナットでリヤワイパアームASSYを図の様に取り付ける。

    基準値
    T = 5.5 N・m{ 55 kgf・cm }

    □ 参 考 □
    アームヒンジ部を手で押さえながらナットを締め付ける。

  4. 水またはウォッシャ液をガラスにかけながらリヤワイパを作動させ、ブレードの拭払状態およびボデーと干渉しないことを確認する。



5. リヤワイパ アーム ヘッドキャップ取り付け
